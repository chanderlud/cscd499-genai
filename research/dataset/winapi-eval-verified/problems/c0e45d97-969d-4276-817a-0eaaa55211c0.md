1. Read entire file (Win32)

* **Spec:** Read the full contents of a file into memory.
* **Constraints:** Use `CreateFileW` + `ReadFile` (loop until EOF). Don’t use `std::fs`.
* **Signature:**

```rust
pub fn read_to_end(path: &Path) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let bytes = read_to_end(Path::new(r"C:\tmp\data.bin"))?;
```
