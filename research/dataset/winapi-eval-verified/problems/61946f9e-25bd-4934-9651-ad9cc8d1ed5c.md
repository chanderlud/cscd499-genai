16. In-memory COM stream roundtrip (COM)

* **Spec:** Create an `IStream` backed by global memory, write bytes, seek to start, read back.
* **Constraints:** Use `CreateStreamOnHGlobal` + `IStream::{Write, Seek, Read}`.
* **Signature:**

```rust
pub fn hglobal_stream_roundtrip(data: &[u8]) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let out = hglobal_stream_roundtrip(b"stream me")?;
```
