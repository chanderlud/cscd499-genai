7. Normalize separators in-place with `PWSTR`

* **Spec:** Modify a null-terminated UTF-16 buffer in place, replacing every `\` with `/`. Return the number of replacements made. A null pointer counts as `0`.
* **Constraints:** Use `PWSTR`; do not allocate; stop at the first terminator.
* **Signature:**

```rust
use windows::core::PWSTR;

pub fn normalize_separators(buf: PWSTR) -> usize;
```

* **Example:**

```rust
let mut data: Vec<u16> = r"C:\tmp\a\b".encode_utf16().chain([0]).collect();
let changed = normalize_separators(PWSTR::from_raw(data.as_mut_ptr()));
assert_eq!(changed, 3);
```
