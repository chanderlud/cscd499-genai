8. Safe “write temp then atomic replace” update

* **Spec:** Write bytes to a temp file in the same directory, flush, then atomically replace destination.
* **Constraints:** Use `GetTempFileNameW`/`CreateFileW`/`WriteFile`/`FlushFileBuffers` + `ReplaceFileW` (fallback `MoveFileExW` as needed).
* **Signature:**

```rust
pub fn atomic_write(path: &Path, data: &[u8]) -> Result<()>;
```

* **Example:**

```rust
atomic_write(Path::new(r"C:\tmp\config.json"), br#"{"x":1}"#)?;
```
