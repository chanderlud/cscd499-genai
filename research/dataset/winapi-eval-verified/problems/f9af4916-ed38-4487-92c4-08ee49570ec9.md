38) Compute allocation granularity and align offsets

* **Spec:** Return `granularity` and aligned mapping base for a requested offset.
* **Constraints:** Use `GetSystemInfo`; alignment is non-negotiable.
* **Signature:**

```rust
pub fn align_mapping_offset(offset: u64) -> Result<(u64 /*aligned*/, u64 /*delta*/, u32 /*granularity*/)>;
```

* **Example:**

```rust
let (base, delta, g) = align_mapping_offset(1_234_567)?;
```
