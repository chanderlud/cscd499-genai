48) WinRT “store”: LocalSettings (ApplicationData)

* **Spec:** Store/read a small string key/value setting in app local settings via `ApplicationData::Current().LocalSettings().Values()`.
* **Constraints:** values in `LocalSettings.Values` are `IInspectable`; this wrapper only writes and reads string values. `local_settings_get` returns `Ok(None)` only when the key is absent. If the key exists but is not a string, return an error. Assume an environment where `ApplicationData::Current()` is available.
* **Signature:**

```rust
pub fn local_settings_set(key: &str, value: &str) -> Result<()>;
pub fn local_settings_get(key: &str) -> Result<Option<String>>;
```

* **Example:**

```rust
local_settings_set("theme", "dark")?;
let v = local_settings_get("theme")?;
```
