18. Resolve final path via handle (normalize weirdness)

* **Spec:** Open a file and return its final NT-style or DOS-style path (your choice, but be consistent).
* **Constraints:** Use `CreateFileW` + `GetFinalPathNameByHandleW`. Handle long-path prefixes correctly.
* **Signature:**

```rust
pub fn final_path(path: &Path) -> Result<PathBuf>;
```

* **Example:**

```rust
let p = final_path(Path::new(r"C:\tmp\a.txt"))?;
```
