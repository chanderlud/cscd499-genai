0. Named Pipe: Uppercase Echo (message mode)

* **Spec:** Create a named pipe server and client (same process is fine, use threads). Client sends a UTF-8 string. Server reads it, converts to ASCII uppercase (only `a-z`), and replies. Return the server’s reply.
* **Constraints:**

    * Use Win32 named pipes (`CreateNamedPipeW`, `ConnectNamedPipe`, `CreateFileW`, `ReadFile`, `WriteFile`).
    * Use **message mode** (`PIPE_TYPE_MESSAGE` + `PIPE_READMODE_MESSAGE`).
    * Must not deadlock. Use a bounded timeout strategy (e.g., overlapped I/O or separate threads with join timeout).
