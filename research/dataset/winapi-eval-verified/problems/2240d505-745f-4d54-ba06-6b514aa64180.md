9. Get stable file identity (Win32)

* **Spec:** Return a value that uniquely identifies a file on a volume (good for "same file?" checks).
* **Constraints:** Use `CreateFileW` + `GetFileInformationByHandleEx(FileIdInfo)` (i.e., `FILE_ID_INFO`).
* **Signature:**

```rust
pub fn file_id(path: &Path) -> Result<[u8; 16]>;
```

* **Example:**

```rust
let id = file_id(Path::new(r"C:\tmp\a.txt"))?;
```
