19. Known folder path (Shell API)

* **Spec:** Return the filesystem path for a known folder (e.g., LocalAppData).
* **Constraints:** Use `SHGetKnownFolderPath` and free with `CoTaskMemFree`.
* **Signature:**

```rust
pub fn known_folder(id: windows::core::GUID) -> Result<PathBuf>;
```

* **Example:**

```rust
use windows::Win32::UI::Shell::FOLDERID_LocalAppData;
let p = known_folder(FOLDERID_LocalAppData)?;
```
