3. WinHTTP client against your own Winsock HTTP server

* **Spec:** Start a minimal HTTP/1.1 server on 127.0.0.1 that serves one GET with a provided body; fetch it using WinHTTP and return the response body.
* **Constraints:** Server: Winsock `accept`/`recv`/`send` (no HTTP.sys). Client: `WinHttpOpen/Connect/OpenRequest/SendRequest/ReceiveResponse/ReadData`.
* **Signature:**

```rust
pub fn winhttp_get_from_local_server(body: &[u8]) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let out = winhttp_get_from_local_server(b"ok")?;
```
