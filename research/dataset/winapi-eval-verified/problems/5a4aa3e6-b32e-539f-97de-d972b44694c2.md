## 06) Convert `std::io::Error` into `windows::core::Error`

* **Spec:** If `raw_os_error()` is present, convert it through `HRESULT::from_win32`. Otherwise return `E_FAIL`.
* **Constraints:** Do not branch on `ErrorKind`; use the raw OS code when available.
* **Signature:**

```rust
pub fn io_error_to_windows(err: std::io::Error) -> windows::core::Error;
```

* **Example:**

```rust
let io = std::io::Error::from_raw_os_error(5);
let win = io_error_to_windows(io);
assert_eq!(win.code(), windows::core::HRESULT::from_win32(5));
```
