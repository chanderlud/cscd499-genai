11. Expand environment variables (Win32)

* **Spec:** Expand `%VAR%` references in a string (e.g., `%TEMP%\a`).
* **Constraints:** Use `ExpandEnvironmentStringsW`.
* **Signature:**

```rust
pub fn expand_env(input: &str) -> Result<std::ffi::OsString>;
```

* **Example:**

```rust
let p = expand_env(r"%TEMP%\x.txt")?;
```
