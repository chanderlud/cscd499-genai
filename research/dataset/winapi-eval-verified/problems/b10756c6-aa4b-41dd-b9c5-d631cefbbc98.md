8. Set file timestamps (Win32)

* **Spec:** Set last-write time to a provided `FILETIME` (leave others unchanged).
* **Constraints:** Use `CreateFileW` + `SetFileTime`. Don’t call PowerShell, don’t use std metadata setters.
* **Signature:**

```rust
pub fn set_last_write_time(path: &Path, ft: windows::Win32::Foundation::FILETIME) -> Result<()>;
```

* **Example:**

```rust
set_last_write_time(Path::new(r"C:\tmp\a.txt"), some_filetime)?;
```
