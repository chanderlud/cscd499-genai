9. Choose between optional `PCWSTR` values

* **Spec:** Return the first non-empty string among `primary` and `fallback`. Treat `None`, a null pointer, or an empty string as missing. Invalid UTF-16 in the selected input should return an error.
* **Constraints:** Use `Option<PCWSTR>`; do not call Windows APIs.
* **Signature:**

```rust
use windows::core::{PCWSTR, Result};

pub fn choose_wide(primary: Option<PCWSTR>, fallback: Option<PCWSTR>) -> Result<String>;
```

* **Example:**

```rust
let p: Vec<u16> = "".encode_utf16().chain([0]).collect();
let f: Vec<u16> = "guest".encode_utf16().chain([0]).collect();

assert_eq!(
    choose_wide(
        Some(PCWSTR::from_raw(p.as_ptr())),
        Some(PCWSTR::from_raw(f.as_ptr()))
    )?,
    "guest"
);
```
