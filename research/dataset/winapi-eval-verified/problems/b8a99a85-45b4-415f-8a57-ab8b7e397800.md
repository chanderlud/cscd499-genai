14. Enumerate values under a registry key (Win32)

* **Spec:** Return all value names under a given HKCU key, sorted.
* **Constraints:** Use `RegOpenKeyExW` + `RegEnumValueW`.
* **Signature:**

```rust
pub fn reg_list_values_hkcu(path: &str) -> Result<Vec<String>>;
```

* **Example:**

```rust
let names = reg_list_values_hkcu(r"Software\MyApp\Tests")?;
```
