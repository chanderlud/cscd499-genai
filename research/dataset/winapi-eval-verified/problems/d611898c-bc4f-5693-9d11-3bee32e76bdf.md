## 01) Detect access denied

* **Spec:** Return `true` only when the error represents access denied.
* **Constraints:** Use `windows::core::Error::code()` and compare against the canonical HRESULT for `ERROR_ACCESS_DENIED`. Do not inspect error messages.
* **Signature:**

```rust
pub fn is_access_denied(err: &windows::core::Error) -> bool;
```

* **Example:**

```rust
use windows::core::{Error, HRESULT};
use windows::Win32::Foundation::ERROR_ACCESS_DENIED;

let err = Error::from_hresult(HRESULT::from_win32(ERROR_ACCESS_DENIED.0));
assert!(is_access_denied(&err));
```
