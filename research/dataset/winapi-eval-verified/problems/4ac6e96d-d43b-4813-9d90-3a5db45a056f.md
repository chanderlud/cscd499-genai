15. COM initialize guard (COM)

* **Spec:** Initialize COM on the current thread and uninitialize it on drop by returning an RAII guard. Treat every successful `CoInitializeEx` call as requiring a matching `CoUninitialize`.
* **Constraints:** Use `CoInitializeEx` + `CoUninitialize`. Support both MTA and STA via a parameter. Treat both `S_OK` and `S_FALSE` as success. Return an error for invalid flags or an incompatible threading model.
* **Signature:**

```rust
pub struct ComGuard { /* opaque */ }

pub fn com_init(coinit: u32) -> Result<ComGuard>;
```

* **Example:**

```rust
let _com = com_init(windows::Win32::System::Com::COINIT_MULTITHREADED.0)?;
```
