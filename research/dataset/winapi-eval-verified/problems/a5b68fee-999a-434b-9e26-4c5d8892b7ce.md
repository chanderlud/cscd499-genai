3. Compare `PCWSTR` to `&str`

* **Spec:** Return `true` when the contents of a null-terminated `PCWSTR` exactly match a Rust `&str`. A null pointer should compare equal only to the empty string.
* **Constraints:** Accept `PCWSTR`; compare contents up to the first terminator.
* **Signature:**

```rust
use windows::core::PCWSTR;

pub fn pcwstr_eq(ptr: PCWSTR, expected: &str) -> bool;
```

* **Example:**

```rust
let data: Vec<u16> = "rust".encode_utf16().chain([0]).collect();
assert!(pcwstr_eq(PCWSTR::from_raw(data.as_ptr()), "rust"));
```
