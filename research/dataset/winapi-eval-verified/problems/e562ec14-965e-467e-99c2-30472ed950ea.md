9. Registry change notification (event-driven)

* **Spec:** Wait for a key to change, then return once it does (or timeout).
* **Constraints:** Use `RegNotifyChangeKeyValue` with an event handle; no polling.
* **Signature:**

```rust
pub fn wait_for_reg_change_hkcu(path: &str, timeout_ms: u32) -> Result<bool>;
```

* **Example:**

```rust
let changed = wait_for_reg_change_hkcu(r"Software\MyApp\Tests", 2000)?;
```
