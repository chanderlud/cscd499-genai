12. Manual PE export resolver (forwarders included)

* **Spec:** Load a DLL without resolving imports, manually locate an exported symbol address by parsing the PE export table. If it’s a forwarder, resolve recursively (e.g., `KERNEL32!Foo` → `KERNELBASE!Foo`).
* **Constraints:** `LoadLibraryExW(DONT_RESOLVE_DLL_REFERENCES)`, parse `IMAGE_DOS_HEADER/NT_HEADERS/EXPORT_DIRECTORY`, handle forwarder strings, `FreeLibrary`.
* **Signature:**

```rust
pub fn resolve_export_manual(dll: &Path, symbol: &str) -> Result<usize>;
```

* **Example:**

```rust
let addr = resolve_export_manual(Path::new(r"C:\Windows\System32\kernel32.dll"), "GetCurrentProcessId")?;
```
