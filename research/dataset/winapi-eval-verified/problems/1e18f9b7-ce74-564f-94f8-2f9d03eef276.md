## 05) Convert a fallible probe into `Result<bool>`

* **Spec:** Interpret `Ok(())` as `Ok(true)`. Interpret `ERROR_FILE_NOT_FOUND` and `ERROR_PATH_NOT_FOUND` as `Ok(false)`. Propagate everything else.
* **Constraints:** Use canonical HRESULT comparison.
* **Signature:**

```rust
pub fn exists_from_result(result: windows::core::Result<()>) -> windows::core::Result<bool>;
```

* **Example:**

```rust
use windows::core::{Error, HRESULT};
use windows::Win32::Foundation::ERROR_FILE_NOT_FOUND;

let r = Err(Error::from_hresult(HRESULT::from_win32(ERROR_FILE_NOT_FOUND.0)));
assert_eq!(exists_from_result(r)?, false);
```
