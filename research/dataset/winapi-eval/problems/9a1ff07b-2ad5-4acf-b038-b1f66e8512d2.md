23. Parse display name to PIDL and back to IShellItem (COM)

* **Spec:** Convert a filesystem path to a PIDL, then to an `IShellItem`, then return its filesystem path.
* **Constraints:** Use `SHParseDisplayName`, `SHCreateItemFromIDList`, and free PIDL with `CoTaskMemFree`.
* **Signature:**

```rust
pub fn pidl_roundtrip(path: &Path) -> Result<PathBuf>;
```

* **Example:**

```rust
let p2 = pidl_roundtrip(Path::new(r"C:\tmp\a.txt"))?;
```
