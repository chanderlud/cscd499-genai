1. IOCP + Winsock TCP echo (AcceptEx + overlapped)

* **Spec:** Spin up a localhost TCP echo server using IOCP, connect a client, send `payload`, receive the echo, shut down cleanly.
* **Constraints:** Use `WSASocketW(WSA_FLAG_OVERLAPPED)`, `CreateIoCompletionPort`, `AcceptEx`, `ConnectEx`, `WSARecv`, `WSASend`, `GetQueuedCompletionStatus`. Fetch extension fns via `WSAIoctl(SIO_GET_EXTENSION_FUNCTION_POINTER)`.
* **Signature:**

```rust
pub fn iocp_tcp_echo(payload: &[u8]) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let out = iocp_tcp_echo(b"hello over iocp")?;
```
