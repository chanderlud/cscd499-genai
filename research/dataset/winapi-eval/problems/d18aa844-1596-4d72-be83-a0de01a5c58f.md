## 2) FLS Destructor Accounting

* **Spec:** Allocate an FLS slot with a destructor callback. Spawn `threads` threads; each thread sets a non-null value into the slot and exits. The destructor callback must run once per thread that set a value. Return the destructor call count.
* **Constraints:**

    * Use `FlsAlloc` with a real callback, `FlsSetValue`, `FlsFree`.
    * The callback must increment a shared `i32` using `InterlockedIncrement` (Win32), not Rust atomics.
    * Threads must be joined before returning.
