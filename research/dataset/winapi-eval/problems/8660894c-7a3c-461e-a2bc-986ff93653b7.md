7. Copy with progress + cancellation threshold

* **Spec:** Copy a file and report progress callbacks; cancel if bytes transferred exceeds `cancel_after` (if provided).
* **Constraints:** Use `CopyFileExW` and a progress routine (must handle continue/cancel).
* **Signature:**

```rust
pub fn copy_with_progress(src: &Path, dst: &Path, cancel_after: Option<u64>) -> Result<u64>;
```

* **Example:**

```rust
let transferred = copy_with_progress(Path::new(r"C:\tmp\a.bin"), Path::new(r"C:\tmp\b.bin"), Some(1024 * 1024))?;
```
