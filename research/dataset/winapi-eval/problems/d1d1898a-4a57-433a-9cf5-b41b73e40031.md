13. Vectored exception handler: “fault once, recover, continue”

* **Spec:** Allocate a page, write a value, mark it `PAGE_NOACCESS`, install a VEH that flips it to readable on access violation, then read the value successfully.
* **Constraints:** `VirtualAlloc`, `VirtualProtect`, `AddVectoredExceptionHandler`, `EXCEPTION_ACCESS_VIOLATION`, `RemoveVectoredExceptionHandler`.
* **Signature:**

```rust
pub fn veh_guarded_read_u32(value: u32) -> Result<u32>;
```

* **Example:**

```rust
let v = veh_guarded_read_u32(0xDEADBEEF)?;
```
