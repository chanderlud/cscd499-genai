16. Job object: kill process tree on timeout

* **Spec:** Launch a process in a job that is configured to kill the whole tree if it exceeds a time limit; return whether it was terminated by the job.
* **Constraints:** Use `CreateJobObjectW`, `SetInformationJobObject` (`JOB_OBJECT_LIMIT_KILL_ON_JOB_CLOSE` and/or time limits), `AssignProcessToJobObject`.
* **Signature:**

```rust
pub fn run_in_job_with_timeout(command_line: &str, timeout_ms: u32) -> Result<bool>;
```

* **Example:**

```rust
let killed = run_in_job_with_timeout(r#"cmd.exe /c timeout /t 5 /nobreak >nul"#, 500)?;
```
