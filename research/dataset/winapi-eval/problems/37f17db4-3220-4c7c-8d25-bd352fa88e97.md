5. Atomically replace file contents (Win32)

* **Spec:** Replace `dst` with `src` atomically (after the call, `dst` has `src`’s contents).
* **Constraints:** Use `ReplaceFileW` (allow “dst doesn’t exist” by falling back to `MoveFileExW`).
* **Signature:**

```rust
pub fn replace_file(src: &Path, dst: &Path) -> Result<()>;
```

* **Example:**

```rust
replace_file(Path::new(r"C:\tmp\new.cfg"), Path::new(r"C:\app\cfg.ini"))?;
```
