1. DPAPI encrypt/decrypt roundtrip

* **Spec:** Encrypt data for the current user and decrypt it back.
* **Constraints:** Use `CryptProtectData` / `CryptUnprotectData`. Support optional “entropy” bytes (may be empty).
* **Signature:**

```rust
pub fn dpapi_roundtrip(data: &[u8], entropy: &[u8]) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let out = dpapi_roundtrip(b"secret", b"pepper")?;
```
