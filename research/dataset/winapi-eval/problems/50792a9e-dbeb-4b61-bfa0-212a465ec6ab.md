7. Sparse file hole punching + allocation verification

* **Spec:** Create a sparse file with a large logical size, punch a hole (zero range), write a tiny tail block, then return `(end_of_file, allocation_size)`.
* **Constraints:** `DeviceIoControl(FSCTL_SET_SPARSE)`, `FSCTL_SET_ZERO_DATA`, `SetFilePointerEx`, `WriteFile`, `GetFileInformationByHandleEx(FileStandardInfo)`.
* **Signature:**

```rust
pub fn sparse_file_stats(path: &Path, logical_size: u64, hole_start: u64, hole_len: u64, tail: &[u8]) -> Result<(u64, u64)>;
```

* **Example:**

```rust
let (eof, alloc) = sparse_file_stats(Path::new(r"C:\tmp\s.bin"), 64<<20, 0, 63<<20, b"x")?;
```
