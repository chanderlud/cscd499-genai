17. Structured Storage: write + read a stream (COM)

* **Spec:** Create a compound file, write bytes to a named stream, reopen and read them back.
* **Constraints:** Use `StgCreateStorageEx` / `StgOpenStorageEx`, `IStorage::CreateStream`, `IStream`.
* **Signature:**

```rust
pub fn stg_stream_roundtrip(path: &Path, stream_name: &str, data: &[u8]) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let out = stg_stream_roundtrip(Path::new(r"C:\tmp\doc.stg"), "Data", b"hi")?;
```
