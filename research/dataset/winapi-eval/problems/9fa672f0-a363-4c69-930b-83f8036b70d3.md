## 9) Environment Block Parsing: GetEnvironmentStringsW

* **Spec:** Retrieve the process environment block via `GetEnvironmentStringsW`, parse it into key/value pairs, and return them. Ensure you free the block.
* **Constraints:**

    * Use `GetEnvironmentStringsW` and `FreeEnvironmentStringsW`.
    * Must parse the double-NUL terminated list of `KEY=VALUE\0 ... \0\0`.
    * Return entries as owned Rust `String`s (UTF-16 to UTF-8 conversion required).
