11. WinTrust Authenticode verification (offline-safe flags)

* **Spec:** Verify the Authenticode signature of a file and return success/failure as `Result<()>`.
* **Constraints:** `WinVerifyTrust` with `WINTRUST_ACTION_GENERIC_VERIFY_V2`, set `WTD_REVOKE_NONE` and `WTD_CACHE_ONLY_URL_RETRIEVAL` so tests don’t depend on network.
* **Signature:**

```rust
pub fn verify_authenticode(path: &Path) -> Result<()>;
```

* **Example:**

```rust
verify_authenticode(Path::new(r"C:\Windows\System32\kernel32.dll"))?;
```
