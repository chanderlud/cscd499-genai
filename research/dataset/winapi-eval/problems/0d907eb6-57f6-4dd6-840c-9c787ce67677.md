18. Shell COM stream binding + BCrypt SHA-256

* **Spec:** Open a file as an `IShellItem`, bind to `BHID_Stream` to get an `IStream`, stream it through BCrypt SHA-256, return digest.
* **Constraints:** `SHCreateItemFromParsingName`, `IShellItem::BindToHandler(BHID_Stream)`, `IStream::Read`, `BCrypt*` SHA-256.
* **Signature:**

```rust
pub fn shell_stream_sha256(path: &Path) -> Result<[u8; 32]>;
```

* **Example:**

```rust
let d = shell_stream_sha256(Path::new(r"C:\tmp\a.bin"))?;
```
