3. Append bytes (Win32)

* **Spec:** Append bytes to the end of an existing file (or create it).
* **Constraints:** Use `CreateFileW` with append semantics (`FILE_APPEND_DATA`) and `WriteFile`.
* **Signature:**

```rust
pub fn append_all(path: &Path, data: &[u8]) -> Result<()>;
```

* **Example:**

```rust
append_all(Path::new(r"C:\tmp\log.txt"), b"line\n")?;
```
