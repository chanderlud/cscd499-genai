## 1) Synchronization Barrier: Phased Counter

* **Spec:** Create a `SYNCHRONIZATION_BARRIER`. Spawn `threads` threads. Each thread runs `phases` phases; in each phase it increments a shared `i64` counter by 1, then waits at the barrier. After all threads complete, return the counter. The result must be `threads as i64 * phases`.
* **Constraints:**

    * Use `InitializeSynchronizationBarrier`, `EnterSynchronizationBarrier`, `DeleteSynchronizationBarrier`.
    * Shared counter must be updated using Win32 interlocked ops (`InterlockedIncrement64` or `InterlockedAdd64`), not Rust atomics.
    * No `std::sync::Barrier`.


* **Example:**

  ```rust
  assert_eq!(out, 4000);
  ```
