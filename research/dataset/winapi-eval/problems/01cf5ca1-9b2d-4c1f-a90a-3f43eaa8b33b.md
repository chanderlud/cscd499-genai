3. Named file mapping: write then read via a second mapping

* **Spec:** Create a named shared memory region, write bytes at an offset, open the same mapping again, read them back.
* **Constraints:** Use `CreateFileMappingW` + `OpenFileMappingW` + `MapViewOfFile` + `UnmapViewOfFile`.
* **Signature:**

```rust
pub fn named_mapping_write_read(name: &str, size: usize, offset: usize, data: &[u8]) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let out = named_mapping_write_read("Local\\my_map", 4096, 10, b"hello")?;
```
