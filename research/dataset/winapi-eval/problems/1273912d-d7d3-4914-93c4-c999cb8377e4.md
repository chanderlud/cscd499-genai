45) File Open dialog pick multiple files

* **Spec:** Allow selecting multiple files and return all paths.
* **Constraints:** Enable `FOS_ALLOWMULTISELECT`; handle empty selection as `None`.
* **Signature:**

```rust
pub fn pick_files(title: &str) -> Result<Option<Vec<std::path::PathBuf>>>;
```

* **Example:**

```rust
if let Some(ps) = pick_files("Pick files")? { println!("{}", ps.len()); }
```
