20. Property System: read PKEY_Size (COM)

* **Spec:** Return the file size using the Windows Property System.
* **Constraints:** Use `SHGetPropertyStoreFromParsingName` (or `PSGetPropertyStoreFromParsingName`) + `IPropertyStore::GetValue` with `PKEY_Size`.
* **Signature:**

```rust
pub fn property_size(path: &Path) -> Result<u64>;
```

* **Example:**

```rust
let sz = property_size(Path::new(r"C:\tmp\a.bin"))?;
```
