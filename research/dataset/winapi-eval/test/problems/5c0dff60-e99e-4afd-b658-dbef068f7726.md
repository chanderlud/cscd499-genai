* **Spec:** Implement a function `milliseconds_since_boot` that returns the number of milliseconds that have elapsed since Windows started by calling `GetTickCount64`. ([Microsoft Learn][1])
* **Constraints:** Use only the `windows` crate v0.62.2, call exactly one Win32 API method, and keep the behavior straightforward by returning the API value as `u64` without extra transformation. Since Microsoft documents `GetTickCount64` as a coarse timer with typical resolution around 10 to 16 ms, tests should only check that the value is non-decreasing and increases after a short sleep. ([Docs.rs][2])
* **Signature:**

  ```rust
  pub fn milliseconds_since_boot() -> u64;
  ```
* **Example:**

  ```rust
  let uptime_ms = milliseconds_since_boot();
  println!("Windows uptime: {uptime_ms} ms");
  ```
