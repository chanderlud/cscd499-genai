- **Spec:**
  - Implement a function `primary_screen_width_px` that returns the width of the primary display in pixels by calling the Win32 API `GetSystemMetrics` with `SM_CXSCREEN`.

- **Constraints:**
  - Use only the `windows` crate version `0.62.2`.
  - Make exactly one Win32 API call inside the function: `GetSystemMetrics`.
  - Do not use any extra crates.
  - Return the width as `i32`.
  - Assume the crate feature for `Win32_UI_WindowsAndMessaging` is enabled.

- **Signature:**
  ```rust
  pub fn primary_screen_width_px() -> i32
````

* **Example:**

  ```rust
  let width = primary_screen_width_px();
  assert!(width > 0);
  ```
