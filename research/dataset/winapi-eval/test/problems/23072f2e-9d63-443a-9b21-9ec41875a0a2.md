- **Spec:** Implement a function `rename_file` that renames or moves an existing file from `from` to `to` by calling the Windows API function `MoveFileW`.

- **Constraints:** Use only the `windows` crate v0.62.2. Call exactly one Windows API function: `MoveFileW`. Accept `&Path` parameters. Do not copy file contents manually or invoke shell commands. Return the Windows error if the move fails. The function should work with Unicode paths.

- **Signature:**
  ```rust
  pub fn rename_file(from: &std::path::Path, to: &std::path::Path) -> windows::core::Result<()>;
````

* **Example:**

  ```rust
  use std::path::Path;

  rename_file(
      Path::new(r"C:\temp\old_name.txt"),
      Path::new(r"C:\temp\new_name.txt"),
  )?;
  ```
