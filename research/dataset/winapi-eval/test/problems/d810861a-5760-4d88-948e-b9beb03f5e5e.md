2. Convert `PCWSTR` to `String`

* **Spec:** Read a null-terminated UTF-16 string from `PCWSTR` and return a Rust `String`. A null pointer should return an empty string. Invalid UTF-16 should return an error.
* **Constraints:** Use `String::from_utf16`; do not call Windows APIs.
* **Signature:**

```rust
use windows::core::{PCWSTR, Result};

pub fn pcwstr_to_string(ptr: PCWSTR) -> Result<String>;
```

* **Example:**

```rust
let data: Vec<u16> = "hi".encode_utf16().chain([0]).collect();
assert_eq!(pcwstr_to_string(PCWSTR::from_raw(data.as_ptr()))?, "hi");
```
