## 03) Check a `WIN32_ERROR` and return `Result<()>`

* **Spec:** Return `Ok(())` for success and `Err(windows::core::Error)` for any nonzero `WIN32_ERROR`.
* **Constraints:** Preserve the original code exactly through `HRESULT::from_win32`.
* **Signature:**

```rust
pub fn check_win32(code: windows::Win32::Foundation::WIN32_ERROR) -> windows::core::Result<()>;
```

* **Example:**

```rust
use windows::Win32::Foundation::{ERROR_ACCESS_DENIED, WIN32_ERROR};

check_win32(WIN32_ERROR(0))?;
assert!(check_win32(ERROR_ACCESS_DENIED).is_err());
```
