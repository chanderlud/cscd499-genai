- **Spec:** Implement `set_process_env_var` so it sets or overwrites an environment variable in the current process by calling the Win32 API function `SetEnvironmentVariableW`.

- **Constraints:**
  - Use only the `windows` crate v0.62.2 for the Windows API call.
  - Call exactly one Windows API function: `SetEnvironmentVariableW`.
  - Do not use `std::env::set_var`.
  - Return `windows::core::Result<()>`.
  - On success, the updated value must be observable through `std::env::var`.

- **Signature:**
  ```rust
  pub fn set_process_env_var(name: &str, value: &str) -> windows::core::Result<()>;
````

* **Example:**

  ```rust
  set_process_env_var("DEMO_FLAG", "enabled")?;
  assert_eq!(std::env::var("DEMO_FLAG")?, "enabled");
  # Ok::<(), Box<dyn std::error::Error>>(())
  ```
