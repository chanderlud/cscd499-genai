* **Spec:**

  * Implement a function `set_file_hidden` that marks an existing file as hidden by calling the Windows API function `SetFileAttributesW`.
  * The function should take a `&Path` and return `windows::core::Result<()>`.
  * You may assume the path already exists and points to a file.

* **Constraints:**

  * Use only the `windows` crate v0.62.2 for the Windows API call.
  * Make exactly one Windows API call inside the function: `SetFileAttributesW`.
  * Set the file's attributes to `FILE_ATTRIBUTE_HIDDEN`.
  * Do not preserve existing file attributes.
  * The function must be deterministically testable by observing the file's hidden attribute afterward.

* **Signature:**

  ```rust
  pub fn set_file_hidden(path: &std::path::Path) -> windows::core::Result<()>;
  ```

* **Example:**

  ```rust
  use std::path::Path;

  fn main() -> windows::core::Result<()> {
      set_file_hidden(Path::new(r"C:\temp\example.txt"))?;
      Ok(())
  }
  ```
