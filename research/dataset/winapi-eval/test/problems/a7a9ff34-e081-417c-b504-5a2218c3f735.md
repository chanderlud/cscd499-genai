- **Spec:** Implement a function `set_last_error_code` that sets the calling thread's Win32 last-error code to the provided value.

- **Constraints:**
  - Use only the `windows` crate v0.62.2.
  - Call exactly one Windows API function inside the solution.
  - The public function should take a plain `u32`.
  - Do not use any file, process, environment variable, or current-directory APIs.
  - The behavior must be deterministically testable with unit tests.

- **Signature:**
  ```rust
  pub fn set_last_error_code(code: u32)
  ```

- **Example:**
  ```rust
  use windows::Win32::Foundation::GetLastError;

  set_last_error_code(123);

  let actual = unsafe { GetLastError() };
  assert_eq!(actual.0, 123);
  ```
