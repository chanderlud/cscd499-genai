8. ConPTY spawn + capture output

* **Spec:** Spawn a console process attached to a pseudo console, capture its output, return bytes.
* **Constraints:** `CreatePseudoConsole`, pipes, `STARTUPINFOEXW`, attribute list with `PROC_THREAD_ATTRIBUTE_PSEUDOCONSOLE`, `ReadFile` from PTY output.
* **Signature:**

```rust
pub fn run_in_conpty(command_line: &str, timeout_ms: u32) -> Result<Vec<u8>, std::io::Error>;
```

* **Example:**

```rust
let out = run_in_conpty(r#"cmd.exe /c echo hello"#, 2000)?;
```
