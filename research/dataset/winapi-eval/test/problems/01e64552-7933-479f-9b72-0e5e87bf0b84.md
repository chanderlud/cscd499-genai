- **Spec:** Implement a function `current_system_time_100ns` that returns the current UTC system time as a `u64` containing the Windows `FILETIME` timestamp, measured in 100-nanosecond intervals since January 1, 1601 UTC.

- **Constraints:**
  - Use only the `windows` crate version `0.62.2`.
  - Call exactly one Win32 API method: `GetSystemTimePreciseAsFileTime`.
  - Do not use `std::time::SystemTime` to obtain the timestamp.
  - Convert the returned `FILETIME` value into a single `u64` using its high and low 32-bit parts.
  - The function must be deterministic enough to unit test by checking that time moves forward.

- **Signature:**
  ```rust
  pub fn current_system_time_100ns() -> u64
````

* **Example:**

  ```rust
  let t1 = current_system_time_100ns();
  std::thread::sleep(std::time::Duration::from_millis(5));
  let t2 = current_system_time_100ns();

  assert!(t2 > t1);
  ```
