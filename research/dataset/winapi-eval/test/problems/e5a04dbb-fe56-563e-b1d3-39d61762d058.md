## 04) Ignore “already exists”

* **Spec:** Normalize a `windows::core::Result<()>` so that `ERROR_ALREADY_EXISTS` and `ERROR_FILE_EXISTS` are treated as success.
* **Constraints:** Compare error codes, not strings. Preserve all other failures unchanged.
* **Signature:**

```rust
pub fn ok_if_already_exists(result: windows::core::Result<()>) -> windows::core::Result<()>;
```

* **Example:**

```rust
use windows::core::{Error, HRESULT};
use windows::Win32::Foundation::ERROR_ALREADY_EXISTS;

let r = Err(Error::from_hresult(HRESULT::from_win32(ERROR_ALREADY_EXISTS.0)));
assert!(ok_if_already_exists(r).is_ok());
```
