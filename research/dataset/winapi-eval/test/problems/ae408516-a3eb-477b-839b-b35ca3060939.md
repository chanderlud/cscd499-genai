- **Spec:** Implement a function that changes the current working directory of the current process to the directory given by `path`.

- **Constraints:**
  - Use the `windows` crate v0.62.2.
  - Call exactly one Win32 API function: `SetCurrentDirectoryW`.
  - Accept any path type that implements `AsRef<Path>`.
  - Convert the Rust path to a null-terminated UTF-16 string before calling the API.
  - Return `std::io::Result<()>`.
  - On failure, return `std::io::Error::last_os_error()`.
  - Assume the input path is intended to name a directory. The function does not need to validate it ahead of time.

- **Signature:**
  ```rust
  pub fn change_current_directory<P: AsRef<std::path::Path>>(path: P) -> std::io::Result<()>;
  ```

- **Example:**
  ```rust
  use std::fs;

  let dir = std::env::temp_dir().join("winapi_setcwd_example");
  fs::create_dir_all(&dir).unwrap();

  change_current_directory(&dir).unwrap();

  assert_eq!(
      fs::canonicalize(std::env::current_dir().unwrap()).unwrap(),
      fs::canonicalize(&dir).unwrap()
  );

  std::env::set_current_dir(std::env::temp_dir()).unwrap();
  fs::remove_dir_all(&dir).unwrap();
  ```
