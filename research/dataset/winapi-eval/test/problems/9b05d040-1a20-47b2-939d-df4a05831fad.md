6. Directory watcher (single event, timeout)

* **Spec:** Watch a directory and return when you observe “a file was created” (or time out).
* **Constraints:** Use `CreateFileW` on directory with `FILE_FLAG_BACKUP_SEMANTICS | FILE_FLAG_OVERLAPPED` + `ReadDirectoryChangesW`.
* **Signature:**

```text
pub fn wait_for_create(dir: &Path, timeout_ms: u32) -> Result<Option<std::ffi::OsString>>;
```

* **Example:**

```text
let name = wait_for_create(Path::new(r"C:\tmp\watch"), 2000)?;
```
