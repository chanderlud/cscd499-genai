4. Write UTF-16 text into a `PWSTR` buffer

* **Spec:** Copy `value` into a mutable UTF-16 buffer as a null-terminated string and return the number of UTF-16 code units written, excluding the terminator.
* **Constraints:** Use `PWSTR`; return an error if the buffer is null or too small; on success always write a terminator.
* **Signature:**

```rust
use windows::core::{PWSTR, Result};

pub fn write_wide(buf: PWSTR, cap: usize, value: &str) -> Result<usize>;
```

* **Example:**

```rust
let mut buf = vec![0u16; 8];
let n = write_wide(PWSTR::from_raw(buf.as_mut_ptr()), buf.len(), "cat")?;
assert_eq!(n, 3);
```
