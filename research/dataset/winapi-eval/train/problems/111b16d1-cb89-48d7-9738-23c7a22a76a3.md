5. Count bytes from `PCSTR`

* **Spec:** Return the number of bytes before the first `0u8` in a null-terminated `PCSTR`. A null pointer counts as `0`.
* **Constraints:** Accept `PCSTR`; do not allocate; treat it as raw bytes.
* **Signature:**

```rust
use windows::core::PCSTR;

pub fn pcstr_len(ptr: PCSTR) -> usize;
```

* **Example:**

```rust
let data = b"hello\0";
assert_eq!(pcstr_len(PCSTR::from_raw(data.as_ptr())), 5);
```
