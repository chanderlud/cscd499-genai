04) Append to file (FILE_APPEND_DATA)

* **Spec:** Append bytes to end of file efficiently on Windows using `FILE_APPEND_DATA`.
* **Constraints:** Open with `FILE_APPEND_DATA`; avoid racey manual seeks. Windows-only implementation.
* **Signature:**

```rust
pub fn append_all(path: &Path, data: &[u8]) -> Result<()>;
```

* **Example:**

```rust
append_all(Path::new(r"C:\tmp\log.txt"), b"line\n")?;
```
