## 10) Convert `HRESULT` into `Result<()>` without breaking `S_FALSE`

* **Spec:** Return `Ok(())` for any success HRESULT, including `S_FALSE`. Return `Err(Error)` for failures.
* **Constraints:** Use `HRESULT::is_ok()` or `HRESULT::is_err()`. Do not treat nonzero as failure.
* **Signature:**

```rust
pub fn hresult_to_result(hr: windows::core::HRESULT) -> windows::core::Result<()>;
```

* **Example:**

```rust
use windows::core::HRESULT;

hresult_to_result(HRESULT(0))?;
hresult_to_result(HRESULT(1))?;
```
