2. Write all bytes (overwrite) (Win32)

* **Spec:** Create or truncate a file and write all bytes.
* **Constraints:** Use `CreateFileW` + `WriteFile` (loop until all bytes written).
* **Signature:**

```rust
pub fn write_all(path: &Path, data: &[u8]) -> Result<()>;
```

* **Example:**

```rust
write_all(Path::new(r"C:\tmp\out.bin"), b"hello")?;
```
