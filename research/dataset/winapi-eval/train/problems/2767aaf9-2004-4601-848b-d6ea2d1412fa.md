10. Collect a slice of `PCWSTR` into `Vec<String>`

* **Spec:** Convert each null-terminated `PCWSTR` in the slice into an owned `String` and return them in order.
* **Constraints:** Use `&[PCWSTR]`; an empty slice should return an empty vector; null pointers or invalid UTF-16 inside the slice should return an error.
* **Signature:**

```rust
use windows::core::{PCWSTR, Result};

pub fn collect_wide_strings(values: &[PCWSTR]) -> Result<Vec<String>>;
```

* **Example:**

```rust
let a: Vec<u16> = "one".encode_utf16().chain([0]).collect();
let b: Vec<u16> = "two".encode_utf16().chain([0]).collect();

let out = collect_wide_strings(&[
    PCWSTR::from_raw(a.as_ptr()),
    PCWSTR::from_raw(b.as_ptr()),
])?;

assert_eq!(out, vec!["one", "two"]);
```
