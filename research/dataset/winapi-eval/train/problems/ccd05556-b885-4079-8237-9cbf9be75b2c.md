2. IOCP async file scatter-read (many in-flight reads)

* **Spec:** Read a file by issuing N overlapped reads at different offsets and collecting completions through an IOCP; return the exact file bytes in correct order.
* **Constraints:** `CreateFileW(FILE_FLAG_OVERLAPPED)`, `CreateIoCompletionPort`, multiple `ReadFile` with distinct `OVERLAPPED.Offset`, `GetQueuedCompletionStatus`.
* **Signature:**

```rust
pub fn read_file_iocp(path: &Path, chunk_size: u32, max_in_flight: u32) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let bytes = read_file_iocp(Path::new(r"C:\tmp\big.bin"), 64 * 1024, 8)?;
```
