3. Append bytes (Win32)

* **Spec:** Append bytes to the end of an existing file without truncating its current contents. If the file does not exist, create it. Appending an empty slice must leave existing contents unchanged; if the file does not exist, it should still be created as an empty file. Parent directories do not need to be created.
* **Constraints:** Use `CreateFileW` with append semantics (`FILE_APPEND_DATA`) and a creation disposition that creates the file if it does not exist (for example, `OPEN_ALWAYS`), then write with `WriteFile`.
* **Signature:**

```rust
pub fn append_all(path: &Path, data: &[u8]) -> Result<()>;
```

* **Example:**

```rust
append_all(Path::new(r"C:\tmp\log.txt"), b"line\n")?;
```
