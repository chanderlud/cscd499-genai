6. Write uppercase ASCII into a `PSTR` buffer

* **Spec:** Copy `value` into a mutable byte buffer as a null-terminated ASCII string, converting `a-z` to `A-Z`. Return the number of bytes written, excluding the terminator.
* **Constraints:** Use `PSTR`; reject non-ASCII input; reject insufficient buffer capacity.
* **Signature:**

```rust
use windows::core::{PSTR, Result};

pub fn write_ascii_upper(buf: PSTR, cap: usize, value: &str) -> Result<usize>;
```

* **Example:**

```rust
let mut buf = vec![0u8; 6];
write_ascii_upper(PSTR::from_raw(buf.as_mut_ptr()), buf.len(), "ab")?;
```
