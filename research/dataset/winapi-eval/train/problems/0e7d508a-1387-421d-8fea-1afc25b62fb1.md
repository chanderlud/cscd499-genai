23) Directory change notifications (ReadDirectoryChangesW)

* **Spec:** Watch a directory for create/delete/modify/rename. `recursive` controls whether subdirectories are included. Send each raw `ReadDirectoryChangesW` notification buffer through the returned receiver; each `Vec<u8>` may contain one or more `FILE_NOTIFY_INFORMATION` records.
* **Constraints:** Needs an open directory handle; `path` must exist and be a directory; overlapped I/O recommended; parse `FILE_NOTIFY_INFORMATION`.
* **Signature:**

```rust
pub fn watch_dir(path: &Path, recursive: bool) -> Result<std::sync::mpsc::Receiver<Vec<u8>>>;
```

* **Example:**

```rust
let rx = watch_dir(Path::new(r"C:\tmp"), true)?;
```
