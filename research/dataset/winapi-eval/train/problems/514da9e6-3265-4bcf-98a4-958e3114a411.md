10. Current user SID string (token → SID → string)

* **Spec:** Return the current process user SID as a string (e.g. `S-1-5-21-...`).
* **Constraints:** Use `OpenProcessToken` + `GetTokenInformation(TokenUser)` + `ConvertSidToStringSidW`.
* **Signature:**

```rust
pub fn current_user_sid_string() -> Result<String>;
```

* **Example:**

```rust
let sid = current_user_sid_string()?;
```
