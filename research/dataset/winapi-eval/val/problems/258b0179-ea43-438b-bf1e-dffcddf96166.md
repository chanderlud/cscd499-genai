8. Join path segments into an `HSTRING`

* **Spec:** Join non-empty path segments with `\` and return the result as an `HSTRING`. Trim leading and trailing `\` from each segment before joining.
* **Constraints:** Input type must be `&[HSTRING]`; ignore empty segments after trimming.
* **Signature:**

```rust
use windows::core::HSTRING;

pub fn join_hstrings(parts: &[HSTRING]) -> HSTRING;
```

* **Example:**

```rust
let parts = [HSTRING::from("C:\\"), HSTRING::from("\\tmp\\"), HSTRING::from("a")];
assert_eq!(join_hstrings(&parts).to_string_lossy(), r"C:\tmp\a");
```
