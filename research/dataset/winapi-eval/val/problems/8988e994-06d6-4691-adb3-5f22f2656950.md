21. Create an IShellItem and get filesystem path (COM)

* **Spec:** Create an `IShellItem` from a path and return its `SIGDN_FILESYSPATH`.
* **Constraints:** Use `SHCreateItemFromParsingName` + `IShellItem::GetDisplayName`.
* **Signature:**

```rust
pub fn shellitem_filesystem_path(path: &Path) -> Result<PathBuf>;
```

* **Example:**

```rust
let p2 = shellitem_filesystem_path(Path::new(r"C:\tmp\a.txt"))?;
```

* **Notes:**
  - The path must be an absolute path to an existing file or directory.
  - If the path is relative or does not exist, the function should return an error.
