10. Named mutex “already exists” detection (Win32)

* **Spec:** Create/open a named mutex and return whether this call created it.
* **Constraints:** Use `CreateMutexW` and check `GetLastError() == ERROR_ALREADY_EXISTS`.
* **Signature:**

```rust
pub fn create_named_mutex(name: &str) -> Result<(windows::Win32::Foundation::HANDLE, bool)>;
```

* **Example:**

```rust
let (_h, created) = create_named_mutex("Global\\my_mutex")?;
```
