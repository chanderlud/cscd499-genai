1. Count UTF-16 code units from `PCWSTR`

* **Spec:** Return the number of UTF-16 code units before the first terminating `0u16`. A null pointer counts as length `0`.
* **Constraints:** Accept `PCWSTR`; do not allocate; stop at the first terminator.
* **Signature:**

```rust
use windows::core::PCWSTR;

pub fn pcwstr_len(ptr: PCWSTR) -> usize;
```

* **Example:**

```rust
let data: Vec<u16> = "hello".encode_utf16().chain([0]).collect();
assert_eq!(pcwstr_len(PCWSTR::from_raw(data.as_ptr())), 5);
```
