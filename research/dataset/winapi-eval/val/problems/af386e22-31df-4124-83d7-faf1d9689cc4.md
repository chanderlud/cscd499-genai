14. Fibers: deterministic round-robin scheduler

* **Spec:** Create `n_fibers` fibers; each fiber appends its id to a shared log `iters` times, yielding each time. Return the log.
* **Constraints:** `ConvertThreadToFiber`, `CreateFiber`, `SwitchToFiber`, `DeleteFiber`.
* **Signature:**

```rust
pub fn fiber_round_robin(n_fibers: u32, iters: u32) -> Result<Vec<u32>>;
```

* **Example:**

```rust
let trace = fiber_round_robin(3, 2)?; // expects [0,1,2,0,1,2]
```
