9. Job object + completion port notifications

* **Spec:** Run a child process in a job configured to post job messages to an IOCP; collect job object message ids (specifically, JOB_OBJECT_MSG_NEW_PROCESS, JOB_OBJECT_MSG_EXIT_PROCESS, JOB_OBJECT_MSG_ABNORMAL_EXIT_PROCESS, and JOB_OBJECT_MSG_ACTIVE_PROCESS_ZERO) until the job becomes empty (i.e., all processes in the job have exited); return the exit code of the initial child process and the collected messages.
* **Constraints:** `CreateJobObjectW`, `CreateIoCompletionPort`, `SetInformationJobObject(JobObjectAssociateCompletionPortInformation)`, `AssignProcessToJobObject`, `GetQueuedCompletionStatus`.
* **Signature:**

```rust
pub fn run_in_job_collect_messages(command_line: &str, timeout_ms: u32) -> Result<(u32, Vec<u32>)>;
```

* **Example:**

```rust
let (code, msgs) = run_in_job_collect_messages(r#"cmd.exe /c exit 7"#, 2000)?;
```
