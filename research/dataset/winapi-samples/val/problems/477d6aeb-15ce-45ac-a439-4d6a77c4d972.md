**Spec:** Write a function that sets the progress bar on the Windows taskbar for a given window to a specified fraction (numerator and denominator). The fraction must be clamped between 0 and 1 (inclusive) and then converted to a percentage (0 to 100) for the progress bar.

**Constraints:**
- Must use the `ITaskbarList3` COM interface from the Windows API
- Must initialize COM with apartment-threaded model before creating the taskbar list instance (caller's responsibility)
- The fraction must be clamped between 0 and 1 (inclusive)
- Must set progress state to normal before setting the value
- The function must convert the fraction to a percentage (0 to 100) and set the progress bar accordingly
- Must handle the case where denominator is zero by returning an error

**Signature:**
```rust
fn set_progress_bar_fraction(hwnd: HWND, numerator: u64, denominator: u64) -> Result<()>
```

**Example:**
```rust
// Assuming COM is already initialized and hwnd is a valid window handle
set_progress_bar_fraction(hwnd, 1, 2)?; // sets to 50%
set_progress_bar_fraction(hwnd, 3, 0)?; // returns an error
```
