**Spec:**
- Get the foreground window handle using `GetForegroundWindow`.
- Obtain the IME context for that window using `ImmGetContext`.
- Retrieve the composition string using `ImmGetCompositionStringW` with `GCS_COMPSTR`.
- Return the composition string as a Rust `String` (or an empty string if no composition is active).

**Constraints:**
- Must handle the case where no composition is active (return empty string).
- Must handle the case where there is no foreground window or no IME context (return empty string).
- Must correctly convert the IME's wide-character (UTF-16) buffer to a Rust `String`.
- Use `windows::core::Result<T>` for error handling when `ImmGetCompositionStringW` fails.

**Signature:**
```rust
fn get_foreground_ime_composition_string() -> windows::core::Result<String>
```

**Example:**
```rust
let composition = get_foreground_ime_composition_string()?;
println!("Current IME composition: {}", composition);
```
