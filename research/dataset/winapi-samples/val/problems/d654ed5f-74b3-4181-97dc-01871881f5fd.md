N) Set Current Process Priority to Below Normal

**Spec:** Write a function that sets the current process's priority class to below normal using the Windows API.

**Constraints:**
- Use the `windows` crate.
- Use `GetCurrentProcess` to obtain the current process handle.
- Use `SetPriorityClass` to set the priority class to `BELOW_NORMAL_PRIORITY_CLASS`.
- The function must return a `Result<()>` and propagate errors.

**Signature:**
```rust
fn set_current_process_priority_below_normal() -> Result<()>;
```

**Example:**
```rust
use windows::core::Result;

fn main() -> Result<()> {
    set_current_process_priority_below_normal()?;
    println!("Process priority set to below normal.");
    Ok(())
}
```
