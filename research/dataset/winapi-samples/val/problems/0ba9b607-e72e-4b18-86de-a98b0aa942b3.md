**Spec:** Implement a function that sets the expiration time for a toast notification, causing it to automatically remove from the Action Center after the specified duration.
**Constraints:**
- Must use `windows::Foundation::DateTime` to represent the expiration time
- Expiration time must be calculated as current time plus the specified minutes
- Must handle conversion from minutes to the WinRT DateTime format (100-nanosecond intervals since January 1, 1601 UTC)
- Function should work with an existing `ToastNotification` instance
**Signature:**
```rust
fn set_notification_expiration(
    notification: &ToastNotification,
    minutes_from_now: u32
) -> windows::core::Result<()>
```
**Example:**
```rust
let toast = ToastNotification::CreateToastNotification(&xml_document)?;
set_notification_expiration(&toast, 60)?; // Expires in 1 hour
toast_notifier.Show(&toast)?;
```
