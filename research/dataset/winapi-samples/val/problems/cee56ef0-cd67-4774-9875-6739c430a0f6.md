**Spec:** Create/open a named mutex and return whether this call created it, without performing any heap allocation.
**Constraints:** 
- Use `CreateMutexW` and check `GetLastError() == ERROR_ALREADY_EXISTS`.
- The function must not allocate memory on the heap (no `Vec`, `String`, etc.).
- The mutex name must be at most 260 characters (wide) in length; return an error if the name is too long.
**Signature:**
```rust
pub fn create_named_mutex(name: &str) -> windows::core::Result<(windows::Win32::Foundation::HANDLE, bool)>;
```
**Example:**
```rust
let (_h, created) = create_named_mutex("Global\\my_mutex")?;
```
