1) Read a string value from the Windows registry

**Spec:** Write a function that opens a registry key and reads a string value (REG_SZ) by name, returning the string.

**Constraints:**
- Use the `windows` crate for Win32 API bindings
- Handle errors for missing keys/values and invalid data types
- Convert the UTF-16 registry data to a Rust String
- Use unsafe blocks only for the necessary FFI calls

**Signature:**
```rust
fn get_registry_string(hive: HKEY, path: &str, value_name: &str) -> Result<String>
```

**Example:**
```rust
use windows::Win32::System::Registry::HKEY_LOCAL_MACHINE;

let version = get_registry_string(
    HKEY_LOCAL_MACHINE,
    "SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion",
    "DisplayVersion",
)?;
println!("Windows version: {}", version);
```
