**Spec:** Write a Rust program that creates a window and displays an animated clock using Direct2D for rendering, Direct3D 11 for device creation, DXGI for swap chain management, and the UI Animation Manager for animating the clock hands when the window is first shown.

- The clock must display the current time with second, minute, and hour hands.
- The clock must have a shadow effect.
- The clock hands must be animated with a swing animation when the window is first shown.
- The program must handle window resizing and occlusion.
