1) Get Window DPI with Fallbacks

**Spec:** Write a function that returns the DPI (dots per inch) for a given window handle using a fallback chain of Win32 APIs.

**Constraints:**
- Must attempt `GetDpiForWindow` first (Windows 10 1607+)
- If that fails, fall back to `GetDpiForMonitor` (Windows 8.1+)
- If that fails, fall back to `GetDeviceCaps` with `LOGPIXELSX` (Vista+)
- Must properly manage and release the device context obtained via `GetDC`
- Should return the horizontal DPI value as a `u32`

**Signature:**
```rust
fn hwnd_dpi(hwnd: HWND) -> u32
```

**Example:**
```rust
let desktop_hwnd = HWND(std::ptr::null_mut()); // NULL HWND gets desktop window
let dpi = hwnd_dpi(desktop_hwnd);
println!("Desktop window DPI: {}", dpi);
```
