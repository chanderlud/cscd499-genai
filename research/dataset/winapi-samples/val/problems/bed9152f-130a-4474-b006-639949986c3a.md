1) Query Service Status

**Spec:** Write a function that queries the status of a Windows service by name and prints whether it is running, stopped, or in another state.

**Constraints:**
- Use the Windows Services API (OpenSCManagerW, OpenServiceW, QueryServiceStatus, CloseServiceHandle).
- The function must take a service name as a string and return a `Result<()>`.
- The function must open the Service Control Manager with SC_MANAGER_CONNECT and the service with SERVICE_QUERY_STATUS.
- The function must print the service state (running, stopped, or other) to the console.

**Signature:**
```rust
fn query_service_status(service_name: &str) -> Result<()>
```

**Example:**
```rust
fn main() -> Result<()> {
    query_service_status("EventLog")?;
    Ok(())
}
```
