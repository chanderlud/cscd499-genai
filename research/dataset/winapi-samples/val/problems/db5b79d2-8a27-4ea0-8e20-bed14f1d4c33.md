1) System Tray Icon

**Spec:** Create a system tray notification icon for a Windows application that displays a tooltip and responds to user interactions via a custom callback message.

**Constraints:**
- Use the `windows` crate and Win32 API functions (`Shell_NotifyIconW`, `RegisterClassW`, `CreateWindowExW`, etc.)
- Implement a builder pattern for configuring the notification icon (window handle, tooltip, icon, callback message)
- Handle the custom tray icon message in the window procedure
- Properly clean up by removing the tray icon when the application exits

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// Create window, load icon, then:
let tray_icon = NotifyIcon::new()
    .window_handle(hwnd)
    .tip("My Application")
    .icon(icon)
    .callback_message(WM_USER + 1);
tray_icon.notify_add()?;
// Run message loop, then:
tray_icon.notify_delete()?;
```
