1) Enumerate IPv6 TCP Connections and Owning Processes

**Spec:** Write a program that uses the Windows IP Helper API to retrieve and display all IPv6 TCP connections, including the local and remote socket addresses, the owning process ID, and the process name.

**Constraints:**
- Use `GetExtendedTcpTable` from the IP Helper API with `AF_INET6` and `TCP_TABLE_OWNER_PID_ALL`
- Handle buffer allocation by first calling with `None` to get required size
- Use `OpenProcess` with `PROCESS_QUERY_LIMITED_INFORMATION` and `QueryFullProcessImageNameW` to retrieve process names
- Extract only the filename from the full process image path
- Print each connection in the format: `local_addr -> remote_addr (PID: pid, Process: name)`
- Handle errors gracefully, returning early on API failures

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// Running the program might output:
// Found 2 IPv6 TCP connections:
//   [::1]:12345 -> [::1]:80 (PID: 1234, Process: firefox.exe)
//   [fe80::1]:54321 -> [fe80::2]:443 (PID: 5678, Process: chrome.exe)
```
