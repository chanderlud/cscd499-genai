**Spec:** Write a function that returns the DPI (dots per inch) for a given window handle using a fallback chain of Win32 APIs, returning a `Result` type and exposing the fallback chain as an iterator of closures.

**Constraints:**
- Must return `windows::core::Result<u32>` instead of panicking on failure
- Must implement the fallback chain as an iterator of `FnOnce() -> Option<u32>` closures
- The function should iterate through the fallbacks, returning the first successful DPI value
- If all fallbacks fail, return an appropriate `windows::core::Error`
- Must properly manage and release the device context obtained via `GetDC`
- Should return the horizontal DPI value as a `u32`

**Signature:**
```rust
fn hwnd_dpi(hwnd: HWND) -> windows::core::Result<u32>
```

**Example:**
```rust
let desktop_hwnd = HWND(std::ptr::null_mut()); // NULL HWND gets desktop window
let dpi = hwnd_dpi(desktop_hwnd)?;
println!("Desktop window DPI: {}", dpi);
```
