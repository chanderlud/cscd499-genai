**Spec:** Verify the Authenticode signature of a file using `WinVerifyTrust` with offline-safe flags, but without any heap allocation in the implementation.
**Constraints:** 
- No heap allocation (no `Vec`, `Box`, `String`, etc.) in the function body.
- Use stack-allocated buffers for wide string conversion.
- Path length must not exceed `MAX_PATH` (260 characters).
- Use `WTD_REVOKE_NONE` and `WTD_CACHE_ONLY_URL_RETRIEVAL` flags.
**Signature:**
```rust
pub fn verify_authenticode(path: &Path) -> windows::core::Result<()>;
```
**Example:**
```rust
verify_authenticode(Path::new(r"C:\Windows\System32\kernel32.dll"))?;
```
