**Spec:** Given an HRESULT, if it is a FACILITY_WIN32 value, return the underlying WIN32_ERROR. Otherwise return None. The function must be const.
**Constraints:**
- The function must be `const fn`.
- Do not hardcode individual error values.
- Inspect the HRESULT layout.
**Signature:**
```rust
pub const fn try_as_win32(hr: HRESULT) -> Option<WIN32_ERROR>;
```
**Example:**
```rust
use windows::core::HRESULT;
use windows::Win32::Foundation::ERROR_ACCESS_DENIED;

let hr = HRESULT::from_win32(ERROR_ACCESS_DENIED.0);
assert_eq!(try_as_win32(hr), Some(ERROR_ACCESS_DENIED));
```
