N) Determine Key Location from Scancode

**Spec:** Write a function that converts a keyboard scancode to its corresponding key location (Left, Right, Numpad, or Standard) using the current keyboard layout.

**Constraints:**
- Must use `MapVirtualKeyExW` with `MAPVK_VSC_TO_VK_EX` to convert scancode to virtual key
- Must handle extended scancodes (0xE000 bit set) which affect key interpretation
- Must distinguish between left/right modifier keys (Shift, Ctrl, Alt, Win)
- Must differentiate between numpad and standard versions of navigation keys (arrows, Home, End, etc.)
- Must treat extended Enter key as Numpad, non-extended as Standard

**Signature:**
```rust
fn get_key_location_from_scancode(scancode: u16, hkl: HKL) -> KeyLocation
```

**Example:**
```rust
let hkl = unsafe { GetKeyboardLayout(0) };
let location = get_key_location_from_scancode(0xE01C, hkl); // Extended Enter
assert_eq!(location, KeyLocation::Numpad);
```
