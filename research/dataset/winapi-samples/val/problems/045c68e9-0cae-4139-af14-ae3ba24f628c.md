**Spec:** Create/open a named mutex and return whether this call created it, without performing any heap allocation.
**Constraints:** 
- Use `CreateMutexW` and check `GetLastError() == ERROR_ALREADY_EXISTS`
- Must not allocate memory on the heap (no `Vec`, `String`, `Box`, etc.)
- The mutex name must be converted to a wide string using only stack-allocated buffers of 260 characters (including null terminator). If the name is too long, return an error.
**Signature:**
```rust
pub fn create_named_mutex_no_alloc(name: &str) -> windows::core::Result<(windows::Win32::Foundation::HANDLE, bool)>;
```
**Example:**
```rust
let (_h, created) = create_named_mutex_no_alloc("Global\\my_mutex")?;
```
