**Spec:** Create/open a named mutex and return whether this call created it, without performing any heap allocation. Returns an error if the mutex name exceeds 260 characters.
**Constraints:** Use `CreateMutexW` and check `GetLastError() == ERROR_ALREADY_EXISTS`. The mutex name must be converted to a wide string using only stack-allocated buffers. The maximum mutex name length is 260 characters (MAX_PATH). If the name is longer than 260 characters, the function returns an error with `ERROR_INVALID_NAME`.
**Signature:**
```rust
pub fn create_named_mutex(name: &str) -> windows::core::Result<(windows::Win32::Foundation::HANDLE, bool)>;
```
**Example:**
```rust
let (_h, created) = create_named_mutex("Global\\my_mutex")?;
```
