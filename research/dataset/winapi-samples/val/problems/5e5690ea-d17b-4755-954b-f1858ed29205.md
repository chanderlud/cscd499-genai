**Spec:** Write a program that conditionally displays message boxes based on user interaction. First, show a message box asking "Continue to next message?" with Yes/No buttons. If the user clicks Yes, display a second message box using the wide-character API. If the user clicks No, display a third message box using the ANSI API. All message boxes must use appropriate string macros and return their result codes.

**Constraints:**
- Use the `windows` crate.
- Use `MessageBoxW` for the conditional message box and the second message box.
- Use `MessageBoxA` for the third message box.
- Use `w!` macro for wide strings and `s!` macro for ANSI strings.
- Handle the return value from the first message box to determine which branch to execute.
- The program must be unsafe because it calls into the Windows API.

**Signature:**
```rust
fn main()
```

**Example:**
```rust
use windows::{core::*, Win32::UI::WindowsAndMessaging::*};

fn main() {
    unsafe {
        let result = MessageBoxW(
            None,
            w!("Continue to next message?"),
            w!("Question"),
            MB_YESNO,
        );
        if result == IDYES {
            MessageBoxW(None, w!("You chose to continue"), w!("Wide Message"), MB_OK);
        } else {
            MessageBoxA(None, s!("You chose to stop"), s!("Ansi Message"), MB_OK);
        }
    }
}
```
