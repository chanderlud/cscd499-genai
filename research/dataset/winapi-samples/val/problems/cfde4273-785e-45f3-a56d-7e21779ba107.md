N) Get Client Rectangle in Screen Coordinates

**Spec:** Write a function that returns the client rectangle of a given window in screen coordinates.

**Constraints:**
- Use `GetClientRect` to obtain the client area dimensions.
- Use `ClientToScreen` to convert the client origin (0,0) to screen coordinates.
- Adjust the rectangle by the screen coordinates of the top-left corner.
- Return a `Result<RECT>`.

**Signature:**
```rust
fn get_client_rect(hwnd: HWND) -> Result<RECT>
```

**Example:**
```rust
let desktop = unsafe { GetDesktopWindow() };
let rect = get_client_rect(desktop)?;
println!("Desktop client rectangle: {:?}", rect);
```
