**Spec:** Implement a function that generates an ECDH key pair on the P-256 curve using the Windows CNG API and returns the raw public key coordinates (x, y) and private key scalar as separate byte vectors.

**Constraints:**  
- Use `BCryptOpenAlgorithmProvider` with `BCRYPT_ECDH_P256_ALGORITHM`  
- Use `BCryptGenerateKeyPair` and `BCryptFinalizeKeyPair`  
- Export via `BCryptExportKey` with `BCRYPT_ECCPUBLIC_BLOB` and `BCRYPT_ECCPRIVATE_BLOB`  
- Must properly clean up all handles using `BCryptDestroyKey` and `BCryptCloseAlgorithmProvider`  
- Parse the exported blobs to extract the raw key material (x, y coordinates and private scalar)  
- Return the raw key material without any additional formatting

**Signature:**
```rust
fn generate_ecdh_p256_keypair() -> windows::core::Result<(Vec<u8>, Vec<u8>, Vec<u8>)>
```

**Example:**
```rust
let (x, y, private_scalar) = generate_ecdh_p256_keypair()?;
// x and y are 32-byte coordinates for P-256 public key
// private_scalar is 32-byte private key scalar
assert_eq!(x.len(), 32);
assert_eq!(y.len(), 32);
assert_eq!(private_scalar.len(), 32);
```
