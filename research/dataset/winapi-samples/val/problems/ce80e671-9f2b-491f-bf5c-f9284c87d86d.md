1) List Process Privileges

**Spec:** Write a function that retrieves and prints the names of all privileges held by the current process.

**Constraints:**
- Use the Win32 API via the `windows` crate
- Must use `OpenProcessToken`, `GetTokenInformation`, and `LookupPrivilegeNameW`
- Must handle dynamic buffer allocation for token information
- Must convert privilege LUIDs to human-readable names

**Signature:**
```rust
fn list_process_privileges() -> Result<()>
```

**Example:**
```rust
list_process_privileges()?;
// Prints privilege names like:
// SeDebugPrivilege
// SeChangeNotifyPrivilege
// SeShutdownPrivilege
// ...
```
