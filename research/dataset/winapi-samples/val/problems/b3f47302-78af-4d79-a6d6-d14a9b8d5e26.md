**Spec:** Given two slices of errors, return the index of the first pair (by position in each slice) that represent the same underlying Win32 error code. Only Windows errors with HRESULT values where the lower 15 bits equal 7 (indicating FACILITY_WIN32) are considered. The Win32 error code is extracted from the lower 16 bits of the HRESULT and compared to the raw OS error code from the IO error.
**Constraints:** 
- Must run in O(n*m) time in the worst case (where n and m are slice lengths)
- Must not allocate heap memory (no `Vec`, `String`, etc.)
- Must use only a fixed amount of stack memory
- Must break early on first match
**Signature:**
```rust
pub fn find_matching_error_pair(
    win_errors: &[windows::core::Error],
    io_errors: &[std::io::Error],
) -> Option<(usize, usize)>;
```
**Example:**
```rust
use windows::core::{Error, HRESULT};
use windows::Win32::Foundation::ERROR_FILE_NOT_FOUND;

let win_errors = [
    Error::from_hresult(HRESULT::from_win32(7)), // ERROR_INVALID_FUNCTION
    Error::from_hresult(HRESULT::from_win32(ERROR_FILE_NOT_FOUND.0)),
];
let io_errors = [
    std::io::Error::from_raw_os_error(ERROR_FILE_NOT_FOUND.0 as i32),
    std::io::Error::from_raw_os_error(7), // ERROR_INVALID_FUNCTION
];

assert_eq!(find_matching_error_pair(&win_errors, &io_errors), Some((0, 1)));
```
