**Spec:** Implement a function that encrypts data using AES in CBC mode by dynamically loading the required BCrypt functions from bcrypt.dll. Define a `delay_load` helper function to load each BCrypt function.

**Constraints:**
- Define a `delay_load` function with the signature: `fn delay_load(dll_name: PCWSTR, func_name: PCSTR) -> windows::core::Result<*const std::ffi::c_void>`
- You must load the following functions from bcrypt.dll:
  - BCryptOpenAlgorithmProvider
  - BCryptSetProperty
  - BCryptGetProperty
  - BCryptGenerateSymmetricKey
  - BCryptEncrypt
  - BCryptCloseAlgorithmProvider
  - BCryptDestroyKey
- Define the function pointer types for each of these functions according to the Windows API.
- Use the loaded function pointers to perform the encryption.
- Properly manage the BCrypt handles (close the algorithm provider and destroy the key).
- Return `windows::core::Result<Vec<u8>>` containing the ciphertext.

**Signature:**
```rust
fn aes_cbc_encrypt_dynamic(key: &[u8; 16], iv: &[u8; 16], plaintext: &[u8]) -> windows::core::Result<Vec<u8>>
```

**Example:**
```rust
let key = [0x00, 0x01, 0x02, 0x03, 0x04, 0x05, 0x06, 0x07,
           0x08, 0x09, 0x0A, 0x0B, 0x0C, 0x0D, 0x0E, 0x0F];
let iv = [0x00, 0x01, 0x02, 0x03, 0x04, 0x05, 0x06, 0x07,
          0x08, 0x09, 0x0A, 0x0B, 0x0C, 0x0D, 0x0E, 0x0F];
let plaintext = b"Hello, World!";
let ciphertext = aes_cbc_encrypt_dynamic(&key, &iv, plaintext)?;
assert!(!ciphertext.is_empty());
```
