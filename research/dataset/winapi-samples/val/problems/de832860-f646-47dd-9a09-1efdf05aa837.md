1) Primary Screen Size

**Spec:** Write a function that returns the width and height of the primary display monitor in pixels.

**Constraints:**
- Use the Win32 `GetSystemMetrics` function with `SM_CXSCREEN` and `SM_CYSCREEN` constants.
- Return an error if either dimension is zero.
- The function must be safe to call from safe Rust code.

**Signature:**
```rust
fn get_primary_screen_size() -> Result<(i32, i32)>
```

**Example:**
```rust
let (width, height) = get_primary_screen_size()?;
println!("Primary screen size: {}x{}", width, height);
```
