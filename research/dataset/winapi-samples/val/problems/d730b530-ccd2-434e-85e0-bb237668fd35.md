**Spec:**
- Write a function that queries the Windows CNG API to retrieve a list of all symmetric encryption algorithms that are FIPS-approved on the current system.
- The function should return a vector of algorithm names (as strings) only if the system is running in FIPS mode; otherwise, return an empty vector.
- Use the `BCryptEnumAlgorithms` function with the `BCRYPT_CIPHER_OPERATION` flag to enumerate symmetric encryption algorithms.
- For each algorithm returned, check its properties to verify it is FIPS-approved (using the `BCRYPT_FIPS_ALGORITHM_FLAG`).

**Constraints:**
- Must call `BCryptGetFipsAlgorithmMode` first to verify FIPS mode is enabled before enumeration.
- Must properly free the algorithm enumeration buffer using `BCryptFreeBuffer`.
- Must handle all Windows API errors and return a `windows::core::Result` with appropriate error propagation.
- The returned algorithm names should be extracted from the `BCRYPT_ALGORITHM_IDENTIFIER` structures.

**Signature:**
```rust
fn get_fips_approved_symmetric_algorithms() -> windows::core::Result<Vec<String>>
```

**Example:**
```rust
let algorithms = get_fips_approved_symmetric_algorithms()?;
if algorithms.is_empty() {
    println!("System not in FIPS mode or no FIPS-approved symmetric algorithms found");
} else {
    for algo in algorithms {
        println!("FIPS-approved symmetric algorithm: {}", algo);
    }
}
```
