**Spec:** Return `true` when both represent the same underlying OS failure.
**Constraints:** Compare canonical HRESULTs. If the `std::io::Error` has no `raw_os_error()`, return `false`. Do not use `HRESULT::from_win32` or any other conversion function from the `windows` crate. Instead, perform the Win32-to-HRESULT conversion manually using the formula: `(value & 0xFFFF) | (7 << 16) | 0x80000000` for positive values.
**Signature:**
```rust
pub fn same_os_error(win: &windows::core::Error, io: &std::io::Error) -> bool;
```
**Example:**
```rust
use windows::core::{Error, HRESULT};
use windows::Win32::Foundation::ERROR_ACCESS_DENIED;

let win = Error::from_hresult(HRESULT::from_win32(ERROR_ACCESS_DENIED.0));
let io = std::io::Error::from_raw_os_error(ERROR_ACCESS_DENIED.0 as i32);

assert!(same_os_error(&win, &io));
```
