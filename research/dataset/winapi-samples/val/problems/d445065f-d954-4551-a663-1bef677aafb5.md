N) Delay Load Function

**Spec:** Implement a generic function that dynamically loads a Windows DLL and retrieves a function pointer by name, returning it as a typed function pointer.

**Constraints:**
- Use `LoadLibraryExA` with `LOAD_LIBRARY_SEARCH_DEFAULT_DIRS` to load the library
- Use `GetProcAddress` to obtain the function address
- Return `None` if library loading fails or function is not found
- Free the library with `FreeLibrary` only when the function is not found
- The returned function pointer must be safely transmuted to the requested type

**Signature:**
```rust
pub unsafe fn delay_load<T>(library: PCSTR, function: PCSTR) -> Option<T>
```

**Example:**
```rust
use windows::{core::*, Win32::System::LibraryLoader::*};

type ShellMessageBoxW = unsafe extern "C" fn(usize, usize, PCWSTR, PCWSTR, u32, ...) -> i32;

fn main() {
    unsafe {
        if let Some(api) = delay_load::<ShellMessageBoxW>(s!("shlwapi.dll"), s!("ShellMessageBoxW")) {
            api(0, 0, w!("Message"), w!("Sample"), 1);
        }
    }
}
```
