Disable EcoQoS for a Process

**Spec:** Write a function that disables EcoQoS (Efficiency Quality of Service) for a process given its process ID (PID).

**Constraints:**
- Must use the Win32 API `SetProcessInformation` with the `ProcessPowerThrottling` information class
- Must open the process with `PROCESS_SET_INFORMATION` and `PROCESS_QUERY_LIMITED_INFORMATION` access rights
- Must properly manage the process handle (close it when done)
- Must use the `windows` crate for Win32 bindings

**Signature:**
```rust
pub fn disable_ecoqos_for_process(pid: u32) -> windows::core::Result<()>
```

**Example:**
```rust
// Disable EcoQoS for the current process
let pid = unsafe { GetCurrentProcessId() };
disable_ecoqos_for_process(pid)?;

// Disable EcoQoS for a specific process (by PID)
disable_ecoqos_for_process(1234)?;
```
