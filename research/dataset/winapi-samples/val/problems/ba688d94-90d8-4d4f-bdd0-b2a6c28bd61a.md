Create and Display a Popup Menu

**Spec:** Write a function that creates a popup menu with three items (Exit, About, Settings) and displays it at the current cursor position. Return the selected menu option as an enum, or None if the user cancels.

**Constraints:**
- Use Win32 API functions for menu creation and tracking
- Display the menu at the current cursor position
- Clean up menu resources properly
- Use the `windows` crate for Win32 bindings

**Signature:**
```rust
pub fn show_menu(hwnd: HWND) -> Result<Option<MenuOptions>>
```

**Example:**
```rust
let hwnd = HWND::default();
if let Some(selection) = show_menu(hwnd)? {
    match selection {
        MenuOptions::Exit => println!("Exit selected"),
        MenuOptions::About => println!("About selected"),
        MenuOptions::Settings => println!("Settings selected"),
    }
}
```
