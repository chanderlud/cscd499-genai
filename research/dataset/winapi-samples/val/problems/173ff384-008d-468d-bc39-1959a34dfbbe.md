**Spec:** Write a function that takes a WinRT `IDirect3DDevice` and creates a 2D texture with the given dimensions and format, then returns the texture as a WinRT `IDirect3DSurface`.
**Constraints:** 
- Use the provided `get_d3d_interface_from_object` helper to obtain the underlying `ID3D11Device`.
- Create the texture with `D3D11_USAGE_DEFAULT` and `D3D11_BIND_SHADER_RESOURCE` flags.
- Convert the resulting `ID3D11Texture2D` to an `IDXGISurface`, then to a WinRT `IDirect3DSurface` using `CreateDirect3D11SurfaceFromDXGISurface`.
- Return the surface on success.
**Signature:**
```rust
pub fn create_texture_surface(
    device: &IDirect3DDevice,
    width: u32,
    height: u32,
    format: DXGI_FORMAT,
) -> windows::core::Result<IDirect3DSurface>
```
**Example:**
```rust
let d3d_device = create_d3d_device()?;
let direct3d_device = create_direct3d_device(&d3d_device)?;
let surface = create_texture_surface(&direct3d_device, 800, 600, DXGI_FORMAT_B8G8R8A8_UNORM)?;
```
