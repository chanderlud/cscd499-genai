**Spec:** Create a hard link at `link_path` pointing to `existing_file` using `CreateHardLinkW`. Fail if the target is a directory.
**Constraints:** 
- Do not allocate heap memory (no `Vec`, `String`, etc.) for the wide string conversions.
- Use only stack-allocated buffers with a maximum size of `MAX_PATH` (260 characters) for each path.
- Return an error if either path exceeds the buffer capacity.
**Signature:**
```rust
pub fn create_hard_link(link_path: &Path, existing_file: &Path) -> windows::core::Result<()>;
```
**Example:**
```rust
create_hard_link(Path::new(r"C:\tmp\alias.txt"), Path::new(r"C:\tmp\real.txt"))?;
```
