1) Display a Popup Menu with Exit Option

**Spec:** Create a function that displays a popup menu with a single "Exit" option at the current cursor position, waits for user selection, and returns the chosen option or an error.

**Constraints:**
- Use Win32 `CreatePopupMenu`, `InsertMenuW`, `TrackPopupMenu`, and related functions via the `windows` crate
- Implement RAII wrapper for the menu handle to ensure proper cleanup
- Set foreground window and send `WM_INITMENUPOPUP` before tracking
- Return `Ok(Some(MenuOptions::Exit))` if Exit selected, `Ok(None)` if cancelled, or `Err` on failure

**Signature:**
```rust
pub fn show_menu(hwnd: HWND) -> Result<Option<MenuOptions>>
```

**Example:**
```rust
let hwnd = /* valid window handle */;
match show_menu(hwnd) {
    Ok(Some(MenuOptions::Exit)) => println!("Exit selected"),
    Ok(None) => println!("Menu cancelled"),
    Err(e) => eprintln!("Error: {}", e),
}
```
