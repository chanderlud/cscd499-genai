1) Parse PE Import Table

**Spec:** Build a map of statically imported functions and their IAT addresses by parsing the current module's PE import table.

**Constraints:**
- Use the `windows` crate for Win32 API bindings
- Validate DOS and NT headers before parsing
- Return a `Result<HashMap<String, usize>>` mapping function names to IAT entry addresses
- Handle modules with no imports gracefully

**Signature:**
```rust
unsafe fn build_import_map() -> Result<HashMap<String, usize>>
```

**Example:**
```rust
let import_map = unsafe { build_import_map()? };
println!("Found {} imports", import_map.len());
for (name, addr) in import_map.iter().take(5) {
    println!("{}: 0x{:08x}", name, addr);
}
```
