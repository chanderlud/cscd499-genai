1) Set Taskbar Progress

**Spec:** Write a function that sets the progress bar on the Windows taskbar for a given window to a specified percentage value.

**Constraints:**
- Must use the `ITaskbarList3` COM interface from the Windows API
- Must initialize COM with apartment-threaded model before creating the taskbar list instance
- Progress value must be clamped between 0 and 100
- Must set progress state to normal before setting the value

**Signature:**
```rust
fn set_progress_bar(hwnd: HWND, value: u32) -> Result<()>
```

**Example:**
```rust
// Assuming COM is already initialized and hwnd is a valid window handle
set_progress_bar(hwnd, 50)?;
```
