**Spec:** Implement a function that encrypts data using AES in CBC mode via the Windows BCrypt API. The function should take a 128-bit key, 128-bit initialization vector (IV), and plaintext bytes, then return the ciphertext.

**Constraints:**
- Use `BCryptOpenAlgorithmProvider` with `BCRYPT_AES_ALGORITHM`
- Set the chaining mode to `BCRYPT_CHAIN_MODE_CBC` via `BCryptSetProperty`
- Generate a symmetric key with `BCryptGenerateSymmetricKey`
- Encrypt using `BCryptEncrypt` with `BCRYPT_BLOCK_PADDING` flag
- Properly manage all BCrypt handles with RAII wrappers
- Return `windows::core::Result<Vec<u8>>` containing the ciphertext

**Signature:**
```rust
fn aes_cbc_encrypt(key: &[u8; 16], iv: &[u8; 16], plaintext: &[u8]) -> windows::core::Result<Vec<u8>>
```

**Example:**
```rust
let key = [0x00, 0x01, 0x02, 0x03, 0x04, 0x05, 0x06, 0x07,
           0x08, 0x09, 0x0A, 0x0B, 0x0C, 0x0D, 0x0E, 0x0F];
let iv = [0x00, 0x01, 0x02, 0x03, 0x04, 0x05, 0x06, 0x07,
          0x08, 0x09, 0x0A, 0x0B, 0x0C, 0x0D, 0x0E, 0x0F];
let plaintext = b"Hello, World!";
let ciphertext = aes_cbc_encrypt(&key, &iv, plaintext)?;
assert!(!ciphertext.is_empty());
```
