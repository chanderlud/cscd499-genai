1) Get the Red Color
**Spec:** Write a function that retrieves the red color from the `Windows.UI.Colors` class and prints it in debug format.
**Constraints:** Use the `windows` crate. The function must return a `windows::core::Result<()>`.
**Signature:**
```rust
fn main() -> windows::core::Result<()>
```
**Example:**
```rust
fn main() -> windows::core::Result<()> {
    use windows::UI::Colors;

    let red = Colors::Red()?;
    println!("Red: {red:?}");

    Ok(())
}
```
