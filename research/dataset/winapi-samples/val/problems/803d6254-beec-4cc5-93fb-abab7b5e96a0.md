**Spec:** Create/open a named manual-reset event with initial state false and return whether this call created it, without performing any heap allocation.
**Constraints:**
- Use `CreateEventW` and check `GetLastError() == ERROR_ALREADY_EXISTS`.
- The function must not allocate memory on the heap (no `Vec`, `String`, etc.).
- The event name must be at most 260 characters (wide) in length; return an error if the name is too long.
- The event must be a manual-reset event with initial state false.
**Signature:**
```rust
pub fn create_named_event(name: &str) -> windows::core::Result<(windows::Win32::Foundation::HANDLE, bool)>;
```
**Example:**
```rust
let (_h, created) = create_named_event("Global\\my_event")?;
```
