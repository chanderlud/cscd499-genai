1) Display Message Boxes with Different APIs

**Spec:** Write a program that displays three message boxes using the Win32 API. The first must use the ANSI version of MessageBox, the second must use the wide-character version, and the third must use the ShellMessageBoxW function. Each message box must have a different title and text.

**Constraints:** 
- Use the `windows` crate.
- Use the `MessageBoxA`, `MessageBoxW`, and `ShellMessageBoxW` functions.
- Use the `s!`, `h!`, and `w!` macros for string literals as appropriate.
- The program must be unsafe because it calls into the Windows API.

**Signature:** 
```rust
fn main()
```

**Example:** 
```rust
use windows::{core::*, Win32::UI::Shell::*, Win32::UI::WindowsAndMessaging::*};

fn main() {
    unsafe {
        MessageBoxA(None, s!("Ansi"), s!("World"), MB_OK);
        MessageBoxW(None, h!("WinRT"), h!("World"), MB_OK);
        ShellMessageBoxW(None, None, w!("Wide"), w!("World"), MB_ICONERROR);
    }
}
```
