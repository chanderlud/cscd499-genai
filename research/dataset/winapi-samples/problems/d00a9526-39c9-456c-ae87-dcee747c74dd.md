1) Overlapped File Read
**Spec:** Write a function that uses Win32 overlapped I/O to read a specified number of bytes from a file at a given offset and returns the data as a string.
**Constraints:**
- Use the `windows` crate with Win32 APIs
- Open the file with `CreateFileA` using `FILE_FLAG_OVERLAPPED`
- Use `ReadFile` with an `OVERLAPPED` structure containing an event created by `CreateEventA`
- Wait for the operation to complete using `WaitForSingleObject` with a 2000ms timeout
- Use `GetOverlappedResult` to retrieve the number of bytes read
- Return an error if the read operation fails or if the byte count doesn't match the requested length
**Signature:**
```rust
fn async_read_file(path: &str, offset: u64, length: usize) -> Result<String>
```
**Example:**
```rust
let content = async_read_file("message.txt", 9, 12)?;
println!("{}", content);
```
