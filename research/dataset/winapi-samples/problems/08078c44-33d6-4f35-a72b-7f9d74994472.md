4) ListBox Selection Check Example

**Spec:** Create a Win32 window application that contains a multi-select ListBox. The ListBox should be populated with a list of strings. Then, programmatically select a few items and check the selection status of each item, printing the results to the console.

**Constraints:**
- Use the Win32 API to create the window and ListBox.
- The ListBox must have the styles: `LBS_MULTIPLESEL`, `LBS_HASSTRINGS`, and `LBS_NOTIFY`.
- Use `SendMessageW` with `LB_ADDSTRING` to add items.
- Use `SendMessageW` with `LB_SETSEL` to select items at specific indices.
- Use `SendMessageW` with `LB_GETSEL` to check the selection status of each item.

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
The program should output the selection status of each item, for example:
```
Checking selection status of ListBox items:
Item 'Apple' (index 0): SELECTED
Item 'Banana' (index 1): not selected
Item 'Cherry' (index 2): SELECTED
Item 'Date' (index 3): not selected
Item 'Elderberry' (index 4): SELECTED
```
