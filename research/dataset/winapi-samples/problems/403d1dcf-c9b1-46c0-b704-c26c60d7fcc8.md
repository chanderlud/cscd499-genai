N) List Unique IPv6 Addresses

**Spec:** Write a function that retrieves all unique IPv6 addresses from the system's network adapters using the Win32 `GetAdaptersAddresses` API, filtering out unspecified (::) and site-local (fec0::/10) addresses.

**Constraints:**
- Must use the `GetAdaptersAddresses` function from the `windows` crate's `Win32::NetworkManagement::IpHelper` module.
- Must handle the two-call pattern: first call to get required buffer size, second call with allocated buffer.
- Must iterate through the linked list of adapters and their DNS server addresses.
- Must filter out unspecified and site-local IPv6 addresses.
- Must return a deduplicated collection of IPv6 addresses.

**Signature:**
```rust
fn get_unique_ipv6_addresses() -> windows::core::Result<std::collections::HashSet<std::net::Ipv6Addr>>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    let addresses = get_unique_ipv6_addresses()?;
    for (i, addr) in addresses.iter().enumerate() {
        println!("{}. {}", i + 1, addr);
    }
    Ok(())
}
```
