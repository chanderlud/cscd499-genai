1) Get Direct3D9 Device from Fixed Address

**Spec:** Write an unsafe function that retrieves the `IDirect3DDevice9` interface pointer stored at a fixed memory address and returns it as an `IUnknown` COM interface.

**Constraints:**
- Must read the pointer from address `0xC97C28`
- Must cast the raw pointer to `IUnknown` using `windows::core::Interface::from_raw`
- Function must be marked `unsafe` due to raw pointer dereference
- Caller must guarantee the address contains a valid `IDirect3DDevice9` pointer in the current process context

**Signature:**
```rust
pub unsafe fn get_d3d9_device() -> IUnknown
```

**Example:**
```rust
let device = unsafe { get_d3d9_device() };
println!("Successfully obtained IDirect3DDevice9 interface");
```
