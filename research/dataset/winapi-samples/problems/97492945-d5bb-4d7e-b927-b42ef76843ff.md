N) Load and Destroy Icon

**Spec:** Write a function that loads an icon from a file path and returns its handle, then demonstrate proper cleanup by destroying the icon.

**Constraints:**
- Use `LoadImageW` with `LR_LOADFROMFILE` and `LR_DEFAULTSIZE` flags
- Convert the file path to a null-terminated wide string for the Win32 API
- Return a `Result<HICON>` to handle potential API errors
- Properly destroy the icon handle using `DestroyIcon` when done

**Signature:**
```rust
fn load_icon_from_path(path: &Path) -> Result<HICON>
```

**Example:**
```rust
fn main() -> Result<()> {
    let icon = load_icon_from_path(Path::new("example.ico"))?;
    // Use the icon handle here
    unsafe { DestroyIcon(icon)?; }
    Ok(())
}
```
