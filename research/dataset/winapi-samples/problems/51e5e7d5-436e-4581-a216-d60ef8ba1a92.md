1) Get Cursor Position

**Spec:** Write a function that returns the current cursor position in screen coordinates.

**Constraints:**
- Use the Win32 API function `GetCursorPos`.
- Handle errors appropriately using the `windows::core::Result` type.

**Signature:**
```rust
fn get_cursor_position() -> Result<POINT>
```

**Example:**
```rust
fn main() -> Result<()> {
    let point = get_cursor_position()?;
    println!("Cursor position: ({}, {})", point.x, point.y);
    Ok(())
}
```
