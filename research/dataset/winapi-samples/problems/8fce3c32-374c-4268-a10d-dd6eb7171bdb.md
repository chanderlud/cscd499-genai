N) Get Process Name from Window Handle

**Spec:** Write a function that takes a window handle (HWND) and returns the name of the process that owns the window as a String.

**Constraints:**
- Use the Windows API to get the process ID of the window
- Use the Toolhelp API (CreateToolhelp32Snapshot, Process32FirstW, Process32NextW) to enumerate processes
- The function must return an error if the process is not found or if any API call fails
- The function must handle the conversion of the process name from a null-terminated UTF-16 string to a Rust String

**Signature:**
```rust
fn get_process_name(target: HWND) -> Result<String>
```

**Example:**
```rust
use windows::Win32::Foundation::HWND;
use windows::Win32::UI::WindowsAndMessaging::GetDesktopWindow;

fn main() -> windows::core::Result<()> {
    let desktop = unsafe { GetDesktopWindow() };
    let name = get_process_name(desktop)?;
    println!("Desktop window process: {}", name);
    Ok(())
}
```
