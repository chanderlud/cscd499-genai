1) ListBox Clear and Repopulate with Storage Optimization

**Spec:** Create a Win32 window containing a list box control. Initially populate the list box with 5 items, then clear it and repopulate with 10 new items, using storage pre-allocation for efficiency.

**Constraints:** 
- Use the `windows` crate for Win32 bindings
- Create a main window with a child list box control (WS_CHILD | WS_VISIBLE | WS_VSCROLL | WS_BORDER | LBS_NOTIFY)
- Clear the list box using LB_RESETCONTENT before repopulating
- Pre-allocate storage for 10 items with LB_INITSTORAGE before adding new items
- Run a standard Win32 message loop

**Signature:** 
```rust
fn main() -> windows::core::Result<()>
```

**Example:** 
```rust
// After creating the list box control:
// Initial population
for i in 0..5 {
    let text = format!("Initial Item {}", i);
    SendMessageW(listbox, LB_ADDSTRING, None, Some(LPARAM(wide_null(&text).as_ptr() as isize)));
}

// Clear and repopulate with storage optimization
SendMessageW(listbox, LB_RESETCONTENT, None, None);
SendMessageW(listbox, LB_INITSTORAGE, Some(WPARAM(10)), Some(LPARAM(20 * 10)));
for i in 0..10 {
    let text = format!("Repopulated Item {}", i);
    SendMessageW(listbox, LB_ADDSTRING, None, Some(LPARAM(wide_null(&text).as_ptr() as isize)));
}
```
