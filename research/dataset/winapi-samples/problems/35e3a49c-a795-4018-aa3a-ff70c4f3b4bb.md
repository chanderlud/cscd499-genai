1) Listbox Selection with LB_SETSEL and LB_GETSEL

**Spec:** Write a function that creates a multi-selection listbox, adds five items, selects items 1 and 3, then deselects item 1, and prints the selection state of each item.

**Constraints:** 
- Use the `windows` crate
- Create a multi-selection listbox with `LBS_MULTIPLESEL` style
- Use `SendMessageW` with `LB_ADDSTRING` to add items
- Use `SendMessageW` with `LB_SETSEL` to select/deselect items by index
- Use `SendMessageW` with `LB_GETSEL` to query selection state

**Signature:**
```rust
fn demonstrate_listbox_selection(parent: HWND, instance: HINSTANCE) -> Result<()>
```

**Example:**
```rust
let hwnd = CreateWindowExW(...); // parent window
demonstrate_listbox_selection(hwnd, instance)?;
// Output:
// Item 0 selected: false
// Item 1 selected: true
// Item 2 selected: false
// Item 3 selected: true
// Item 4 selected: false
//
// After deselecting item 1:
// Item 0 selected: false
// Item 1 selected: false
// Item 2 selected: false
// Item 3 selected: true
// Item 4 selected: false
```
