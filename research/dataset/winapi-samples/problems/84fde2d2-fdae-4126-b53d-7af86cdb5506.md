Calculate DPI-aware window insets for non-client area rendering.

**Spec:** Implement a function that calculates the left, top, right, and bottom insets for a window's non-client area at a given DPI, using the appropriate system metrics for DPI-aware applications.

**Constraints:**
- Must use `GetSystemMetricsForDpi` when available (Windows 10 1607+), falling back to `GetSystemMetrics` for older systems
- Frame thickness should be calculated as `SM_CXSIZEFRAME + SM_CXPADDEDBORDER`
- For Windows 11 (build 22000+), the top inset requires an additional DPI-scaled adjustment
- The function must handle any valid DPI value

**Signature:**
```rust
fn calculate_insets_for_dpi(dpi: u32) -> RECT
```

**Example:**
```rust
let insets = calculate_insets_for_dpi(96);
assert_eq!(insets.left, 8);
assert_eq!(insets.right, 8);
assert_eq!(insets.bottom, 8);
```
