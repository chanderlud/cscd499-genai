1) Monitor Processor Time

**Spec:** Write a program that uses the PDH API to continuously monitor and print the processor time percentage for the first CPU core every second.

**Constraints:**
- Use the `windows` crate and the `Win32::System::Performance` module
- The program must run in an infinite loop, collecting data every second
- The counter path must be exactly `"\\Processor(0)\\% Processor Time"`
- Output must be formatted as a floating-point number with two decimal places
- Handle PDH API calls with proper error checking (only print when successful)

**Signature:**
```rust
fn main()
```

**Example:** The program will output processor time values every second:
```
12.34
15.67
8.91
```
