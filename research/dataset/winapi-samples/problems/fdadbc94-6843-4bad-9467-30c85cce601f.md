1) Check if a window is maximized

**Spec:** Write a function that determines whether a given window is currently in the maximized state using Win32 API.

**Constraints:**
- Use the `windows` crate and Win32 API functions
- Must use `GetWindowPlacement` to retrieve window state
- Return a `Result<bool>` to handle potential API errors
- The function should work with any valid window handle

**Signature:**
```rust
fn is_maximized(window: HWND) -> Result<bool>
```

**Example:**
```rust
use windows::Win32::UI::Input::KeyboardAndMouse::GetActiveWindow;

fn main() -> Result<()> {
    let window = unsafe { GetActiveWindow() };
    if window.is_invalid() {
        println!("No active window");
        return Ok(());
    }
    let maximized = is_maximized(window)?;
    println!("Active window is maximized: {}", maximized);
    Ok(())
}
```
