1) Text Manipulation in an Edit Control

**Spec:** Create a Win32 window application that creates an edit control, sets its text, and then retrieves and prints the text and its length.

**Constraints:**
- Use the `windows` crate for Win32 bindings
- The main window must be created with a window class and a window procedure
- In the window procedure, handle WM_CREATE to create the edit control and perform the text operations
- Use SetWindowTextW to set the text, GetWindowTextLengthW to get the length, and GetWindowTextW to get the text

**Signature:**
```rust
fn main() -> Result<()>
```

**Example:**
```rust
// When run, the program should output:
// Edit control text: Hello from SetWindowTextW!
// Text length: 25
```
