1) Desktop Window Rectangle

**Spec:** Write a function that retrieves the bounding rectangle of the desktop window (the full screen area) and returns it as a `RECT` structure.

**Constraints:**
- Use the `windows` crate (version 0.58.0 or later) and the Win32 API functions `GetDesktopWindow` and `GetWindowRect`.
- The function must handle errors by returning a `Result<RECT>`.
- The function must be named `get_desktop_window_rect`.

**Signature:**
```rust
fn get_desktop_window_rect() -> Result<RECT>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    let rect = get_desktop_window_rect()?;
    println!(
        "Desktop window rect: left={}, top={}, right={}, bottom={}",
        rect.left, rect.top, rect.right, rect.bottom
    );
    Ok(())
}
```
