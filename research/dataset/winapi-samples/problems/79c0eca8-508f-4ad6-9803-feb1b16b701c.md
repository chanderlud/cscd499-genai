N) Set and Retrieve Font in Edit Control
**Spec:** Create a window with an edit control, set a specific font (Arial, size 16, normal weight) to the edit control, and then retrieve the font to verify it was set successfully.
**Constraints:** Use the `windows` crate and Win32 API. The program must create a font with `CreateFontW`, set it via `SendMessageW` with `WM_SETFONT`, and retrieve it via `SendMessageW` with `WM_GETFONT`. The program must also handle the window message loop and clean up appropriately.
**Signature:** The program should be a complete Rust program with a `main` function that returns `Result<()>`.
**Example:** When run, the program displays a window with an edit control containing "Sample text" in the Arial font. The console outputs "Font successfully set and retrieved from edit control".
