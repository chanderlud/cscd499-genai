1) Create a Multi-Select ListBox and Print Selected Indices

**Spec:** Create a Win32 window containing a multi-select listbox. Populate the listbox with items, pre-select some items, and when the user clicks the window, print the indices of all currently selected items.

**Constraints:** Use the `windows` crate for Win32 bindings. The listbox must be created with `LBS_MULTIPLESEL` style. Handle `WM_LBUTTONUP` to retrieve selected indices using `LB_GETSELCOUNT` and `LB_GETSELITEMS` messages.

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// Creates window with listbox containing ["Apple", "Banana", "Cherry", "Date", "Elderberry"]
// Pre-selects indices 0, 2, and 4
// Clicking window prints: "Selected 3 items: 0, 2, 4"
```
