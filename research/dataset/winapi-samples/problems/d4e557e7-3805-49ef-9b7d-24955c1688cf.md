1) Vertically Center Text in a Static Label

**Spec:** Write a subclass procedure for a static label that vertically centers its text by adjusting the non-client area. The procedure must calculate the required client area height based on the label's font and text content, then modify the non-client area to center the text vertically within the label's window rectangle.

**Constraints:**
- Use the `windows` crate for Win32 API bindings
- The label must support multi-line text (newline-separated)
- Handle `WM_NCCALCSIZE` to adjust the client area for vertical centering
- Handle `WM_NCPAINT` to paint the non-client area with a specified background brush
- Handle `WM_SIZE` to trigger recalculation when the label is resized
- Use `SetWindowSubclass` to attach the subclass procedure

**Signature:**
```rust
unsafe extern "system" fn label_subclass_proc(
    hwnd: HWND,
    msg: u32,
    wparam: WPARAM,
    lparam: LPARAM,
    _uidsubclass: usize,
    dwrefdata: usize,
) -> LRESULT
```

**Example:**
```rust
// Create a static label and subclass it for vertical centering
let label = CreateWindowExW(
    WINDOW_EX_STYLE::default(),
    PCWSTR(wide_null("STATIC").as_ptr()),
    PCWSTR(wide_null("Line 1\nLine 2\nLine 3").as_ptr()),
    WINDOW_STYLE(WS_CHILD.0 | WS_VISIBLE.0 | BS_LEFT as u32),
    50, 50, 300, 200,
    Some(parent_hwnd),
    None,
    Some(HINSTANCE(instance.0)),
    None,
)?;

let brush = CreateSolidBrush(COLORREF(0x00FFFFFF)); // White background
SetWindowSubclass(label, Some(label_subclass_proc), 0, brush.0 as usize);
```
