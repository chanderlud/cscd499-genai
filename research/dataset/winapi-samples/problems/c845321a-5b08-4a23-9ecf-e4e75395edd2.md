1) Parse URI with COM

**Spec:** Write a Rust program that initializes COM, parses a URI string using the Windows COM API, and prints the domain and port components.

**Constraints:**
- Must use the `windows` crate and the `windows::Win32::System::Com` module
- Must initialize COM with `CoInitialize` before parsing and uninitialize with `CoUninitialize` after
- Must use `CreateUri` with default flags to parse the URI
- Must extract domain and port using `GetDomain()` and `GetPort()` methods
- Must handle errors using `windows::core::Result` and the `?` operator

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// For URI "http://kennykerr.ca", the program should output:
// kennykerr.ca (80)
```
