N) ListBox Selection Count

**Spec:** Create a window containing a multi-select list box populated with fruit names, programmatically select specific items, then display the count of selected items in a message box.

**Constraints:**
- Use Win32 API via the `windows` crate
- List box must support multiple selection (`LBS_MULTIPLESEL`)
- Populate list with: "Apple", "Banana", "Cherry", "Date", "Elderberry"
- Programmatically select items at indices 0, 2, and 4
- Display selection count via `MessageBoxW` before entering message loop
- Handle `WM_CREATE` and `WM_DESTROY` messages appropriately

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// Creates window with list box, selects 3 items, shows "Number of selected items: 3"
main()?;
```
