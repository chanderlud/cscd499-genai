1) ListBox Item Count

**Spec:** Create a ListBox control as a child of a given window, add a list of string items to it, and return the total number of items in the ListBox.

**Constraints:**
- Use the Win32 API via the `windows` crate.
- The ListBox must be created with the style `WS_CHILD | WS_VISIBLE | WS_VSCROLL | WS_BORDER`.
- The ListBox must be created at position (10, 10) with width 200 and height 200.
- The function must add each string in the `items` slice to the ListBox using the `LB_ADDSTRING` message.
- The function must return the count of items in the ListBox using the `LB_GETCOUNT` message.

**Signature:**
```rust
fn create_listbox_with_items(parent: HWND, items: &[&str]) -> Result<u32>
```

**Example:**
```rust
let count = create_listbox_with_items(hwnd, &["Apple", "Banana", "Cherry"])?;
println!("Number of items: {}", count);
```
