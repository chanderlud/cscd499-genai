1) Task Dialog with Custom Button and Callback

**Spec:** Create a function that displays a Windows task dialog with a custom button using command links, allows dialog cancellation, and handles button click events via a callback function that prints a message when any button is clicked.

**Constraints:**
- Use the `windows` crate and `Win32::UI::Controls` module
- Configure the dialog with `TDF_USE_COMMAND_LINKS` and `TDF_ALLOW_DIALOG_CANCELLATION` flags
- Implement a callback function that prints "button clicked" on `TDN_BUTTON_CLICKED` notifications
- Return the selected button ID from the dialog

**Signature:**
```rust
fn show_task_dialog(
    window_title: &str,
    main_instruction: &str,
    content: &str,
    button_text: &str,
    button_id: i32,
) -> windows::core::Result<i32>
```

**Example:**
```rust
let selection = show_task_dialog(
    "Window title",
    "Main instruction",
    "Content",
    "Let's do it",
    123,
)?;
if selection == 123 {
    println!("custom button");
}
```
