1) Button Text Manipulation

**Spec:** Write a function that retrieves the current text of a button control, then sets it to a new string, returning the original text.

**Constraints:** Use the `windows` crate and Win32 API. The function must handle errors and use `SendMessageW` for text operations. Use the provided `wide_null` helper for string conversion.

**Signature:**
```rust
fn get_set_button_text(hwnd: HWND, new_text: &str) -> Result<String>
```

**Example:**
```rust
fn wide_null(s: &str) -> Vec<u16> {
    use std::iter::once;
    s.encode_utf16().chain(once(0)).collect()
}

let old_text = get_set_button_text(button_hwnd, "Updated Text")?;
println!("Old text: {}", old_text);
```
