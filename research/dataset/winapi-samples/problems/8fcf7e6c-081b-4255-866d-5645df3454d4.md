1) Send EM_UNDO to an edit control

**Spec:** Write a function that sends the EM_UNDO message to a specified edit control window.

**Constraints:** Use the Windows API function `SendMessageW` with the `EM_UNDO` message constant. The function must be unsafe and take a window handle as its parameter.

**Signature:**
```rust
unsafe fn send_em_undo(hwnd: HWND)
```

**Example:**
```rust
let edit_hwnd: HWND = ...; // obtained from CreateWindowExW or elsewhere
unsafe { send_em_undo(edit_hwnd) };
```
