1) Create an edit control with center text alignment

**Spec:** Create a window and an edit control (a text input field) with centered text alignment.

**Constraints:**
- Use the Win32 API via the `windows` crate
- The edit control must be a child of the main window
- The edit control must have the `ES_CENTER` style to center the text
- The program must run a standard message loop and handle the `WM_DESTROY` message

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
The program should display a window with an edit control that has the text "Centered Text" centered.
