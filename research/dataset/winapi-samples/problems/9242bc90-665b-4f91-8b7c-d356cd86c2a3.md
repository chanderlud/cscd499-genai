1) Check if a process is running by name

**Spec:** Write a function that checks if a process with a given name is currently running by enumerating processes using the ToolHelp API.

**Constraints:**
- Use `CreateToolhelp32Snapshot` with `TH32CS_SNAPPROCESS` to get a process snapshot
- Use `Process32FirstW` and `Process32NextW` to iterate through processes
- Perform case-insensitive comparison of process names (ASCII)
- Properly close the snapshot handle with `CloseHandle`
- Return `Result<bool>` indicating whether the process was found

**Signature:**
```rust
fn is_process_running(process_name: &str) -> Result<bool>
```

**Example:**
```rust
fn main() -> Result<()> {
    if is_process_running("explorer.exe")? {
        println!("explorer.exe is running");
    }
    Ok(())
}
```
