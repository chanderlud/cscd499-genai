N) Get and Set Child Window Position and Size

**Spec:** Write a function that retrieves a child window's current position and size relative to its parent's client area, sets a new position and size, and verifies the change by printing the updated coordinates.

**Constraints:**
- Use `GetWindowRect` to obtain the child window's screen coordinates.
- Use `ScreenToClient` to convert screen coordinates to the parent's client coordinates.
- Use `SetWindowPos` to update the child window's position and size.
- Print both the initial and updated position/size in the parent's client coordinate system.

**Signature:**
```rust
fn update_child_window_position(
    parent_hwnd: windows::Win32::Foundation::HWND,
    child_hwnd: windows::Win32::Foundation::HWND,
    new_x: i32,
    new_y: i32,
    new_width: i32,
    new_height: i32,
) -> windows::core::Result<()>
```

**Example:**
```rust
// Assuming parent_hwnd and child_hwnd are valid window handles
update_child_window_position(parent_hwnd, child_hwnd, 100, 80, 150, 40)?;
// Output:
// Initial button position: (50, 50), size: (120, 30)
// Set button to position: (100, 80), size: (150, 40)
// New button position: (100, 80), size: (150, 40)
```
