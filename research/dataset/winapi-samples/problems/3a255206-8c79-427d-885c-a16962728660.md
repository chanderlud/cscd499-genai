1) Set Process DPI Aware

**Spec:** Write a function that sets the current process as DPI aware by calling the Win32 API `SetProcessDPIAware`.

**Constraints:**
- Use the `windows` crate to call the Win32 API
- The function should be safe to call (handle unsafe internally)
- Return a `Result<()>` to propagate any errors from the API call

**Signature:**
```rust
fn set_process_dpi_aware() -> windows::core::Result<()>
```

**Example:**
```rust
use windows::core::Result;

fn main() -> Result<()> {
    set_process_dpi_aware()?;
    println!("Process set as DPI aware");
    Ok(())
}
```
