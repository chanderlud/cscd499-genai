1) Insert String into ListBox

**Spec:** Write a function that inserts a string into a list box control at a specified index using Win32 API messages.

**Constraints:**
- Must use `SendMessageW` with `LB_INSERTSTRING` message
- Must convert Rust string to null-terminated UTF-16 for Win32 API
- Must handle error cases where insertion fails (negative return value)
- Index parameter: `usize::MAX` means append to end, otherwise insert at specified position

**Signature:**
```rust
unsafe fn insert_string_into_listbox(hwnd: HWND, index: usize, text: &str) -> Result<()>
```

**Example:**
```rust
// Insert "Third" at index 2 in existing list box
unsafe {
    insert_string_into_listbox(listbox_hwnd, 2, "Third")?;
}
```
