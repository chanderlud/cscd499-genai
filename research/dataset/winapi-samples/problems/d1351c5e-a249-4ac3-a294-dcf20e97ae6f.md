1) Create an Icon from RGBA Pixel Data

**Spec:** Implement a function that creates a Win32 icon from raw RGBA pixel data using the `CreateIcon` API.

**Constraints:**
- Input pixel data is in RGBA format (4 bytes per pixel: red, green, blue, alpha)
- Must convert RGBA to BGRA format for the icon's color data
- Must generate an AND mask by inverting the alpha channel values
- Must use the Win32 `CreateIcon` function from the `windows` crate
- Must properly handle icon creation errors via `Result`

**Signature:**
```rust
fn create_icon_from_rgba(rgba: Vec<u8>, width: u32, height: u32) -> Result<HICON>
```

**Example:**
```rust
let rgba_data = vec![
    255, 0, 0, 255,  // Red, fully opaque
    255, 0, 0, 128,  // Red, half transparent
];
let icon = create_icon_from_rgba(rgba_data, 2, 1)?;
// Use the icon handle...
unsafe { DestroyIcon(icon)? };
```
