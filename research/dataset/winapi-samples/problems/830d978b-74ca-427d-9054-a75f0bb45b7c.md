1) Enumerate Top-Level Windows

**Spec:** Write a function that enumerates all top-level windows and prints the handle and title of each window that is visible, enabled, not cloaked by the Desktop Window Manager (DWM), and does not have the extended window styles WS_EX_APPWINDOW or WS_EX_NOACTIVATE.

**Constraints:**
- Use the `windows` crate for Win32 bindings.
- The function must handle errors from Win32 API calls appropriately.
- The function must be safe to call (i.e., encapsulate unsafe operations).

**Signature:**
```rust
pub fn enumerate_windows() -> windows::core::Result<()>
```

**Example:**
```rust
fn main() {
    if let Err(e) = enumerate_windows() {
        eprintln!("Error: {:?}", e);
    }
}
```
