1) Get Screen Resolution

**Spec:** Write a function that retrieves the current screen resolution (width and height) in pixels using the Win32 API.

**Constraints:**
- Use the `windows` crate and the `GetSystemMetrics` function with `SM_CXSCREEN` and `SM_CYSCREEN`.
- The function must return a `Result` containing a tuple of two `i32` values (width, height).
- If the system metrics are negative (which indicates an error), return an error using `windows::core::Error::from_thread()`.

**Signature:**
```rust
fn get_screen_resolution() -> Result<(i32, i32)>
```

**Example:**
```rust
use windows::core::Result;

fn main() -> Result<()> {
    let (width, height) = get_screen_resolution()?;
    println!("Display resolution: {} x {}", width, height);
    Ok(())
}
```
