1) Get Instance Handle via ImageBase

**Spec:** Write a function that returns the instance handle (HMODULE) of the current module by taking the address of the linker-provided `__ImageBase` symbol.

**Constraints:**
- Must use the `windows` crate and the `HMODULE` type from `windows::Win32::Foundation`
- Must declare the external `__ImageBase` symbol of type `IMAGE_DOS_HEADER`
- The function must be safe to call (unsafe block is allowed for the pointer cast)
- Should work in both executables and DLLs

**Signature:**
```rust
pub fn get_instance_handle() -> HMODULE
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    let instance = get_instance_handle();
    println!("Instance handle: {:?}", instance);
    Ok(())
}
```
