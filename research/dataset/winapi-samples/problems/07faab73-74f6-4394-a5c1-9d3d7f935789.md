1) Set Button Font

**Spec:** Write a function that creates a custom font and applies it to a button control using Win32 API calls.

**Constraints:**
- Use the `windows` crate for Win32 bindings
- Create a bold Arial font with height 24
- Apply the font to the button via `WM_SETFONT` message
- Handle string conversion to wide (UTF-16) format

**Signature:**
```rust
fn set_button_font(button_hwnd: HWND) -> Result<()>
```

**Example:**
```rust
let button_hwnd = /* obtain button HWND */;
set_button_font(button_hwnd)?;
// Button now displays with bold Arial font, size 24
```
