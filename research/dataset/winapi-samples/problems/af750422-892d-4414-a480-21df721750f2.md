Dynamic Ordinal Lookup

**Spec:** Write a function that loads a specified DLL by name and retrieves a function pointer by its ordinal number.

**Constraints:** Use the `windows` crate and Win32 `LoadLibraryA`/`GetProcAddress` APIs. Return a `Result` containing the raw function pointer as `*const std::ffi::c_void`, or an error if the DLL cannot be loaded or the ordinal is not found.

**Signature:**
```rust
use windows::{
    core::{Result, PCSTR},
    Win32::System::LibraryLoader::{GetProcAddress, LoadLibraryA},
};

fn get_function_by_ordinal(dll_name: PCSTR, ordinal: u16) -> Result<*const std::ffi::c_void>
```

**Example:**
```rust
let func_ptr = get_function_by_ordinal(windows::core::s!("uxtheme.dll"), 135)?;
println!("Function pointer: {:?}", func_ptr);
```
