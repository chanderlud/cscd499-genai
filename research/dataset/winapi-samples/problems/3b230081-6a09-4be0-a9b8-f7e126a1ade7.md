1) Get DNS Servers for All Adapters

**Spec:** Write a function that retrieves and returns all DNS server IP addresses (both IPv4 and IPv6) configured on the system's network adapters.

**Constraints:**
- Must use the Win32 `GetAdaptersAddresses` API with `AF_UNSPEC` to get both IPv4 and IPv6 adapters
- Must handle the two-call pattern: first to get required buffer size, then to retrieve data
- Must iterate through the linked list of adapters and their DNS server addresses
- Must filter out unspecified addresses (0.0.0.0 or ::) and site-local IPv6 addresses (fec0::/10)
- Must properly align and cast the buffer to `IP_ADAPTER_ADDRESSES_LH` structures

**Signature:**
```rust
fn get_all_dns_servers() -> Result<Vec<IpAddr>>
```

**Example:**
```rust
let dns_servers = get_all_dns_servers()?;
for ip in dns_servers {
    println!("DNS Server: {}", ip);
}
```
