1) Convert Virtual Key to Scan Code

**Spec:** Write a function that converts a virtual key code to a scan code for a specified keyboard layout using the Win32 API.

**Constraints:**
- Use `MapVirtualKeyExW` with the `MAPVK_VK_TO_VSC_EX` mapping type
- Return an error if the conversion fails (when the API returns 0)
- The function must accept a `VIRTUAL_KEY` and an `HKL` (keyboard layout handle)

**Signature:**
```rust
use windows::core::Result;
use windows::Win32::UI::Input::KeyboardAndMouse::{HKL, VIRTUAL_KEY};

fn virtual_key_to_scan_code(vk: VIRTUAL_KEY, hkl: HKL) -> Result<u16>;
```

**Example:**
```rust
use windows::Win32::UI::Input::KeyboardAndMouse::{GetKeyboardLayout, VK_A};

let hkl = unsafe { GetKeyboardLayout(0) };
let scan_code = virtual_key_to_scan_code(VK_A, hkl)?;
println!("Scan code for VK_A: 0x{:04X}", scan_code);
```
