1) Toggle Window Enabled State

**Spec:** Write a function that toggles the enabled state of a window and returns the previous state.

**Constraints:**
- Use the `EnableWindow` and `IsWindowEnabled` Win32 functions
- The function must return the previous enabled state (true if the window was enabled before the toggle, false otherwise)

**Signature:**
```rust
unsafe fn toggle_window_enabled(hwnd: HWND) -> bool
```

**Example:**
```rust
// Assume we have a window handle `hwnd`
let was_enabled = unsafe { toggle_window_enabled(hwnd) };
// The window's enabled state is now toggled, and `was_enabled` indicates the previous state.
```
