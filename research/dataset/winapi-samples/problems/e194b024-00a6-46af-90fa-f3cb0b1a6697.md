Check stdin for available input without blocking

**Spec:** Write a function that checks if there is pending input available on the standard input (stdin) without blocking, using Win32 API. The function should return `true` if input is available, `false` if not, and propagate any errors.

**Constraints:** Must use `GetStdHandle`, `GetConsoleMode`, `SetConsoleMode`, and `WaitForSingleObject` from the Win32 API. Must handle `INVALID_HANDLE_VALUE` appropriately. Must temporarily set the console mode to enable processed and virtual terminal input, then restore the original mode.

**Signature:**
```rust
fn is_stdin_available() -> windows::core::Result<bool>
```

**Example:**
```rust
match is_stdin_available() {
    Ok(true) => println!("Input available"),
    Ok(false) => println!("No input available"),
    Err(e) => eprintln!("Error: {}", e),
}
```
