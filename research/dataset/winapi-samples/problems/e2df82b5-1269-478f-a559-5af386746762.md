Write bytes to a temp file in the same directory, flush, then atomically replace destination, without using heap allocation.

**Spec:**
- Write `data` to a temporary file in the same directory as `path`
- Flush the temporary file to ensure data is persisted
- Atomically replace the destination file with the temporary file
- Use `GetTempFileNameW`/`CreateFileW`/`WriteFile`/`FlushFileBuffers` + `ReplaceFileW` (fallback `MoveFileExW` as needed)

**Constraints:**
- Temporary file path buffer must be a fixed-size array on the stack (260 u16s, `MAX_PATH`)
- Assume directory paths are at most `MAX_PATH` (260 characters) in length
- Must handle both existing and non-existing destination files

**Signature:**
```rust
pub fn atomic_write(path: &Path, data: &[u8]) -> windows::core::Result<()>;
```

**Example:**
```rust
atomic_write(Path::new(r"C:\tmp\config.json"), br#"{"x":1}"#)?;
```
