1) Generate GDI to Monitor Name Lookup

**Spec:** Write a function that returns a mapping from GDI device names (as used by GDI functions) to the corresponding monitor friendly names for all active displays.

**Constraints:**
- Use the Windows Display Configuration API (`QueryDisplayConfig` and `DisplayConfigGetDeviceInfo`)
- Only consider active paths (use `QDC_ONLY_ACTIVE_PATHS` and `QDC_VIRTUAL_MODE_AWARE` flags)
- Return a `windows::core::Result<HashMap<String, String>>`

**Signature:**
```rust
fn generate_gdi_to_monitor_name_lookup() -> windows::core::Result<HashMap<String, String>>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    let lookup = generate_gdi_to_monitor_name_lookup()?;
    for (gdi_name, monitor_name) in &lookup {
        println!("{} -> {}", gdi_name, monitor_name);
    }
    Ok(())
}
```
