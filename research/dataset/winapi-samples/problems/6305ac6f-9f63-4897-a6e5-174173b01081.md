1) Display a message box

**Spec:** Write a function that displays a Windows message box with a given message and title, using the OK button.

**Constraints:**
- Use the `windows` crate and the `MessageBoxW` function from `windows::Win32::UI::WindowsAndMessaging`
- The function must be safe to call (i.e., no `unsafe` in the public API)
- The function must convert the input strings to null-terminated UTF-16 using a helper function
- The parent window handle should be null (i.e., no owner)

**Signature:**
```rust
fn show_message_box(message: &str, title: &str)
```

**Example:**
```rust
fn main() {
    show_message_box("Hello from Windows API!", "Example");
}
```
