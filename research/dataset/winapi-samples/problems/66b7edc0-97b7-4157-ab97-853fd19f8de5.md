1) Print Enabled Window Titles

**Spec:** Write a program that enumerates all top-level windows and prints the title of each enabled window that has a non-empty title.

**Constraints:**
- Use the `EnumWindows` function to enumerate windows
- Use a callback function that checks if the window is enabled (using `IsWindowEnabled`) and has a non-empty title (using `GetWindowTextLengthW` and `GetWindowTextW`)
- The callback must be an `unsafe extern "system"` function with the signature expected by `EnumWindows`
- Use the `windows` crate for Win32 bindings

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// The program, when run, will print the titles of all enabled top-level windows.
// Example output (will vary by system):
// Enabled window: Calculator
// Enabled window: File Explorer
```
