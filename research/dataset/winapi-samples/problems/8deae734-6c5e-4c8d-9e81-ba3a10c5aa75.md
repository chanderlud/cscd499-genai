N) Get Process Name from PID

**Spec:** Write a function that retrieves the executable filename (e.g., "notepad.exe") for a given process ID by using Win32 process information APIs.

**Constraints:**
- Must use `OpenProcess` with `PROCESS_QUERY_LIMITED_INFORMATION` access
- Must use `QueryFullProcessImageNameW` to get the full image path
- Must properly close the process handle using `CloseHandle`
- Must convert the wide-character path to a Rust `String` and extract only the filename component
- Must return a `Result<String>` with appropriate error handling

**Signature:**
```rust
fn get_process_name_from_pid(pid: u32) -> Result<String>
```

**Example:**
```rust
let current_pid = std::process::id();
match get_process_name_from_pid(current_pid) {
    Ok(name) => println!("Process name: {}", name),
    Err(e) => println!("Failed: {}", e),
}
```
