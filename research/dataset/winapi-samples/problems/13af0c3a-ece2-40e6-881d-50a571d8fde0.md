**Spec:** Copy `value` into a mutable UTF-16 buffer as a null-terminated string and return the number of UTF-16 code units written, excluding the terminator. The implementation must not perform any heap allocations (no `Vec`, `Box`, etc.).
**Constraints:** Use `PWSTR`; return `E_INVALIDARG` if the buffer is null; return `ERROR_INSUFFICIENT_BUFFER` if the buffer is too small; on success always write a terminator; must be implemented without any dynamic memory allocation.
**Signature:**
```rust
pub fn write_wide_noalloc(buf: PWSTR, cap: usize, value: &str) -> Result<usize>;
```
**Example:**
```rust
let mut buf = vec![0u16; 8];
let n = write_wide_noalloc(PWSTR::from_raw(buf.as_mut_ptr()), buf.len(), "cat")?;
assert_eq!(n, 3);
```
