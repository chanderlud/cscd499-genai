1) Vertically Centered Edit Control

**Spec:** Create an edit control that vertically centers its text by adjusting the non-client area and painting the non-client area with a solid color.

**Constraints:**
- Use window subclassing to handle WM_NCCALCSIZE, WM_NCPAINT, and WM_SIZE messages
- The non-client area should be painted with a light gray brush (0xC8C8C8)
- The edit control should be created as a child of a given parent window

**Signature:**
```rust
fn create_centered_edit_control(parent: HWND, text: &str, x: i32, y: i32, width: i32, height: i32) -> Result<HWND>
```

**Example:**
```rust
let parent = ...; // some parent window
let edit = create_centered_edit_control(parent, "Centered Text", 50, 50, 200, 30)?;
```
