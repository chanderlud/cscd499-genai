1) Pre-allocate Listbox Storage

**Spec:** Create a listbox control as a child of the given parent window, pre-allocate storage for 1000 items with an average string length of 20 characters, clear any existing content, and then add 100 items labeled "Item 0" through "Item 99".

**Constraints:** Use the Win32 API via the `windows` crate. The listbox must be created with the styles: WS_VISIBLE, WS_CHILD, WS_BORDER, WS_VSCROLL, LBS_HASSTRINGS, LBS_NOTIFY. Use `LB_INITSTORAGE` to pre-allocate storage, then `LB_RESETCONTENT` to clear, and `LB_ADDSTRING` to add items.

**Signature:**
```rust
fn create_listbox_with_storage(parent: HWND) -> Result<HWND>
```

**Example:**
```rust
// In the window procedure, during WM_CREATE:
let listbox = create_listbox_with_storage(hwnd)?;
```
