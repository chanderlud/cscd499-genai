1) Get Monitor Information from Point

**Spec:** Write a function that, given a point, returns the device name and monitor rectangle for the monitor containing that point. If no monitor contains the point, return an error.

**Constraints:**
- Use the Windows API `MonitorFromPoint` with `MONITOR_DEFAULTTONULL` flag
- Use `GetMonitorInfoW` to retrieve monitor information
- Convert the wide-character device name to a Rust `String`
- Return an error if the point is not on any monitor or if API calls fail

**Signature:**
```rust
fn get_monitor_info_from_point(point: POINT) -> Result<(String, RECT)>
```

**Example:**
```rust
let point = POINT { x: 100, y: 100 };
let (device_name, rect) = get_monitor_info_from_point(point)?;
println!("Monitor device name: {}", device_name);
println!("Monitor rectangle: {:?}", rect);
```
