1) Implement Read for IStream

**Spec:** Create a wrapper struct that adapts a Windows COM `IStream` object to implement Rust's `std::io::Read` trait.

**Constraints:**
- The wrapper must hold a reference to an `IStream`
- Must implement `From<&IStream>` for the wrapper
- The `Read` implementation must call `IStream::Read` and convert COM errors to `std::io::Error`
- Must handle partial reads correctly by returning the actual bytes read

**Signature:**
```rust
pub struct WinStream<'a> {
    stream: &'a IStream,
}

impl<'a> From<&'a IStream> for WinStream<'a> {
    fn from(stream: &'a IStream) -> Self;
}

impl Read for WinStream<'_> {
    fn read(&mut self, buf: &mut [u8]) -> Result<usize, std::io::Error>;
}
```

**Example:**
```rust
use std::io::Read;
use windows::Win32::System::Com::IStream;

// Assume stream is a valid IStream instance
let mut win_stream = WinStream::from(&stream);
let mut buffer = [0u8; 1024];
let bytes_read = win_stream.read(&mut buffer)?;
```
