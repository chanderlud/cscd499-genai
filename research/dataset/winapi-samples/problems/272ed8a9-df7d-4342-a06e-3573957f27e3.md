1) Create Edit Control and Toggle Readonly

**Spec:** Write a function that creates an edit control as a child window, verifies its initial readonly state, toggles the readonly state off, and verifies the change.

**Constraints:**
- Must use the `windows` crate with Win32 UI APIs
- Edit control must be created with `ES_READONLY` style initially
- Must verify readonly state by checking window style bits before and after toggling
- Must toggle readonly state using `EM_SETREADONLY` message with WPARAM(0)
- Must print verification results to console

**Signature:**
```rust
fn create_edit_control_and_toggle_readonly(parent: HWND, text: &str) -> Result<()>
```

**Example:**
```rust
let parent = /* main window handle */;
create_edit_control_and_toggle_readonly(parent, "Initial text")?;
// Console output:
// Edit control readonly state: true
// Edit control readonly state after toggle: false
```
