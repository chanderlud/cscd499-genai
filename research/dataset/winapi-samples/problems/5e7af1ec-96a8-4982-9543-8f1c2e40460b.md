1) Request User Consent Verification

**Spec:** Write an unsafe function that uses the Windows `UserConsentVerifier` to request user verification for a given window handle and message, returning the verification result.

**Constraints:**
- Use the `windows` crate with `Security::Credentials::UI`, `Win32::Foundation`, and `Win32::System::WinRT` features
- Use the `windows_future` crate for async operations
- Must use `IUserConsentVerifierInterop` interface via `factory` function
- Return `Result<UserConsentVerificationResult>`

**Signature:**
```rust
unsafe fn request_user_consent_verification(
    window: HWND,
    message: &HSTRING,
) -> Result<UserConsentVerificationResult>
```

**Example:**
```rust
use windows::Win32::Foundation::HWND;
use windows::core::*;

fn main() -> Result<()> {
    let window = HWND::default(); // Replace with actual window handle
    let message = h!("Please verify your identity");
    let result = unsafe { request_user_consent_verification(window, message) }?;
    println!("{:?}", result);
    Ok(())
}
```
