1) Create a window with a button and set focus

**Spec:** Write a program that creates a window with a button control, sets input focus to the button, and verifies the focus was set correctly by printing a message.

**Constraints:**
- Use the `windows` crate (version 0.52.0 or compatible) and Win32 API
- Register a window class with a basic window procedure handling `WM_DESTROY`
- Create an overlapped window with `WS_OVERLAPPEDWINDOW | WS_VISIBLE` styles
- Create a child button window with `WS_CHILD | WS_VISIBLE` styles using the "BUTTON" class
- Use `SetFocus` to set focus to the button immediately after creation
- Use `GetFocus` to verify the button has focus and print appropriate message
- Implement a standard message loop with `GetMessageW`, `TranslateMessage`, and `DispatchMessageW`

**Signature:** The main function should have the following signature:
```rust
fn main() -> windows::core::Result<()>
```

**Example:** When run, the program should create a window titled "Button Focus Example" with a "Click Me" button, and print "Button successfully received focus" if focus was set correctly.
