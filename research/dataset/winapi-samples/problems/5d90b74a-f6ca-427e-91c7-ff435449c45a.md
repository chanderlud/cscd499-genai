1) Clear ListBox Items

**Spec:** Create a ListBox control, add multiple string items, and then clear all items using the LB_RESETCONTENT message.

**Constraints:**
- Use the Win32 ListBox control with the styles: WS_VISIBLE, WS_CHILD, WS_VSCROLL, WS_BORDER, WS_TABSTOP, LBS_HASSTRINGS, LBS_NOTIFY, LBS_MULTIPLESEL.
- Add at least 5 items to the ListBox.
- Use the LB_RESETCONTENT message to clear the ListBox.

**Signature:**
```rust
fn clear_listbox(hwnd: HWND) -> Result<()>
```

**Example:**
```rust
// Assume hwnd is a handle to a ListBox control with items.
clear_listbox(hwnd)?;
// Now the ListBox is empty.
```
