1) Primary Monitor Video Modes

**Spec:** Write a function that enumerates all available video modes for the primary monitor and returns them as a sorted set. Each mode must include resolution (width × height), color bit depth, and refresh rate.

**Constraints:**
- Use Win32 API via the `windows` crate
- Must handle errors and return a `Result`
- Video modes must be sorted by resolution, then bit depth, then refresh rate
- Only include modes where all required fields (width, height, bit depth, refresh rate) are available

**Signature:**
```rust
fn get_primary_monitor_video_modes() -> Result<BTreeSet<VideoMode>>
```

**Example:**
```rust
let modes = get_primary_monitor_video_modes()?;
for mode in modes {
    println!("{}x{} @ {}Hz, {} bit", mode.size.0, mode.size.1, mode.refresh_rate, mode.bit_depth);
}
```
