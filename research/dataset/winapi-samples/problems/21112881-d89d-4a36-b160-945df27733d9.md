1) Set and Retrieve Edit Control Text Limit

**Spec:** Write a function that creates an edit control within a parent window, sets its maximum text length to a specified limit, and retrieves the current limit.

**Constraints:** Use the `windows` crate to create the edit control with appropriate styles and send `EM_SETLIMITTEXT` and `EM_GETLIMITTEXT` messages via `SendMessageW`. The function must handle window creation errors and return the retrieved limit.

**Signature:**
```rust
fn set_and_get_edit_limit(parent_hwnd: HWND, limit: u32) -> Result<u32>
```

**Example:**
```rust
// In a window procedure during WM_CREATE:
let parent_hwnd = hwnd; // Parent window handle
let limit = set_and_get_edit_limit(parent_hwnd, 10)?;
println!("Text limit set to: {}", limit);
```
