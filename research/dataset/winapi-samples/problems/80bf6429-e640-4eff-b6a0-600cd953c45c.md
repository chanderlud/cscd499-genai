1) Load Icon from Resource

**Spec:** Write a function that loads an icon from the current module's resources by its resource ID and returns a handle that automatically destroys the icon when dropped.

**Constraints:**
- Use the `windows` crate and the `Win32` API.
- The function must return a `Result` type, propagating any errors from the Win32 API.
- The returned handle must be a wrapper around `HICON` that implements `Drop` to call `DestroyIcon`.
- The function must use `GetModuleHandleW` to get the current module handle and convert it to an `HINSTANCE`.
- The function must use `LoadImageW` to load the icon, passing the resource ID as a `PCWSTR` (by casting the `u16` to a pointer).
- The function must allow specifying the width and height of the icon, but if both are 0, use the default size (via `LR_DEFAULTSIZE`).

**Signature:**
```rust
fn load_icon_from_resource(resource_id: u16, width: i32, height: i32) -> Result<IconHandle>
```

**Example:**
```rust
fn main() -> Result<()> {
    let icon = load_icon_from_resource(1, 0, 0)?;
    println!("Icon loaded successfully from resource.");
    Ok(())
}
```
