1) Create a Bitmap Button Window

**Spec:** Create a Win32 window containing a button that displays a 16x16 red bitmap image.

**Constraints:**
- Use the `windows` crate for Win32 bindings
- Create the button with `BS_BITMAP` style
- Generate the bitmap using `CreateBitmap` with 32-bit ARGB format
- Set the button image via `SendMessageW` with `BM_SETIMAGE`
- Implement a standard Win32 message loop
- Clean up the bitmap resource with `DeleteObject` before exit

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// Program creates a window with a red bitmap button
// Running the program shows a window with a 16x16 red square as a button
```
