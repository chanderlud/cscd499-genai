N) Set and measure edit control text without allocation

**Spec:** Write a program that creates a window with an edit control, sets the text of the edit control to a given string, and then retrieves and prints the length of the text without allocating a string for the text.

**Constraints:**
- Use the `windows` crate for Win32 API bindings.
- The text length must be obtained using the `EM_LINELENGTH` message without retrieving the string content.
- The program must create a window and an edit control, set the text, and then enter a message loop.

**Signature:** The main function should have the following signature:
```rust
fn main() -> windows::core::Result<()>
```

**Example:** When the program runs, it should create a window with an edit control containing "Hello, World!" and print the length of the text (which is 13) to the console.
