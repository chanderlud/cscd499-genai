N) Create a Window with a Colored Static Label

**Spec:** Write a Rust program that creates an overlapped window containing a static control (label) with a custom background color. The background color must be set to light blue (RGB: 173, 216, 230) by handling the `WM_CTLCOLORSTATIC` message.

**Constraints:**
- Use the `windows` crate for Win32 bindings.
- The static control must be created as a child window of the main window in the `WM_CREATE` message handler.
- The window procedure must handle `WM_CTLCOLORSTATIC` to set the background color and return a brush handle.
- The program must run a standard message loop until the window is closed.

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:** When run, the program displays a window with a static label that has a light blue background.
