1) Get Window Long

**Spec:** Write a function that retrieves a window attribute (such as style) for a given window handle using the appropriate Win32 API for the target platform (32-bit or 64-bit). The function must handle errors by checking the return value and thread error state.

**Constraints:**
- Use the `windows` crate for Win32 bindings
- Must handle both 32-bit and 64-bit platforms via conditional compilation
- Must distinguish between a legitimate zero return value and an error condition
- Must be safe to call (encapsulate the unsafe FFI call)

**Signature:**
```rust
fn get_window_long(hwnd: HWND, nindex: WINDOW_LONG_PTR_INDEX) -> Result<isize>
```

**Example:**
```rust
use windows::Win32::UI::WindowsAndMessaging::{GetDesktopWindow, GWL_STYLE};

let hwnd = unsafe { GetDesktopWindow() };
let style = get_window_long(hwnd, GWL_STYLE)?;
println!("Desktop window style: 0x{:X}", style);
```
