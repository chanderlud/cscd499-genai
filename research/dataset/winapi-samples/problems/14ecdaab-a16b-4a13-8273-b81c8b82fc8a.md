1) ListBox Text Retrieval
**Spec:** Create a ListBox control, populate it with string items, set the selection to a specific index, and retrieve the text of the currently selected item.
**Constraints:** Must use Win32 ListBox messages (LB_ADDSTRING, LB_SETCURSEL, LB_GETCURSEL, LB_GETTEXTLEN, LB_GETTEXT) and handle the WM_CREATE message to perform these operations.
**Signature:**
```rust
fn listbox_operations(hwnd_listbox: HWND, items: &[&str], select_index: usize) -> Result<String>
```
**Example:**
```rust
let items = ["Apple", "Banana", "Cherry", "Date", "Elderberry"];
let selected_text = listbox_operations(hwnd_listbox, &items, 1)?;
// selected_text should be "Banana"
```
