1) Create a Static Label Control

**Spec:** Write a function that creates a static label control (a child window of the "STATIC" class) in a given parent window. The label should display the specified text, be positioned at (x, y) with the given width and height, and use the default GUI font.

**Constraints:**
- Use the `windows` crate for Win32 API calls.
- The function must return the `HWND` of the created label.
- The function must set the label's font to the default GUI font by sending a `WM_SETFONT` message with the stock object `DEFAULT_GUI_FONT`.

**Signature:**
```rust
fn create_static_label(parent: HWND, text: &str, x: i32, y: i32, width: i32, height: i32) -> Result<HWND>
```

**Example:**
```rust
// Inside a window procedure for WM_CREATE:
let label = create_static_label(hwnd, "Hello, Windows!", 10, 10, 200, 25)?;
// Store the label handle for later use.
```
