7) Set and Get Window Property

**Spec:** Write a program that creates a window, sets a property on it using `SetPropW`, retrieves the property with `GetPropW`, and prints the retrieved value.

**Constraints:**
- The property value must be stored in a static variable to avoid dangling pointers.
- The program must run a message loop until the window is destroyed.
- The program must clean up by destroying the window and unregistering the window class.

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:** The program should output: `Retrieved property value: 42`
