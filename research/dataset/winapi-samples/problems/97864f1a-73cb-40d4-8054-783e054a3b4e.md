N) Get Instance Handle via Linker Symbol

**Spec:** Write a function that returns the instance handle (HMODULE) of the current module by using the linker symbol `__ImageBase`.

**Constraints:**
- Use the `windows` crate.
- Use the `IMAGE_DOS_HEADER` type and the `__ImageBase` linker symbol.
- The function must be safe to call (encapsulate any `unsafe` code).

**Signature:**
```rust
fn get_instance_handle() -> HMODULE
```

**Example:**
```rust
fn main() {
    let instance = get_instance_handle();
    println!("Instance handle: {:?}", instance);
}
```
