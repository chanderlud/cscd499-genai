1) Simulate Button Click

**Spec:** Create a window with a button control and programmatically simulate a button click using `SendMessageW` with the `BM_CLICK` message. The window procedure must handle the `BN_CLICKED` notification and print a confirmation message.

**Constraints:** Use the `windows` crate for Win32 API bindings. The button must be created as a child window with the `BS_PUSHBUTTON` style. The simulation must occur without user interaction, and the program must enter a standard message loop after the simulated click.

**Signature:**
```rust
fn simulate_button_click() -> windows::core::Result<()>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    simulate_button_click()
}
```
