1) Get MessageBoxA Address

**Spec:** Write a function that retrieves the memory address of the `MessageBoxA` function from `user32.dll` using Win32 API calls.

**Constraints:**
- Use the `windows` crate for Win32 bindings
- Must use `GetModuleHandleW` to obtain the module handle for `user32.dll`
- Must use `GetProcAddress` to retrieve the function address
- Return the address as a `usize` wrapped in a `Result`
- Handle errors appropriately using the `windows::core::Error` type

**Signature:**
```rust
fn get_message_box_addr() -> Result<usize>
```

**Example:**
```rust
fn main() -> Result<()> {
    let addr = get_message_box_addr()?;
    println!("MessageBoxA address: {:p}", addr as *const ());
    Ok(())
}
```
