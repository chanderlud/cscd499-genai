1) Find Executable Memory Region

**Spec:** Write a function that scans memory regions starting from a given base address and returns the first region that is committed and has execute-read protection.

**Constraints:**
- Use the Windows API function `VirtualQuery` to query memory information
- The function must return `Option<(usize, usize)>` where the tuple is (start address, region size) of the first executable region, or `None` if none is found
- The function should stop scanning and return `None` if `VirtualQuery` fails (i.e., returns 0)

**Signature:**
```rust
fn get_first_executable_memory_region(base_address: usize) -> Option<(usize, usize)>
```

**Example:**
```rust
// Example: Find the first executable region starting at address 0x10000000
let base_address = 0x10000000;
if let Some((addr, size)) = get_first_executable_memory_region(base_address) {
    println!("Executable region at 0x{:X} with size {}", addr, size);
} else {
    println!("No executable region found");
}
```
