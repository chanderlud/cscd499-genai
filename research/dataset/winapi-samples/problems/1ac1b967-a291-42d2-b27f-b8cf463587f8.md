N) Create and manage a pseudo-console (ConPTY) using dynamic loading

**Spec:** Write a function that dynamically loads the ConPTY API functions from kernel32.dll, creates a pseudo-console with size 80x25, resizes it to 120x30, and then closes it.

**Constraints:**
- Use `GetModuleHandleW` to obtain the module handle for kernel32.dll.
- Use `GetProcAddress` to retrieve the function pointers for `CreatePseudoConsoleW`, `ResizePseudoConsoleW`, and `ClosePseudoConsoleW`.
- Use null handles for the input and output of the pseudo-console.
- The function must return `Result<()>` and propagate any errors from the Win32 API calls.

**Signature:**
```rust
fn manage_pseudo_console() -> Result<()>
```

**Example:**
```rust
manage_pseudo_console()?;
```
