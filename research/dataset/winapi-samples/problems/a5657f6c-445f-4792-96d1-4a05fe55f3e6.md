1) Check High Contrast Mode
**Spec:** Implement a function that checks if high contrast mode is enabled on the Windows system using the SystemParametersInfoA API.
**Constraints:** Must use the ANSI version of the API (SystemParametersInfoA), properly initialize the HIGHCONTRASTA struct with its size, handle the unsafe call, and return a Result<bool> where true indicates high contrast is on.
**Signature:**
```rust
fn is_high_contrast() -> Result<bool>
```
**Example:**
```rust
fn main() -> Result<()> {
    let high_contrast = is_high_contrast()?;
    println!("High contrast mode is {}", if high_contrast { "ON" } else { "OFF" });
    Ok(())
}
```
