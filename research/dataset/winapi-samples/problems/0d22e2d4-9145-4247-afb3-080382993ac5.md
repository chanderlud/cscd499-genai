N) Check High Contrast Mode

**Spec:** Write a function that checks whether the system is currently in high contrast mode.

**Constraints:**
- Use the `windows` crate and Win32 API
- Must call `SystemParametersInfoW` with `SPI_GETHIGHCONTRAST`
- Must properly initialize and use the `HIGHCONTRASTW` structure
- Must check the `HCF_HIGHCONTRASTON` flag in the returned flags
- Return a `Result<bool>` indicating whether high contrast is enabled

**Signature:**
```rust
fn is_high_contrast() -> Result<bool>
```

**Example:**
```rust
let high_contrast = is_high_contrast()?;
println!("High contrast mode is {}", if high_contrast { "enabled" } else { "disabled" });
```
