1) OCR Image Text Recognition

**Spec:** Write a function that takes a file path to an image and returns the text recognized by the Windows OCR engine.

**Constraints:**
- Use the `windows` crate and the WinRT APIs for OCR (specifically `OcrEngine` and `BitmapDecoder`).
- The function must handle the async operations by blocking until they complete (using `.join()` or similar).
- The function should return a `windows::core::Result<String>`.

**Signature:**
```rust
fn recognize_text_from_image(path: &std::path::Path) -> windows::core::Result<String>
```

**Example:**
```rust
let path = std::path::Path::new("message.png");
let text = recognize_text_from_image(path)?;
println!("Recognized text: {}", text);
```
