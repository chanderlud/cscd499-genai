1) List Visible Windows with Titles

**Spec:** Write a Rust program that enumerates all top-level windows using Win32 APIs, filters those that are visible and have a non-empty title, and prints each matching window's title along with its top-left coordinates.

**Constraints:** 
- Use the `windows` crate for Win32 bindings.
- Implement enumeration with `EnumWindows` and a callback function of type `WNDENUMPROC`.
- The callback must use `extern "system"` calling convention and return `BOOL`.
- Retrieve window text with `GetWindowTextW` and window info with `GetWindowInfo`.
- Check for `WS_VISIBLE` style and non-empty title before printing.
- Output format: "title (left, top)" for each window.

**Signature:** 
```rust
fn main() -> windows::core::Result<()>
```

**Example:** 
```rust
// When run, the program outputs lines like:
// "Untitled - Notepad (100, 200)"
// "Calculator (300, 400)"
// for each visible window with a title.
```
