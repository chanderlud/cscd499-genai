1) Create Icon Button

**Spec:** Write a function that creates a button control with the system application icon (IDI_APPLICATION) and the BS_ICON style. The button must be a child of the given parent window and have the styles: WS_VISIBLE, WS_CHILD, WS_TABSTOP, BS_NOTIFY, and BS_ICON.

**Constraints:**
- Use the `windows` crate.
- The function must return a `Result<HWND>` for the button window handle.
- The button must be created at the given position and size.

**Signature:**
```rust
fn create_icon_button(parent: HWND, instance: HINSTANCE, x: i32, y: i32, width: i32, height: i32) -> Result<HWND>
```

**Example:**
```rust
let button = create_icon_button(hwnd, hinstance, 50, 50, 200, 50)?;
```
