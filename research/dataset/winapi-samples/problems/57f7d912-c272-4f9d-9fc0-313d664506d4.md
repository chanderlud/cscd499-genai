1) Get Module File Name

**Spec:** Write a function that returns the full file path of a module (executable or DLL) given its instance handle.

**Constraints:**
- Use the `GetModuleFileNameW` Win32 API.
- The function must return a `Result<String>` and handle errors by returning an appropriate `Error`.
- The buffer for the path must be of size `MAX_PATH`.

**Signature:**
```rust
fn get_module_path(instance: HINSTANCE) -> Result<String>
```

**Example:**
```rust
let module_path = get_module_path(HINSTANCE::default())?;
println!("Module path: {}", module_path);
```
