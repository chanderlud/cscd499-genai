N) Dynamically Load GetDpiForWindow

**Spec:** Write a program that dynamically loads the `GetDpiForWindow` function from `user32.dll` and prints the DPI of the desktop window. If the function is not available, print a message.

**Constraints:**
- Use `LoadLibraryW` and `GetProcAddress` to load the function.
- The function name must be null-terminated.
- Use `GetDesktopWindow` to get the desktop window handle.
- Use the `windows` crate for Win32 API bindings.

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
Running the program might output:
```
DPI for desktop window: 96
```
or if the function is not available:
```
GetDpiForWindow not available on this system
```
