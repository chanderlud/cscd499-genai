1) Undecorated Window Drag and Resize

**Spec:** Create a child window that enables dragging and resizing of its parent window through border interactions, even when the parent window has no standard title bar or borders (undecorated).

**Constraints:**
- The child window must handle `WM_NCHITTEST` to return appropriate hit-test values for resize borders
- The child window must handle `WM_NCLBUTTONDOWN` to initiate drag/resize operations by posting messages to the parent
- The parent window must be subclassed to handle `WM_SIZE` and adjust the child window's position and region accordingly
- The child window must set a region using `SetWindowRgn` to define the draggable border areas
- The solution must use the `windows` crate and Win32 API functions
- Border width and height should be configurable (default 8 pixels)

**Signature:**
```rust
fn create_drag_resize_window(parent: HWND, has_undecorated_shadows: bool) -> Result<HWND>
```

**Example:**
```rust
let parent = /* existing window handle */;
let drag_window = create_drag_resize_window(parent, false)?;
// User can now drag/resize parent by interacting with drag_window's borders
```
