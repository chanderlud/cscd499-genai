Read System App Theme Setting

**Spec:**  
Read the Windows registry value `AppsUseLightTheme` under `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Themes\Personalize` and return whether the system uses light theme for apps.

**Constraints:**  
- Use the `windows` crate and `RegGetValueW` Win32 API
- The registry value is a `DWORD` (32-bit unsigned integer)
- Return `Ok(Some(true))` if the value is non-zero (light theme)
- Return `Ok(Some(false))` if the value is zero (dark theme)
- Return an error if the registry read fails

**Signature:**  
```rust
fn read_apps_use_light_theme() -> Result<Option<bool>>
```

**Example:**  
```rust
match read_apps_use_light_theme()? {
    Some(true) => println!("System uses light theme for apps"),
    Some(false) => println!("System uses dark theme for apps"),
    None => println!("Could not read theme setting"),
}
```
