15) Set ListBox Selection

**Spec:** Write a function that sets the current selection in a Win32 list box control to the item at the specified index.

**Constraints:**
- Use the `SendMessageW` function from the `windows` crate
- Send the `LB_SETCURSEL` message
- Return an error if the index is out of range (the message returns -1)

**Signature:**
```rust
fn set_listbox_selection(listbox: HWND, index: usize) -> Result<()>
```

**Example:**
```rust
// In a window creation function, after creating the list box and adding items:
set_listbox_selection(listbox, 2)?;
```
