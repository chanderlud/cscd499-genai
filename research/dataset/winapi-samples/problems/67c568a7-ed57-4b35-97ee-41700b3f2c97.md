1) Check Process Priority Class

**Spec:** Write a program that retrieves the priority class of the current process and prints whether it is below normal or not.

**Constraints:**
- Use the `windows` crate and the Win32 API functions `GetCurrentProcess` and `GetPriorityClass`
- Handle errors by converting the last OS error to an `HRESULT` and returning a `Result`
- Check the handle returned by `GetCurrentProcess` for validity (i.e., not `INVALID_HANDLE_VALUE`)
- Check the return value of `GetPriorityClass` for zero (indicating failure)
- Compare the priority class with `BELOW_NORMAL_PRIORITY_CLASS` (using its inner value)

**Signature:**
```rust
fn main() -> Result<()>
```

**Example:**
```rust
// When the process priority is below normal, the program prints:
// "Process priority: Below Normal"
// Otherwise, it prints:
// "Process priority: Normal or higher"
```
