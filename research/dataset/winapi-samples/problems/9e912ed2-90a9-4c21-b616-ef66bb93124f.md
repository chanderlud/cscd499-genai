1) Memory Card Matching Game

**Spec:** Create a window that displays a 3x6 grid of cards with 3D flip animations. Each card shows a letter on the front and an image on the back. When clicked, cards flip with a smooth rotation animation. Matching pairs (same letter, case-insensitive) remain face-up; non-matching pairs flip back. The game must handle DPI changes and window resizing.

**Constraints:** Use DirectComposition for 3D transforms and visual composition, Direct2D/DirectWrite for rendering, and UI Animation for smooth transitions. Cards must be 150x210 logical pixels with 15-pixel margins. The window must be DPI-aware and adjust layout accordingly. Randomly assign letter pairs (uppercase and lowercase of same letter) to cards.

**Signature:**
```rust
fn main() -> Result<()>
```

**Example:**
```rust
fn main() -> Result<()> {
    unsafe {
        CoInitializeEx(None, COINIT_MULTITHREADED).ok()?;
        SetProcessDpiAwarenessContext(DPI_AWARENESS_CONTEXT_PER_MONITOR_AWARE_V2)?;
    }
    let mut window = Window::new()?;
    window.run()
}
```
