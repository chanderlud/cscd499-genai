N) Get Monitor Information for a Window

**Spec:** Write a function that retrieves the monitor information (bounds and work area) for the monitor that a given window is mostly displayed on.

**Constraints:**
- Use `MonitorFromWindow` with `MONITOR_DEFAULTTONEAREST` to obtain the monitor handle
- Use `GetMonitorInfoW` to populate a `MONITORINFO` structure
- Handle errors appropriately and return a `Result`
- The function must work with any valid window handle

```rust
}
```

**Example:**
```rust
use windows::Win32::UI::WindowsAndMessaging::GetDesktopWindow;

fn main() -> Result<()> {
    let desktop_window = unsafe { GetDesktopWindow() };
    let monitor_info = get_monitor_info(desktop_window)?;
    println!("Monitor bounds: {:?}", monitor_info.rcMonitor);
    println!("Work area: {:?}", monitor_info.rcWork);
    Ok(())
}
```
