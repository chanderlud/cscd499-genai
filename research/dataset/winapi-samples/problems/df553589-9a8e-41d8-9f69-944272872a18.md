1) Temporary Memory Protection Change

**Spec:** Implement an unsafe function that temporarily changes memory protection to allow writing, executes a closure with mutable access to the target memory, then restores the original protection.

**Constraints:**
- Must use `VirtualProtect` from the `windows` crate
- Must restore original protection flags after closure execution
- Must handle errors via `windows::core::Result`
- Closure receives a mutable reference to the value at the specified address

**Signature:**
```rust
unsafe fn with_mut_ref<T, F: FnOnce(&mut T)>(address: usize, block: F) -> windows::core::Result<()>
```

**Example:**
```rust
static mut DATA: u32 = 42;

unsafe {
    with_mut_ref(&mut DATA as *mut u32 as usize, |val| {
        *val = 100;
    })?;
    // DATA is now 100 with original protection restored
}
```
