N) Window Frame Thickness

**Spec:** Write a function that returns the total thickness of a window's frame (including resize border and padding) in pixels for the current system.

**Constraints:** Must use the Win32 `GetSystemMetrics` API with the `SM_CXSIZEFRAME` and `SM_CXPADDEDBORDER` constants.

**Signature:**
```rust
fn get_frame_thickness() -> i32
```

**Example:**
```rust
let thickness = get_frame_thickness();
println!("Window frame thickness: {} pixels", thickness);
```
