1) Query File Attributes

**Spec:** Write a function that uses the Windows API to retrieve and display file attributes, timestamps, and size for a given file path.

**Constraints:**
- Convert the file path to a null-terminated wide string (UTF-16) for Windows API calls
- Use `GetFileAttributesExW` from the `windows` crate to query file information
- Display the file attributes as both a hexadecimal value and individual flag names (READONLY, HIDDEN, SYSTEM, DIRECTORY, ARCHIVE, REPARSE_POINT)
- Display creation, last access, and last write timestamps
- Display the file size in bytes (combining high and low parts into a 64-bit value)
- Handle errors gracefully with appropriate error messages

**Signature:**
```rust
fn query_file_attributes(file_path: &str) -> Result<()>
```

**Example:**
```rust
query_file_attributes("C:\\Windows\\System32\\kernel32.dll")?;
// Output:
// File: C:\Windows\System32\kernel32.dll
// Attributes: 0x00000020
//   - FILE_ATTRIBUTE_ARCHIVE
// 
// Timestamps:
//   Creation Time: ...
//   Last Access Time: ...
//   Last Write Time: ...
// 
// File Size: 1234567 bytes
```
