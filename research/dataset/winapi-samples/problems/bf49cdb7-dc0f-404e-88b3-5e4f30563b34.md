1) D3D12 Hello Triangle
**Spec:** Write a Rust program that creates a Direct3D 12 application rendering a colored triangle using the Windows API. The program must initialize D3D12, create a window, set up the rendering pipeline, and display a triangle with vertex colors.
**Constraints:**
- Use the `windows` crate for Win32/D3D12 bindings
- Support command-line argument `/warp` to use WARP software adapter
- Implement proper resource cleanup and frame synchronization with fences
- Compile HLSL shaders from an external `shaders.hlsl` file
- Use flip-model swap chain with two back buffers
**Signature:**
```rust
fn run_sample<S: DXSample>() -> windows::core::Result<()>
```
**Example:**
```rust
fn main() -> windows::core::Result<()> {
    run_sample::<d3d12_hello_triangle::Sample>()?;
    Ok(())
}
```
