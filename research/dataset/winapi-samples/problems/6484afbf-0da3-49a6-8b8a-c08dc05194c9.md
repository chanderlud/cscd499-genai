1) Button Visibility Control

**Spec:** Write a program that creates a window with a button and demonstrates the use of `IsWindowVisible` and `ShowWindow` to check and change the button's visibility.

**Constraints:** 
- Use the `windows` crate for Win32 API bindings.
- The program must create a main window and a button control.
- The program must check the initial visibility of the button, then hide it, check again, then show it, and check again, printing the results.
- The program must run a standard Win32 message loop.

**Signature:** The main function should return `windows::core::Result<()>`.

**Example:** The program should output the visibility states as in the code.
