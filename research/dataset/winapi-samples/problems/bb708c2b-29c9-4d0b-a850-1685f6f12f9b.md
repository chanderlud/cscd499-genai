1) Set and Get Password Character

**Spec:** Create a window with an edit control, set its password character to '*', then retrieve and print the current password character.

**Constraints:** Use the Windows API via the `windows` crate. The edit control must be created with the ES_PASSWORD style. Use the EM_SETPASSWORDCHAR and EM_GETPASSWORDCHAR messages to set and get the password character.

**Signature:** The program should have a `main` function that returns `windows::core::Result<()>`.

**Example:**
```rust
// Assume we have an edit control handle `edit_hwnd` created with ES_PASSWORD style.
let password_char = '*' as u16;
SendMessageW(edit_hwnd, EM_SETPASSWORDCHAR, Some(WPARAM(password_char as usize)), Some(LPARAM(0)));
let retrieved_char = SendMessageW(edit_hwnd, EM_GETPASSWORDCHAR, Some(WPARAM(0)), Some(LPARAM(0)));
if retrieved_char != LRESULT(0) {
    let ch = char::from_u32(retrieved_char.0 as u32).unwrap_or('?');
    println!("Password character is: '{}'", ch);
}
```
