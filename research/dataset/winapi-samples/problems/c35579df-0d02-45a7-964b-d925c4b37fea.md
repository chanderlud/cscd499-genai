1) ListBox Range Selection

**Spec:** Create a window containing a multi-select ListBox control, populate it with 10 items, and programmatically select a contiguous range of items (indices 2 through 5 inclusive) using the LB_SELITEMRANGEEX message.

**Constraints:**
- Must use Win32 API via the `windows` crate
- ListBox must be created with LBS_MULTIPLESEL style
- Selection must be performed using LB_SELITEMRANGEEX message
- Must handle window creation and message loop

**Signature:**
```rust
extern "system" fn wndproc(hwnd: HWND, msg: u32, wparam: WPARAM, lparam: LPARAM) -> LRESULT;
```

**Example:**
```rust
// In WM_CREATE handler:
// 1. Create ListBox with multi-select style
// 2. Add 10 items using LB_ADDSTRING
// 3. Select items 2-5 using LB_SELITEMRANGEEX
SendMessageW(listbox, LB_SELITEMRANGEEX, Some(WPARAM(2)), Some(LPARAM(5)));
```
