1) Read Registry String Value

**Spec:**
- Write a function that reads a REG_SZ value from the Windows registry under HKEY_LOCAL_MACHINE and returns it as a String.

**Constraints:**
- Use the Windows Registry API (RegOpenKeyExW, RegQueryValueExW, RegCloseKey) from the `windows` crate
- Open the key with KEY_READ access
- Verify the registry value type is REG_SZ, returning an error if not
- Handle errors by converting Win32 error codes to HRESULT and returning a windows::core::Error
- Convert the UTF-16 registry data to a Rust String

**Signature:**
```rust
fn read_registry_value(key_path: &str, value_name: &str) -> Result<String, windows::core::Error>
```

**Example:**
```rust
let key_path = r"SOFTWARE\Microsoft\Windows\CurrentVersion";
let value_name = "ProductName";
let value = read_registry_value(key_path, value_name)?;
println!("Product Name: {}", value);
```
