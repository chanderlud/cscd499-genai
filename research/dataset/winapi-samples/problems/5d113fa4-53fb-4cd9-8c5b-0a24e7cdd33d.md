1) Notify Shell of Association Change

**Spec:** Write a function that notifies the Windows shell that file associations have changed.

**Constraints:**
- Use the `SHChangeNotify` function from the `windows::Win32::UI::Shell` module.
- Use the `SHCNE_ASSOCCHANGED` event and `SHCNF_IDLIST` flag.
- The function should be safe to call (not marked `unsafe`), but may contain unsafe blocks internally.
- No return value is required.

**Signature:**
```rust
fn shell_change_notify()
```

**Example:**
```rust
fn main() {
    shell_change_notify();
}
```
