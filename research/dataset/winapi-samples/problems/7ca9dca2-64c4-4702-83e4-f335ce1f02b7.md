1) ListBox Search and Select

**Spec:** Write a function that searches a Win32 ListBox control for the first item starting with a given prefix string, selects that item, and returns its zero-based index. If no match is found, return -1.

**Constraints:**
- Use the Win32 API `SendMessageW` with the `LB_SELECTSTRING` message.
- The search must be case-insensitive and start from the beginning of the list.
- Convert the input string to a null-terminated UTF-16 wide string for the API call.

**Signature:**
```rust
unsafe fn select_listbox_item_starting_with(hwnd: HWND, prefix: &str) -> i32
```

**Example:**
```rust
// Assuming `hwnd` is a valid ListBox handle with items ["Apple", "Banana", "Cherry"]
let index = unsafe { select_listbox_item_starting_with(hwnd, "Ch") };
assert_eq!(index, 2); // "Cherry" is selected
```
