1) Minimal Win32 Window

**Spec:** Create a minimal Win32 window that displays and handles basic window messages.

**Constraints:**
- Use the `windows` crate for Win32 bindings
- Register a window class with a custom window procedure
- Create an overlapped window with standard styles
- Implement a message loop that processes messages until quit
- Handle `WM_DESTROY` to post a quit message
- Clean up by unregistering the window class

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// Running the program displays a 400x300 window titled "Minimal Window"
// at position (100,100). Closing the window terminates the program.
```
