1) Enumerate Monitors

**Spec:** Write a function that enumerates all available display monitors and returns their device names and dimensions.

**Constraints:**
- Must use the Win32 `EnumDisplayMonitors` API to enumerate monitors
- Must use `GetMonitorInfoW` to retrieve monitor information for each handle
- Must convert wide-character device names to Rust `String`
- Must return a collection of monitor information (device name, width, height)
- Must handle Win32 API errors appropriately

**Signature:**
```rust
fn available_monitors() -> Result<Vec<(String, i32, i32)>>
```

**Example:**
```rust
let monitors = available_monitors()?;
for (name, width, height) in monitors {
    println!("Monitor: {} ({}x{})", name, width, height);
}
```
