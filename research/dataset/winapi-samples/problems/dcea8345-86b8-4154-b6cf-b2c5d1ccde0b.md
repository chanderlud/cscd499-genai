N) Enumerate Network Adapters and DNS Servers

**Spec:** Write a program that enumerates all network adapters and their DNS servers, printing each adapter's description and the IP addresses of its DNS servers. For IPv6 DNS servers, exclude site-local addresses (fec0::/10).

**Constraints:** Use the `GetAdaptersAddresses` function from the `IpHelper` module. Handle both IPv4 and IPv6 addresses. The program should print the information to the console.

**Signature:** The program should be structured as a `main` function that returns a `Result<()>`.

**Example:** When run, the program should print a list of adapters and their DNS servers, for example:
```
Adapter: Ethernet0
    DNS Server (IPv4): 192.168.1.1
    DNS Server (IPv6): 2001:db8::1
```
