1) Create a Window with a Button

**Spec:** Write a Rust program that creates a Win32 window with a button control. When the button is clicked, the program must print "Button clicked!" to the console.

**Constraints:**
- Use the `windows` crate for Win32 bindings
- Register a window class with a custom window procedure
- Create a main window and a child button window
- The button must use the "BUTTON" class and styles: WS_CHILD, WS_VISIBLE, WS_TABSTOP, and BS_NOTIFY
- Handle WM_DESTROY to quit the application
- Handle WM_LBUTTONDOWN in the window procedure to print the message

**Signature:** The program must have a `main` function with the following signature:
```rust
fn main() -> windows::core::Result<()>
```

**Example:** When the program runs, a window appears with a button labeled "Click Me". Clicking the button prints "Button clicked!" to the console.
