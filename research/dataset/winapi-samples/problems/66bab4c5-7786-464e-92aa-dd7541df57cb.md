2) Abstract GetWindowLongPtr for 32/64 bit

**Spec:** Write a function that abstracts the Win32 `GetWindowLongPtrW` and `GetWindowLongW` functions to handle 32-bit and 64-bit Windows, and use it to retrieve and print the window procedure of the desktop window.

**Constraints:**
- On 64-bit, use `GetWindowLongPtrW`.
- On 32-bit, use `GetWindowLongW`.
- The function must return the value as an `isize`.

**Signature:**
```rust
fn get_window_long_ptr_w(window: HWND, index: WINDOW_LONG_PTR_INDEX) -> isize
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    let hwnd = unsafe { GetDesktopWindow() };
    let wnd_proc = get_window_long_ptr_w(hwnd, GWL_WNDPROC);
    println!("Window procedure address: {:#x}", wnd_proc);
    Ok(())
}
```
