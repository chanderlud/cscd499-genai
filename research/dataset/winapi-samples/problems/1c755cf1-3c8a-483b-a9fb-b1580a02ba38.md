1) Update Window Title from Edit Control

**Spec:** Create a Win32 window application with an edit control. When the user types in the edit control, the main window's title must be updated to match the edit control's text. If the edit control is empty, the title should be cleared.

**Constraints:**
- Use the `windows` crate for Win32 bindings
- The edit control must be a child window of the main window
- Subclass the edit control to intercept keyboard input
- Store and restore the original edit control window procedure
- Clean up allocated resources on window destruction

**Signature:**
```rust
fn main() -> Result<()>
```

// Typing "Hello" in the edit control sets the window title to "Hello"
// Clearing the edit control removes the window title
