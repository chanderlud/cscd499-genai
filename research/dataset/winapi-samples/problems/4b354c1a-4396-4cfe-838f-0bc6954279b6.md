ConPTY Function Pointer Demo

**Spec:** Dynamically load the ConPTY APIs (CreatePseudoConsole, ResizePseudoConsole, ClosePseudoConsole) from kernel32.dll and provide safe wrappers for creating, resizing, and closing a pseudo-console.

**Constraints:** 
- Must check for ConPTY API availability (Windows 10+)
- Must use dynamic loading via GetProcAddress
- Must handle cases where APIs are unavailable
- Must use Once for thread-safe initialization

**Signature:** 
```rust
pub unsafe fn create_pseudo_console(
    size: COORD,
    h_input: HANDLE,
    h_output: HANDLE,
    dw_flags: u32,
) -> Result<HPCON>;
```

**Example:** 
```rust
if is_conpty_available() {
    let size = COORD { X: 80, Y: 24 };
    let h_input = HANDLE::default();
    let h_output = HANDLE::default();
    
    match unsafe { create_pseudo_console(size, h_input, h_output, 0) } {
        Ok(h_pc) => {
            // Use pseudo-console handle
            unsafe { close_pseudo_console(h_pc) };
        }
        Err(e) => println!("Failed: {:?}", e),
    }
}
```
