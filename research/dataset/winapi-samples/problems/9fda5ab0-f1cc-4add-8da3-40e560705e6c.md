N) Get DNS Servers from Network Adapters

**Spec:** Write a function that uses the Win32 `GetAdaptersAddresses` API to retrieve all DNS server addresses configured on the system's network adapters. The function must parse both IPv4 and IPv6 addresses from the returned adapter structures and return a deduplicated list of unique DNS server IP addresses.

**Constraints:**
- Must use the `GetAdaptersAddresses` function from the `IpHelper` module
- Must handle the two-call pattern (first call to get buffer size, second call to fill buffer)
- Must parse `SOCKADDR_IN` for IPv4 and `SOCKADDR_IN6` for IPv6 addresses
- Must filter out unspecified addresses (0.0.0.0 and ::) and site-local IPv6 addresses (fec0::/10)
- Must return a deduplicated list of unique DNS servers

**Signature:**
```rust
fn get_dns_servers() -> Result<Vec<std::net::IpAddr>>
```

**Example:**
```rust
let dns_servers = get_dns_servers()?;
for (i, ip) in dns_servers.iter().enumerate() {
    println!("{}. {}", i + 1, ip);
}
```
