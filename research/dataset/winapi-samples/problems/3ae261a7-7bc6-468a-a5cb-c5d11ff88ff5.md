1) ListBox Selection and Retrieval

**Spec:** Create a window with a ListBox control, populate it with string items, programmatically select the third item, and retrieve and print the selected item's text.

**Constraints:** Use Win32 ListBox control messages (LB_ADDSTRING, LB_SETCURSEL, LB_GETCURSEL, LB_GETTEXTLEN, LB_GETTEXT) via `SendMessageW`. Handle errors for invalid indices. The program must create a window, run a message loop, and clean up properly.

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// Add items to ListBox
let items = ["Apple", "Banana", "Cherry", "Date", "Elderberry"];
for item in items {
    let wide_item = wide_null(OsStr::new(item));
    SendMessageW(listbox_hwnd, LB_ADDSTRING, Some(WPARAM(0)), Some(LPARAM(wide_item.as_ptr() as isize)));
}

// Select third item (index 2) and retrieve its text
SendMessageW(listbox_hwnd, LB_SETCURSEL, Some(WPARAM(2)), Some(LPARAM(0)));
let selected_index = SendMessageW(listbox_hwnd, LB_GETCURSEL, Some(WPARAM(0)), Some(LPARAM(0)));
if selected_index.0 != LB_ERR as isize {
    let text_len = SendMessageW(listbox_hwnd, LB_GETTEXTLEN, Some(WPARAM(selected_index.0 as usize)), Some(LPARAM(0)));
    let mut buffer = vec![0u16; (text_len.0 as usize) + 1];
    SendMessageW(listbox_hwnd, LB_GETTEXT, Some(WPARAM(selected_index.0 as usize)), Some(LPARAM(buffer.as_mut_ptr() as isize)));
    let selected_string = String::from_utf16_lossy(&buffer[..text_len.0 as usize]);
    println!("Selected: {}", selected_string);
}
```
