1) Edit Control Modified Flag

**Spec:** Create a Win32 window application that demonstrates the modified flag functionality of an edit control by programmatically setting and checking the flag state.

**Constraints:**
- Use the `windows` crate for Win32 API bindings
- Create a main window with class name "ModifiedFlagExample" and title "Edit Control Modified Flag Example"
- Create an edit control as a child window with ES_AUTOHSCROLL style
- Set a text limit of 100 characters using EM_SETLIMITTEXT
- Demonstrate EM_GETMODIFY and EM_SETMODIFY messages by:
  - Checking initial modified state (should be false)
  - Setting modified flag to true and verifying
  - Resetting modified flag to false and verifying
- Print the modified state at each step to stdout

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// Program output:
// Initial modified state: false
// After setting modified flag: true
// After resetting modified flag: false
```
