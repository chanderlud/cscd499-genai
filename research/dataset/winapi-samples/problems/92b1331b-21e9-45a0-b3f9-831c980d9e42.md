1) Hide and Show Cursor

**Spec:** Write a function that hides or shows the system cursor based on a boolean parameter, only calling the underlying Win32 API when the state actually changes.

**Constraints:**
- Use the Win32 `ShowCursor` function from the `windows` crate
- Track cursor visibility state with an atomic boolean for thread safety
- Only invoke `ShowCursor` when transitioning between hidden and visible states

**Signature:**
```rust
fn set_cursor_hidden(hidden: bool);
```

**Example:**
```rust
// Hide the cursor
set_cursor_hidden(true);

// Wait for 3 seconds
thread::sleep(Duration::from_secs(3));

// Show the cursor again
set_cursor_hidden(false);
```
