1) Set Window Dark Mode

**Spec:**
- Write a function that enables or disables dark mode for a window's title bar and non-client area using the Desktop Window Manager (DWM) API.
- The function must optionally force a redraw of the title bar after changing the mode.

**Constraints:**
- Use the `windows` crate and the `DwmSetWindowAttribute` Win32 API.
- The dark mode attribute must be set using `DWMWA_USE_IMMERSIVE_DARK_MODE`.
- Title bar redraw must be achieved by toggling the `WM_NCACTIVATE` message via `DefWindowProcW`.
- Handle both active and inactive window states correctly when redrawing.

**Signature:**
```rust
fn set_window_dark_mode(hwnd: HWND, is_dark_mode: bool, redraw_title_bar: bool) -> Result<()>
```

**Example:**
```rust
use windows::Win32::UI::Input::KeyboardAndMouse::GetActiveWindow;

let hwnd = unsafe { GetActiveWindow() };
set_window_dark_mode(hwnd, true, true)?;
```
