1) Wide String Conversion

**Spec:** Implement two functions for converting between Rust strings and Windows wide strings (UTF-16 with null terminator). The first function converts an `OsStr` to a null-terminated UTF-16 vector. The second function converts a null-terminated UTF-16 string pointer back to a Rust `String`.

**Constraints:**
- The first function must accept an `&OsStr` and return a `Vec<u16>` containing the UTF-16 representation with an appended null terminator.
- The second function must accept a `*const u16` pointer to a null-terminated UTF-16 string and return a `String`.
- The second function must panic if the pointer is null.
- The second function must handle invalid UTF-16 sequences via lossy conversion.

**Signature:**
```rust
fn wide_null(s: &std::ffi::OsStr) -> Vec<u16>;
fn from_wide_ptr(ptr: *const u16) -> String;
```

**Example:**
```rust
let original = "Hello, Windows!";
let wide = wide_null(std::ffi::OsStr::new(original));
let converted = from_wide_ptr(wide.as_ptr());
assert_eq!(original, converted);
```
