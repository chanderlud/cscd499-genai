**Spec:**
- Write a function that retrieves the current sentence mode of the Input Method Editor (IME) for the foreground window.
- Use `GetForegroundWindow` to obtain the active window handle.
- Use `ImmGetDefaultIMEWnd` to get the IME window handle.
- Send `WM_IME_CONTROL` with `IMC_GETSENTENCEMODE` (0x0008) via `SendMessageA` to query the mode.
- Return the sentence mode as an integer.

**Constraints:**
- Must use the `windows` crate and Win32 API functions.
- Return `windows::core::Result<i32>` where the integer represents the sentence mode.
- Handle potential invalid window handles gracefully by returning an error.
- The function must be safe to call from any thread.

**Signature:**
```rust
fn get_ime_sentence_mode() -> windows::core::Result<i32>
```

**Example:**
```rust
let mode = get_ime_sentence_mode()?;
println!("Current IME sentence mode: {}", mode);
```
