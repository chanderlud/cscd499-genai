1) System Memory Information

**Spec:** Write a function that retrieves and displays current system memory statistics using the Win32 API, including total and available physical memory, page file, and virtual memory in both bytes and gigabytes, along with usage percentages for physical and virtual memory.

**Constraints:**
- Must use the `GlobalMemoryStatusEx` Win32 function
- Must properly initialize the `MEMORYSTATUSEX` structure with its size
- Must handle API errors via `Result<()>` return type
- Must format output exactly as shown in the example

**Signature:**
```rust
fn print_memory_status() -> Result<()>
```

**Example:**
```rust
fn main() -> Result<()> {
    print_memory_status()
}
```
Output:
```
=== System Memory Information ===
Total Physical Memory: 17179869184 bytes (16.00 GB)
Available Physical Memory: 1234567890 bytes (1.15 GB)
...
Memory Usage Summary:
  Physical Memory: 7.18% used (1.15 GB used / 16.00 GB total)
  Virtual Memory: 25.00% used (4.00 GB used / 16.00 GB total)
```
