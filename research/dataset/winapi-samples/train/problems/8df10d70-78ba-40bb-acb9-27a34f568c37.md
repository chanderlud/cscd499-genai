1) Get Current Module Base Address

**Spec:** Write a function that returns the base address of the current executable module as a `usize` using the Win32 API.

**Constraints:** Must use `GetModuleHandleW` from the `windows` crate with a null module name to obtain the handle to the current process's executable. The function should return a `Result` to handle potential API failures.

**Signature:**
```rust
fn get_base_address() -> Result<usize>
```

**Example:**
```rust
fn main() {
    match get_base_address() {
        Ok(base_addr) => println!("Base address: 0x{:x}", base_addr),
        Err(e) => eprintln!("Failed to get base address: {}", e),
    }
}
```
