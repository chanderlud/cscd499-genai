1) Create and Manage a Pseudo Console

**Spec:** Write a function that creates a pseudo console (ConPTY) with a specified size, launches a command-line process attached to it, waits for the process to exit, and returns the exit code. The function must properly initialize the extended startup info with the pseudo console attribute and clean up all allocated resources.

**Constraints:**
- Use the `windows` crate and Win32 API for pseudo console operations
- Must create anonymous pipes for the pseudo console's input/output
- Must use `STARTUPINFOEXW` with `PROC_THREAD_ATTRIBUTE_PSEUDOCONSOLE` to attach the process
- Must wait for the process to complete and retrieve its exit code
- Must close all handles and the pseudo console before returning

**Signature:**
```rust
fn run_command_in_pseudo_console(command: &str, size: COORD) -> Result<u32>
```

**Example:**
```rust
let size = COORD { X: 80, Y: 25 };
let exit_code = run_command_in_pseudo_console("cmd.exe /c echo Hello", size)?;
assert_eq!(exit_code, 0);
```
