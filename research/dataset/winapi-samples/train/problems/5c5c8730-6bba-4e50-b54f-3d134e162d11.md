1) Get Primary Monitor Information

**Spec:** Write a program that retrieves and displays the primary monitor's device name and dimensions using Win32 APIs.

**Constraints:**
- Use `MonitorFromPoint` with the origin (0,0) and `MONITOR_DEFAULTTOPRIMARY` to get the primary monitor handle
- Use `GetMonitorInfoW` to retrieve monitor information into a `MONITORINFOEXW` structure
- Extract the device name from the `szDevice` field and calculate width/height from the monitor rectangle
- Print the device name, dimensions (width × height), and the default screen DPI (96)
- Handle Win32 API calls safely with proper error checking

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// Running the program should output something like:
// Primary Monitor:
//   Device: \\.\DISPLAY1
//   Size: 1920x1080
//   Default DPI: 96
```
