**Spec:** Implement a function that changes a local user's password using `NetUserChangePassword`, then verifies the change by retrieving the user's info with `NetUserGetInfo` (level 3) and confirming the password age is reset to zero. If the password age is not zero, the function returns an error.

**Constraints:**
- Must use `NetUserChangePassword` and `NetUserGetInfo` (with level 3) from the NetManagement API.
- The function should handle string conversions to wide strings (UTF-16) for API calls.
- Must free the buffer allocated by `NetUserGetInfo` using `NetApiBufferFree`.
- Return `windows::core::Result<()>` with appropriate error handling for API failures and if the password age is not zero.
- Assume the caller has sufficient privileges to change the target user's password.

**Signature:**
```rust
fn change_user_password_and_verify(
    username: &str,
    old_password: &str,
    new_password: &str,
) -> windows::core::Result<()>
```

**Example:**
```rust
// Change password for user "testuser" from "OldPass123!" to "NewPass456!"
change_user_password_and_verify("testuser", "OldPass123!", "NewPass456!")?;
println!("Password changed and verified successfully");
```
