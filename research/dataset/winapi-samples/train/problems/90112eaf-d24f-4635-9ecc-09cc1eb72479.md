**Spec:**
- The function must take a process ID and return a list of thread IDs that were terminated.
- It must use ToolHelp32 snapshots to enumerate threads (`TH32CS_SNAPTHREAD`).
- It must identify the main thread (the one with the earliest creation time) and skip it.
- It must terminate each non-main thread using `TerminateThread`.
- It must return a `Result` containing a vector of terminated thread IDs.

**Constraints:**
- Use the Windows API via the `windows` crate.
- The function must properly handle snapshot creation and cleanup using RAII guards.
- Thread enumeration must use `TH32CS_SNAPTHREAD` with `CreateToolhelp32Snapshot`.
- To get the creation time of a thread, open it with `THREAD_QUERY_INFORMATION` and use `GetThreadTimes`.
- Thread termination must be done by opening the thread with `THREAD_TERMINATE` and then calling `TerminateThread`.
- The main thread must be identified by comparing creation times from thread information.

**Signature:**
```rust
pub fn terminate_process_threads(process_id: u32) -> windows::core::Result<Vec<u32>>
```
