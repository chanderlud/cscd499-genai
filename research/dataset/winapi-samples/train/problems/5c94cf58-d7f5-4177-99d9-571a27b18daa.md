**Spec:**
- Implement a function that schedules a toast notification to appear after a specified delay.
- The function must create a `ScheduledToastNotification` from an XML template containing title and body text.
- The notification must be scheduled using `ToastNotificationManager` with the provided `app_id`.
- The delay is specified as a `std::time::Duration` from the current time.

**Constraints:**
- Must use `windows::UI::Notifications::ScheduledToastNotification` for scheduling.
- Must handle COM initialization internally (call `CoInitializeEx` with `COINIT_APARTMENTTHREADED`).
- Must properly escape XML content in title and body.
- Must return `windows::core::Result<()>` indicating success or failure.

**Signature:**
```rust
pub fn schedule_toast(app_id: &str, title: &str, body: &str, delay: std::time::Duration) -> windows::core::Result<()>
```

**Example:**
```rust
// Schedule a toast to appear in 5 minutes
let delay = std::time::Duration::from_secs(300);
schedule_toast("MyAppId", "Reminder", "Time to stretch!", delay)?;
```
