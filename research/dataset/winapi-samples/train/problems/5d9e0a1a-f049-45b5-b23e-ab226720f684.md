**Spec:**
- Write a function that generates cryptographically secure random bytes using only FIPS-approved algorithms
- The function must first verify that Windows is operating in FIPS mode
- If FIPS mode is not enabled, return an error
- Use the CNG API to open a FIPS-compliant random number generator provider
- Generate the requested number of random bytes using this provider
- Ensure proper cleanup of cryptographic handles

**Constraints:**
- Must use `BCryptGetFipsAlgorithmMode` to check FIPS status
- Must use `BCryptOpenAlgorithmProvider` with `BCRYPT_RNG_ALGORITHM` and `BCRYPT_OPEN_ALGORITHM_PROVIDER_FLAGS(0)`
- Must use `BCryptGenRandom` with `BCRYPTGENRANDOM_FLAGS(0)`
- Must use `BCryptCloseAlgorithmProvider` for cleanup
- Return `windows::core::Result<Vec<u8>>` with the random bytes on success
- Return an error with `ERROR_INVALID_FUNCTION` if FIPS mode is not enabled, or an appropriate error if any CNG operation fails

**Signature:**
```rust
pub fn generate_fips_random_bytes(byte_count: usize) -> windows::core::Result<Vec<u8>>
```

**Example:**
```rust
// Generate 32 random bytes using FIPS-approved RNG
let random_bytes = generate_fips_random_bytes(32)?;
println!("Generated {} random bytes", random_bytes.len());
```
