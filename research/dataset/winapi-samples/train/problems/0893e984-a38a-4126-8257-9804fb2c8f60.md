**Spec:**
- Write a function that removes all toast notifications for all applications
- Must use the Windows ToastNotificationManager and its History API
- Should initialize COM (apartment threaded) and clean up properly

**Constraints:**
- Must return `windows::core::Result<()>` for error handling
- Must handle COM initialization and uninitialization
- Should work on Windows 8.1 and later (where History API is available)

**Signature:**
```rust
fn clear_all_toasts(app_id: &str) -> windows::core::Result<()>;
```

**Example:**
```rust
// Clear all toasts for all applications (app_id parameter is unused)
clear_all_toasts("{1AC14E77-02E7-4E5D-B744-2EB1AE5198B7}\\WindowsPowerShell\\v1.0\\powershell.exe")?;
```
