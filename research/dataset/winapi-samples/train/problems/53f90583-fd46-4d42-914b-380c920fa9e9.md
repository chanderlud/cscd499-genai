1) Pageant IPC Client

**Spec:** Write a function that sends a message to the Pageant SSH agent window via shared memory and WM_COPYDATA, then returns the agent's response.

**Constraints:**
- Must use named shared memory (file mapping) with a fixed name and size (16384 bytes)
- Must write message data prefixed with a 4-byte big-endian length
- Must locate Pageant window by class and title ("Pageant")
- Must use COPYDATASTRUCT with magic value 0x804e50ba
- Must clean up all handles and mappings
- Must return Result<Vec<u8>> with response data

**Signature:**
```rust
fn send_to_pageant(message: &[u8]) -> Result<Vec<u8>>
```

**Example:**
```rust
let response = send_to_pageant(b"Hello Pageant!")?;
println!("Agent response: {}", String::from_utf8_lossy(&response));
```
