1) Retrieve RSS Feed Titles

**Spec:** Write an async function that uses the `SyndicationClient` to retrieve an RSS feed from a given URL and prints the title of each item.

**Constraints:**
- Use the `windows` crate and the `SyndicationClient` from `Windows.Web.Syndication`
- Set a custom User-Agent header before retrieving the feed
- The function must be async and return a `Result<()>`
- Use the `futures` crate to block on the async function in `main`

**Signature:**
```rust
async fn print_feed_titles(url: &str, user_agent: &str) -> Result<()>
```

**Example:**
```rust
fn main() -> Result<()> {
    futures::executor::block_on(print_feed_titles(
        "https://blogs.windows.com/feed",
        "Mozilla/5.0 (compatible; MSIE 10.0; Windows NT 6.2; WOW64; Trident/6.0)"
    ))
}
```
