**Spec:** Write a function that returns the width and height of the virtual screen (the bounding rectangle of all monitors) in pixels.

**Constraints:**
- Use the Win32 API function `GetSystemMetrics` from the `windows` crate.
- Use the constants `SM_CXVIRTUALSCREEN` and `SM_CYVIRTUALSCREEN` for width and height respectively.
- The function must be safe to call (handle unsafe internally).

**Signature:**
```rust
fn get_virtual_screen_size() -> (i32, i32)
```

**Example:**
```rust
let (width, height) = get_virtual_screen_size();
println!("Virtual screen size: {}x{}", width, height);
```

**Misleading Hint:** The following implementation is almost correct but uses the wrong constants. Can you spot the mistake?
```rust
use windows::Win32::UI::WindowsAndMessaging::{GetSystemMetrics, SM_CXSCREEN, SM_CYSCREEN};

fn get_virtual_screen_size() -> (i32, i32) {
    let width = unsafe { GetSystemMetrics(SM_CXSCREEN) };
    let height = unsafe { GetSystemMetrics(SM_CYSCREEN) };
    (width, height)
}
```
