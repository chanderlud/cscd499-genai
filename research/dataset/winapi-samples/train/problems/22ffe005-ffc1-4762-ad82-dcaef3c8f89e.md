1) Set Desktop Wallpaper
**Spec:** Write a function that sets the desktop wallpaper to the image at the given file path.
**Constraints:**
- Use the Win32 API `SystemParametersInfoW` with `SPI_SETDESKWALLPAPER`.
- Convert the image path to a null-terminated wide string (UTF-16) for the API call.
- Update the user profile and send a change notification (using `SPIF_UPDATEINIFILE | SPIF_SENDCHANGE`).
- Return a `windows::core::Result<()>`.
**Signature:**
```rust
fn set_desktop_wallpaper(image_path: &str) -> windows::core::Result<()>;
```
**Example:**
```rust
fn main() -> windows::core::Result<()> {
    set_desktop_wallpaper("C:\\path\\to\\your\\image.jpg")?;
    Ok(())
}
```
