**Spec:** Implement helper functions for creating and using Windows events. Create a manual-reset event with a given name, wait for the event to be signaled, and close the event handle. Additionally, demonstrate error handling by converting a Win32 error code to an HRESULT and then to a `windows::core::Error`. Show how to safely pass an event handle across threads by converting it to a `usize` and back.
**Constraints:**
- Use `CreateEventW` to create a manual-reset event.
- Use `WaitForSingleObject` with an infinite timeout.
- The event name is provided as an `OsStr` and must be converted to a wide null-terminated string.
- The functions must return `windows::core::Result<T>` for fallible operations.
**Signature:**
```rust
fn wide_null(s: &OsStr) -> Vec<u16>
fn create_event_manually() -> windows::core::Result<windows::Win32::Foundation::HANDLE>
fn wait_for_event(handle: windows::Win32::Foundation::HANDLE) -> windows::core::Result<()>
fn close_event_handle(handle: windows::Win32::Foundation::HANDLE) -> windows::core::Result<()>
fn example_hresult_conversion() -> windows::core::Result<()>
fn thread_safe_example() -> windows::core::Result<()>
```
**Example:**
```rust
// Example of creating an event and waiting for it
let event_handle = create_event_manually()?;
// ... set the event somewhere else ...
wait_for_event(event_handle)?;
close_event_handle(event_handle)?;

// Example of error conversion
example_hresult_conversion()?;

// Example of thread-safe handle passing
thread_safe_example()?;
```
