Find Process ID by Name

**Spec:** Write a function that returns the process ID of the first instance of a given process name in the current snapshot of running processes.

**Constraints:**
- Use the Windows Toolhelp API (`CreateToolhelp32Snapshot`, `Process32FirstW`, `Process32NextW`).
- The function must return a `Result<u32>` with the PID on success, or an error if the process is not found or if any API call fails.
- The snapshot handle must be closed with `CloseHandle` before returning.
- The function must be safe to call (i.e., encapsulate any unsafe operations).

**Signature:**
```rust
fn find_process_id(process_name: &str) -> windows::core::Result<u32>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    let pid = find_process_id("explorer.exe")?;
    println!("Found explorer.exe with PID: {}", pid);
    Ok(())
}
```
