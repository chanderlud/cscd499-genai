Write a function that retrieves all local groups a specified user account belongs to on the local system.

**Spec:**
- Accept a username string and return a vector of group name strings
- Use `NetUserGetLocalGroups` with level 0 (`LOCALGROUP_USERS_INFO_0`)
- Handle buffer allocation and reallocation for `ERROR_MORE_DATA` by doubling the buffer size
- Properly free the API-allocated buffer with `NetApiBufferFree` in both success and error paths

**Constraints:**
- Must handle usernames that don't exist (return appropriate error from the API)
- Must work with Unicode (UTF-16) strings for Windows API compatibility
- Must properly manage memory and avoid leaks
- Should handle both local and domain accounts

**Signature:**
```rust
fn get_user_local_groups(username: &str) -> windows::core::Result<Vec<String>>
```

**Example:**
```rust
let groups = get_user_local_groups("test")?;
for group in groups {
    println!("User belongs to: {}", group);
}
```
