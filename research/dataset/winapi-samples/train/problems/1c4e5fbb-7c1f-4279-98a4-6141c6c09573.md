1) Get System Power Status

**Spec:** Write a program that retrieves and displays the system's current power status using the Windows API, including AC line status, battery presence, battery life percentage, estimated runtime, and battery flag details.

**Constraints:**
- Must use the `GetSystemPowerStatus` function from the `windows::Win32::System::Power` module
- Must handle the `SYSTEM_POWER_STATUS` structure and interpret its fields correctly
- Must properly handle unknown values (255 for battery percent, 0 for battery life time)
- Must use unsafe block for the Win32 API call
- Must return `windows::core::Result<()>` from main

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
use windows::core::Result;
use windows::Win32::System::Power::{GetSystemPowerStatus, SYSTEM_POWER_STATUS};

fn main() -> Result<()> {
    let mut status = SYSTEM_POWER_STATUS::default();
    unsafe { GetSystemPowerStatus(&mut status)?; }
    
    // Display power status information
    println!("AC Line Status: {}", if status.ACLineStatus == 1 { "Online" } else { "Offline" });
    println!("Battery Present: {}", if status.BatteryFlag != 0 { "Yes" } else { "No" });
    // ... additional output formatting
    Ok(())
}
```
