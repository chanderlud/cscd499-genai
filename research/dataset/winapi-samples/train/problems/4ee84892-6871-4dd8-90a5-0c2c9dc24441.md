**Spec:**
- Write a function that sets the IME composition window position for the foreground window to the specified screen coordinates.
- Use `GetForegroundWindow` to obtain the foreground window handle.
- Use `ImmGetContext` to obtain the IME context for the foreground window.
- Use `ImmSetCompositionWindow` with `CFS_POINT` style to position the window.
- Release the IME context with `ImmReleaseContext` after setting.

**Constraints:**
- Must handle cases where the foreground window is invalid (return appropriate error).
- Must handle cases where the foreground window has no IME context (return appropriate error).
- Must validate that coordinates are non-negative (return `E_INVALIDARG` if not).
- Must use `windows::core::Result<()>` for error handling.

**Signature:**
```rust
fn set_ime_composition_position(x: i32, y: i32) -> windows::core::Result<()>
```

**Example:**
```rust
// Position the composition window at screen coordinates (100, 200)
set_ime_composition_position(100, 200)?;
```
