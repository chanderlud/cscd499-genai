**Spec:**
- Write a function that uses `NetUserEnum` with level 1 and filter `FILTER_NORMAL_ACCOUNT` to retrieve all local user accounts.
- For each user, extract the username (`usri1_name`) and comment (`usri1_comment`).
- Return a vector of tuples containing the username and comment as Rust `String`s.
- Handle buffer allocation/deallocation using `NetApiBufferFree`.
- Must handle `ERROR_MORE_DATA` by retrying with a larger buffer (doubling buffer size) and resetting the resume handle.

**Constraints:**
- Must handle `ERROR_MORE_DATA` by retrying with a larger buffer.
- Must properly convert `PWSTR` fields to Rust strings.
- Must return `windows::core::Result<Vec<(String, String)>>`.

**Signature:**
```rust
fn enumerate_users() -> windows::core::Result<Vec<(String, String)>>
```

**Example:**
```rust
let users = enumerate_users()?;
for (name, comment) in users {
    println!("User: {}, Comment: {}", name, comment);
}
```
