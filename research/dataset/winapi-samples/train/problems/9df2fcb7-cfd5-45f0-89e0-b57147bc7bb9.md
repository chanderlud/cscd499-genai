N) Console Handles and Raw Mode

**Spec:** Implement a `ConsoleSession` struct that retrieves and manages Windows standard console handles (stdin, stdout, stderr). The struct must provide methods to query and set console modes, check for pending input without blocking, and configure stdin for raw mode input processing.

**Constraints:**
- Use the `windows` crate for Win32 API bindings
- Validate all retrieved handles (reject `INVALID_HANDLE_VALUE`)
- Use `WaitForSingleObject` with zero timeout for non-blocking input checks
- Raw mode must disable: echo, line input, mouse input, processed input
- Raw mode must enable: extended flags, insert mode, quick edit mode, virtual terminal input
- All operations must return `WinResult` for proper error handling

**Signature:**
```rust
use windows::core::Result as WinResult;
use windows::Win32::Foundation::HANDLE;
use windows::Win32::System::Console::CONSOLE_MODE;

#[derive(Debug, Clone)]
struct ConsoleSession {
    stdin: HANDLE,
    stdout: HANDLE,
    stderr: HANDLE,
}

impl ConsoleSession {
    fn new() -> WinResult<Self>;
    fn get_console_mode(&self, handle: HANDLE) -> WinResult<CONSOLE_MODE>;
    fn set_console_mode(&self, handle: HANDLE, mode: CONSOLE_MODE) -> WinResult<()>;
    fn is_stdin_empty(&self) -> WinResult<bool>;
    fn set_raw_stdin(&self) -> WinResult<()>;
}
```

**Example:**
```rust
fn main() -> WinResult<()> {
    let console = ConsoleSession::new()?;
    
    // Check for input without blocking
    if console.is_stdin_empty()? {
        println!("No input available");
    }
    
    // Set raw mode and restore original
    let original_mode = console.get_console_mode(console.stdin)?;
    console.set_raw_stdin()?;
    // ... read raw input ...
    console.set_console_mode(console.stdin, original_mode)?;
    
    Ok(())
}
```
