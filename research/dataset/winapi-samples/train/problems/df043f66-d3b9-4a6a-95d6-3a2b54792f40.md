**Spec:**
- Implement a function that generates cryptographically secure random bytes using the Windows BCrypt API.
- The function must open the RNG algorithm provider, generate the requested number of random bytes, and properly close the provider.
- Return the random bytes as a `Vec<u8>`.

**Constraints:**
- Use `BCryptOpenAlgorithmProvider` with `BCRYPT_RNG_ALGORITHM` and default flags.
- Use `BCryptGenRandom` to fill a buffer of the specified length.
- Ensure the algorithm provider is closed via `BCryptCloseAlgorithmProvider` after generation.
- Handle all BCrypt errors and convert them to `windows::core::Error`.

**Signature:**
```rust
fn generate_random_bytes(len: usize) -> windows::core::Result<Vec<u8>>
```

**Example:**
```rust
let random_bytes = generate_random_bytes(32)?;
assert_eq!(random_bytes.len(), 32);
// Use the random_bytes for cryptographic purposes
```
