1) Console Mode Restoration

**Spec:**
Implement a `ConsoleSession` struct that captures and restores original console modes for stdin, stdout, and stderr handles, with the ability to set stdin to raw mode and check for pending input without blocking.

**Constraints:**
- Use the `windows` crate for Win32 API calls
- Must retrieve standard handles using `GetStdHandle`
- Must query current console modes using `GetConsoleMode`
- Raw mode must disable echo, line input, mouse input, and processed input while enabling extended flags, insert mode, quick edit mode, and virtual terminal input
- Must restore all original modes on demand
- Must check for pending stdin input using `WaitForSingleObject` with zero timeout

**Signature:**
```rust
use windows::core::Result;
use windows::Win32::Foundation::HANDLE;
use windows::Win32::System::Console::CONSOLE_MODE;

struct ConsoleSession {
    // ... fields ...
}

impl ConsoleSession {
    fn current() -> Result<Self>;
    fn set_raw_stdin(&self) -> Result<()>;
    fn restore_modes(&self) -> Result<()>;
    fn is_stdin_empty(&self) -> Result<bool>;
}
```

**Example:**
```rust
fn main() -> Result<()> {
    let console = ConsoleSession::current()?;
    console.set_raw_stdin()?;
    // ... perform raw mode operations ...
    console.restore_modes()?;
    Ok(())
}
```
