1) Disk Free Space

**Spec:** Write a function that retrieves disk space information for a given drive path using the Win32 API, returning the free bytes available to the caller, total bytes on disk, and total free bytes on disk.

**Constraints:**
- Must use `GetDiskFreeSpaceExW` from the Windows API
- Must handle unsafe FFI call appropriately
- Must return results via mutable output parameters
- Must use wide string for the drive path

**Signature:**
```rust
use windows::core::{PCWSTR, Result};

unsafe fn get_disk_free_space(
    drive_path: PCWSTR,
    free_bytes_available: Option<&mut u64>,
    total_bytes: Option<&mut u64>,
    total_free_bytes: Option<&mut u64>,
) -> Result<()>;
```

**Example:**
```rust
use windows::core::w;

let mut free = 0u64;
let mut total = 0u64;
let mut total_free = 0u64;

unsafe {
    get_disk_free_space(
        w!("C:\\"),
        Some(&mut free),
        Some(&mut total),
        Some(&mut total_free),
    ).unwrap();
}

println!("Free: {}, Total: {}, Total Free: {}", free, total, total_free);
```
