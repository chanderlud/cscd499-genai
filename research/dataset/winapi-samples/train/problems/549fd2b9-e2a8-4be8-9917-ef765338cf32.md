**Spec:** Implement a function that computes an HMAC (Hash-based Message Authentication Code) using the BCrypt API. The function should take a key, message, and algorithm choice, then return the HMAC digest.

**Constraints:**
- Use `BCryptOpenAlgorithmProvider` with `BCRYPT_ALG_HANDLE_HMAC_FLAG` to get an HMAC algorithm handle
- Use `BCryptCreateHash` with the algorithm handle and key to initialize the HMAC context
- Support SHA-256, SHA-384, and SHA-512 HMAC algorithms
- Handle the hash context lifecycle correctly (create, update, finish, destroy)
- Return the result as a `Vec<u8>` with length matching the algorithm's output length

**Signature:**
```rust
#[derive(Debug, Clone, Copy)]
pub enum HmacAlgorithm {
    Sha256,
    Sha384,
    Sha512,
}

pub fn compute_hmac(
    algorithm: HmacAlgorithm,
    key: &[u8],
    message: &[u8],
) -> windows::core::Result<Vec<u8>>
```

**Example:**
```rust
let key = b"secret-key";
let message = b"authenticate this";
let hmac = compute_hmac(HmacAlgorithm::Sha256, key, message)?;
println!("HMAC-SHA256: {:02x?}", hmac);
```
