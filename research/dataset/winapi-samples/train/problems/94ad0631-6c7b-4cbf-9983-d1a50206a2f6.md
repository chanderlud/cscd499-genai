**Spec:**
- Get the foreground window handle using `GetForegroundWindow`.
- Obtain the default IME window handle for the foreground window using `ImmGetDefaultIMEWnd`.
- Send a message to query the current conversion mode using `SendMessageW` with `WM_IME_CONTROL` and a locally defined `IMC_GETCONVERSIONMODE` constant.
- Return the conversion mode value as an integer.

**Constraints:**
- Use `GetForegroundWindow`, `ImmGetDefaultIMEWnd`, and `SendMessageW` with `WM_IME_CONTROL`.
- Define `IMC_GETCONVERSIONMODE` as a constant with value `0x0001` since it is not provided by the windows crate.
- Return a `windows::core::Result` with the conversion mode value on success.
- Handle potential null/invalid window handles by returning an appropriate error using `Error::from_thread()`.

**Signature:**
```rust
fn get_ime_conversion_mode() -> windows::core::Result<i32>
```

**Example:**
```rust
match get_ime_conversion_mode() {
    Ok(mode) => println!("IME conversion mode: {}", mode),
    Err(e) => eprintln!("Failed to get IME conversion mode: {}", e),
}
```
