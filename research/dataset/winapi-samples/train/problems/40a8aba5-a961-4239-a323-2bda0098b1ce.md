1) Check if the current process is running as administrator

**Spec:** Write a function that determines whether the current process is running with administrator privileges by checking its token elevation status.

**Constraints:**
- Must use Win32 token APIs to query elevation information
- Must properly handle token opening, querying, and cleanup
- Should return `true` if elevated, `false` otherwise or on any error

**Signature:**
```rust
pub fn is_admin() -> bool
```

**Example:**
```rust
if is_admin() {
    println!("Running as administrator");
} else {
    println!("Not running as administrator");
}
```
