**Spec:** Write a function that modifies an existing user's account flags (e.g., disabling the account) and then retrieves the user's information to verify the change was applied.

**Constraints:**
- Use `NetUserSetInfo` with level 1 to set the `usri1_flags` field.
- Use `NetUserGetInfo` with level 1 to retrieve the updated user information.
- The function must handle errors from the API calls (including the case where the user does not exist) and return an appropriate error.
- The function must verify that the flags were set correctly by comparing the retrieved flags with the set flags, and return an error if they don't match.
- Properly manage memory for the retrieved user info buffer using `NetApiBufferFree`.
- Return a `windows::core::Result<()>` indicating success or failure.

**Signature:**
```rust
fn set_and_verify_user_flags(server: Option<&str>, username: &str, flags_to_set: u32) -> windows::core::Result<()>
```

**Example:**
```rust
// Disable the user account by setting the UF_ACCOUNTDISABLE flag
set_and_verify_user_flags(None, "test_user", 0x0002)?;
```
