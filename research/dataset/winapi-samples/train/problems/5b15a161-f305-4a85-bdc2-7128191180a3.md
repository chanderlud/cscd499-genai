1) Get Primary Monitor Rectangle

**Spec:** Write a function that sets the process as DPI aware and returns the rectangle of the primary monitor.

**Constraints:**
- Must call `SetProcessDPIAware` before querying monitor information
- Use `MonitorFromWindow` with a null window handle and `MONITOR_DEFAULTTONEAREST` to get the primary monitor handle
- Use `GetMonitorInfoW` to retrieve the monitor information
- Return the monitor rectangle (the `rcMonitor` field from `MONITORINFO`)

**Signature:**
```rust
fn get_primary_monitor_rect() -> windows::core::Result<windows::Win32::Foundation::RECT>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    let rect = get_primary_monitor_rect()?;
    println!("Primary monitor rectangle:");
    println!("  Left: {}", rect.left);
    println!("  Top: {}", rect.top);
    println!("  Right: {}", rect.right);
    println!("  Bottom: {}", rect.bottom);
    println!("  Width: {}", rect.right - rect.left);
    println!("  Height: {}", rect.bottom - rect.top);
    Ok(())
}
```
