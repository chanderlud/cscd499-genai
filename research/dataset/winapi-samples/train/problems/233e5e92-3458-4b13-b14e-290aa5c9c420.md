**Spec:**
- Create a layered overlay window that magnifies a screen region beneath the cursor.
- The magnifier must follow the cursor position in real-time.
- Implement dynamic zoom control via mouse wheel (1x to 10x magnification).
- Use WS_EX_LAYERED with per-pixel alpha for smooth transparency at edges.

**Constraints:**
- Must use `GetCursorPos` and `GetDC`/`ReleaseDC` for screen capture.
- Must use `BitBlt` for magnification rendering.
- Must handle `WM_MOUSEWHEEL` for zoom adjustments.
- Must maintain aspect ratio during magnification.
- Window must be click-through (WS_EX_TRANSPARENT) and topmost.

**Signature:**
```rust
pub fn create_magnifier_overlay(width: i32, height: i32, initial_zoom: f32) -> windows::core::Result<HWND>
```

**Example:**
```rust
let hwnd = create_magnifier_overlay(300, 200, 2.0)?;
// Window automatically follows cursor and responds to mouse wheel
```
