1) System Tray Icon Management
**Spec:** Write a function that creates a system tray icon, modifies it after a delay, and then removes it, using the Windows Shell_NotifyIcon API.
**Constraints:**
- Must use `Shell_NotifyIconW` with `NIM_ADD`, `NIM_MODIFY`, and `NIM_DELETE` operations
- Must set icon, callback message, and tooltip for the notification icon
- Must use a hidden window to receive tray icon messages
- Must include a basic window procedure that handles `WM_DESTROY`
**Signature:**
```rust
unsafe fn manage_tray_icon(hwnd: HWND, icon1: HICON, icon2: HICON, tip1: &str, tip2: &str) -> Result<()>
```
**Example:**
```rust
let hwnd = /* create hidden window */;
let icon1 = LoadIconW(None, IDI_APPLICATION)?;
let icon2 = LoadIconW(None, IDI_APPLICATION)?;
unsafe { manage_tray_icon(hwnd, icon1, icon2, "Initial", "Modified")? };
```
