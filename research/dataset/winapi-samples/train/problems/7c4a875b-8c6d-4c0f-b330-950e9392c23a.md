**Spec:**
- Write a function that removes a specific toast notification from the notification history for the current application.
- The function must identify the toast by its tag and group (group is optional).
- It must initialize COM with `COINIT_APARTMENTTHREADED` before any WinRT operations and uninitialize after.
- It must use the `ToastNotificationManager` to access the notification history and remove the specified toast.

**Constraints:**
- Use the `windows` crate for WinRT toast notification APIs.
- The function must return `windows::core::Result<()>`.
- The group parameter is optional; if `None`, the toast is removed from the default group.
- The function must handle the case where the toast does not exist gracefully (return `Ok(())`).

**Signature:**
```rust
fn remove_toast_by_tag(app_id: &str, tag: &str, group: Option<&str>) -> windows::core::Result<()>
```

**Example:**
```rust
// Remove a toast with tag "download_complete" from the default group
remove_toast_by_tag("MyAppId", "download_complete", None)?;

// Remove a toast with tag "message" from group "chats"
remove_toast_by_tag("MyAppId", "message", Some("chats"))?;
```
