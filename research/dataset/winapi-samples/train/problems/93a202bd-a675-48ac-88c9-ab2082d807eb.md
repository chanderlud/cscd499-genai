1) Read game menu active state from fixed memory address
**Spec:** Write a function that reads a boolean value from a fixed memory address (0xBA67A4) representing the menu active state in the GTA: San Andreas process.
**Constraints:** 
- Must use unsafe Rust to dereference a raw pointer to the fixed address
- The address is hardcoded and assumed to be valid in the target process context
- No error handling is required for the memory read operation
**Signature:** 
```rust
pub fn is_gta_menu_active() -> bool
```
**Example:**
```rust
fn main() {
    let active = is_gta_menu_active();
    println!("GTA menu active: {}", active);
}
```
