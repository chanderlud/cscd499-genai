**Spec:** Write a function that removes a specified user from a local group and then deletes the user account entirely. The function should handle the operations in sequence, returning appropriate errors if either step fails. If the user is not a member of the group or the group doesn't exist, the function should ignore those errors and proceed to delete the user account.

**Constraints:**
- Must use `NetLocalGroupDelMembers` to remove the user from the group
- Must use `NetUserDel` to delete the user account
- Should ignore errors when the user is not a member of the group (`NERR_MemberNotFound`) or the group doesn't exist (`NERR_GroupNotFound`)
- Must properly manage UTF-16 string conversions for Windows API calls
- Should use the local machine (server name as empty string)

**Signature:**
```rust
fn remove_user_from_group_and_delete(username: &str, groupname: &str) -> windows::core::Result<()>
```

**Example:**
```rust
// Remove user "testuser" from "Users" group and delete the account
remove_user_from_group_and_delete("testuser", "Users")?;
println!("User removed from group and deleted successfully");
```
