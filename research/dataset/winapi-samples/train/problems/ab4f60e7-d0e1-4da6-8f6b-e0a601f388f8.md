**Spec:**
- Implement a function that enumerates all heap blocks for a given process ID using Toolhelp snapshots.
- For each heap block, invoke a provided callback with the block's address, size, and flags.
- Do not allocate heap memory (no `Vec`, `String`, etc.) during enumeration.

**Constraints:**
- Use `CreateToolhelp32Snapshot` with `TH32CS_SNAPHEAPLIST`.
- Iterate through heap lists with `Heap32ListFirst`/`Heap32ListNext`.
- For each heap list, iterate through heap entries with `Heap32First`/`Heap32Next`.
- Close the snapshot handle after enumeration.
- Return `windows::core::Result<()>`.
- The callback must be invoked for each block in the order they are enumerated.
- The function must not allocate any heap memory (no `Vec`, `Box`, `String`, etc.).

**Signature:**
```rust
#[derive(Debug)]
pub struct HeapBlock {
    pub address: usize,
    pub size: usize,
    pub flags: u32,
}

pub fn for_each_heap_block<F>(process_id: u32, mut callback: F) -> windows::core::Result<()>
where
    F: FnMut(HeapBlock) -> windows::core::Result<()>,
```

**Example:**
```rust
let mut total_size = 0usize;
for_each_heap_block(1234, |block| {
    total_size += block.size;
    Ok(())
}).unwrap();
println!("Total heap size: {}", total_size);
```
