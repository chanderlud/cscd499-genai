1) Enumerate clipboard formats

**Spec:** Write a function that enumerates all available clipboard formats and returns a vector of tuples containing each format's numeric ID and its human-readable name (if available).

**Constraints:**
- Must use the Win32 clipboard enumeration APIs (`OpenClipboard`, `EnumClipboardFormats`, `GetClipboardFormatNameW`, `CloseClipboard`)
- Must handle UTF-16 to UTF-8 string conversion for format names
- Must properly open and close the clipboard around enumeration
- Should return an empty vector if no formats are available

**Signature:**
```rust
fn enumerate_clipboard_formats() -> Result<Vec<(u32, String)>, windows::core::Error>
```

**Example:**
```rust
let formats = enumerate_clipboard_formats()?;
for (id, name) in formats {
    println!("Format {}: {}", id, name);
}
```
