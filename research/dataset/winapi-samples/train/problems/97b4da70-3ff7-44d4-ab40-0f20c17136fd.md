# Create a System Tray Icon and Set Focus

**Spec:** Write a function that creates a system tray icon with a tooltip and sets focus to it. The function should create a window, load a system icon, add the notification icon with the specified tooltip and callback message, then set focus to the icon.

**Constraints:** Use the Windows API via the `windows` crate. The function must handle the complete lifecycle: window creation, icon setup, focus management, message loop, and cleanup. The notification icon must respond to user interaction via the specified callback message.

**Signature:**
```rust
fn create_tray_icon_with_focus(tooltip: &str, callback_msg: u32) -> Result<()>
```

**Example:**
```rust
create_tray_icon_with_focus("Click to focus", WM_USER + 1)?;
// Program runs until window is closed, then cleans up tray icon
```
