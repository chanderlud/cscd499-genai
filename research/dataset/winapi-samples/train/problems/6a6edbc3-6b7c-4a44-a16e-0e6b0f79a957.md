1) Check Window Visibility

**Spec:** Write a function that checks whether a given window is visible using the Win32 API.

**Constraints:** 
- Use the `windows` crate and its Win32 bindings
- The function must call `IsWindowVisible` from `WindowsAndMessaging`
- Handle the HWND parameter and BOOL return type correctly

**Signature:**
```rust
use windows::Win32::Foundation::HWND;
use windows::Win32::UI::WindowsAndMessaging::IsWindowVisible;

unsafe fn is_window_visible(hwnd: HWND) -> bool
```

**Example:**
```rust
use windows::Win32::UI::WindowsAndMessaging::GetDesktopWindow;

fn main() {
    let desktop_hwnd = unsafe { GetDesktopWindow() };
    let visible = unsafe { is_window_visible(desktop_hwnd) };
    println!("Desktop window is visible: {}", visible);
}
```
