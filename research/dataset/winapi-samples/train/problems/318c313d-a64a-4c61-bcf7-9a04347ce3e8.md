**Spec:** Write a function that sets a user account's expiration date using the `NetUserSetInfo` API with `USER_INFO_1017` level. The function should convert a Rust `SystemTime` to a Unix timestamp (seconds since 1970-01-01) and apply it to the specified user account.

**Constraints:**
- Must use `NetUserSetInfo` with level 1017 (`USER_INFO_1017`)
- Must convert `std::time::SystemTime` to a Unix timestamp (seconds since 1970-01-01) as a `u32`
- Must handle both setting a specific expiration date and setting "never expires" (using `0xFFFFFFFF` for never)
- Must properly manage memory for the `USER_INFO_1017` structure (by creating a local instance and passing a pointer)
- Must return appropriate Windows error codes on failure (via `windows::core::Result<()>`)

**Signature:**
```rust
fn set_user_expiration_date(
    server_name: PCWSTR,
    username: &str,
    expiration: Option<std::time::SystemTime>,
) -> windows::core::Result<()>
```

**Example:**
```rust
let server = PCWSTR::default();
let username = "testuser";
// Set expiration to 30 days from now
let expiration = std::time::SystemTime::now() + std::time::Duration::from_secs(30 * 24 * 60 * 60);
set_user_expiration_date(server, username, Some(expiration))?;

// Set account to never expire
set_user_expiration_date(server, username, None)?;
```
