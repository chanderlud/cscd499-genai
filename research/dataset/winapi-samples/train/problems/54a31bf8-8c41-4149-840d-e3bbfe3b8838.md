- **Spec:** Write a function that creates a layered overlay window which continuously magnifies a specified screen region. The magnified content should be rendered in the overlay window, and the source region should be configurable. The overlay must be click-through and topmost.
- **Constraints:** Must use `WS_EX_LAYERED` and `WS_EX_TRANSPARENT` for click-through behavior. Must capture screen content from the specified source rectangle and scale it to fit the overlay window. Must handle window resizing by updating the magnified region accordingly. Must update the magnified content continuously (at approximately 60 FPS) to reflect changes in the source region.
- **Signature:**
```rust
pub fn create_magnifier_overlay(
    source_rect: &RECT,
    overlay_width: i32,
    overlay_height: i32,
) -> windows::core::Result<HWND>
```
- **Example:**
```rust
let source = RECT { left: 100, top: 100, right: 400, bottom: 300 };
let hwnd = create_magnifier_overlay(&source, 300, 200)?;
// Overlay now shows magnified content from screen region (100,100)-(400,300) and updates continuously.
```
