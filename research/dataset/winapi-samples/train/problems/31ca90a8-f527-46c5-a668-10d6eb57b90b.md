N) Terminate a Process by PID

**Spec:** Write a function that terminates a process given its process ID (PID) using Win32 APIs.

**Constraints:**
- Use `OpenProcess` with `PROCESS_TERMINATE` access right
- Use `TerminateProcess` to end the process
- Use `CloseHandle` to clean up the process handle
- Return a `Result` to handle potential errors

**Signature:**
```rust
fn terminate_process_by_pid(pid: u32) -> Result<()>
```

**Example:**
```rust
fn main() -> Result<()> {
    terminate_process_by_pid(1234)?;
    Ok(())
}
```
