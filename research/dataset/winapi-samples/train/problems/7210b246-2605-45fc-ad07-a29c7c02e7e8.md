**Spec:**
- Write a function that intercepts calls to a target function by overwriting its first 5 bytes with a jump to a provided detour function.
- The function must return a trampoline that allows calling the original function.
- The trampoline must contain the original overwritten bytes followed by a jump to the remainder of the target function.
- The function must handle memory protection changes safely.

**Constraints:**
- Use `LoadLibraryA` and `GetProcAddress` to locate the target function.
- Use `VirtualProtect` to temporarily make memory writable.
- The detour function must have the same calling convention and signature as the target.
- The trampoline must be executable and properly aligned.

**Signature:**
```rust
pub unsafe fn intercept_function(
    module_name: &str,
    function_name: &str,
    detour: *const std::ffi::c_void,
) -> windows::core::Result<*const std::ffi::c_void>
```

**Example:**
```rust
unsafe extern "system" fn my_messagebox(
    hwnd: isize,
    text: *const u16,
    caption: *const u16,
    msgtype: u32,
) -> i32 {
    // Custom implementation
    1 // IDOK
}

let trampoline = intercept_function("user32.dll", "MessageBoxW", my_messagebox as _)?;
// Now MessageBoxW calls are intercepted by my_messagebox
// Original function can be called via trampoline
```
