1) Set Process Priority to Below Normal

**Spec:** Write a function that sets the current process's priority class to below normal using the Windows API.

**Constraints:**
- Use the `windows` crate for Win32 bindings.
- The function must return a `Result<()>` to handle potential API errors.
- Use `GetCurrentProcess` to obtain a handle to the current process.
- Use `SetPriorityClass` with `BELOW_NORMAL_PRIORITY_CLASS`.

**Signature:**
```rust
fn set_current_process_priority_below_normal() -> Result<()>
```

**Example:**
```rust
use windows::core::Result;

fn main() -> Result<()> {
    set_current_process_priority_below_normal()?;
    println!("Process priority set to below normal.");
    Ok(())
}
```
