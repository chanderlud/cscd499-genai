**Spec:** Write a function that injects and executes arbitrary shellcode in a remote process by allocating memory, writing the shellcode, creating a remote thread to run it, waiting for completion, and cleaning up resources. Must return an error for empty shellcode input.
**Constraints:** Must use `VirtualAllocEx` with `MEM_COMMIT | MEM_RESERVE` and `PAGE_EXECUTE_READWRITE`, `WriteProcessMemory`, `CreateRemoteThreadEx`, `WaitForSingleObject` with `INFINITE` timeout, and clean up with `VirtualFreeEx` and `CloseHandle`.
**Signature:**
```rust
pub fn inject_shellcode(handle: HANDLE, shellcode: &[u8]) -> windows::core::Result<()>
```
**Example:**
```rust
let process = open_process("target.exe")?;
let shellcode = [0x90, 0x90, 0xC3]; // NOP NOP RET
inject_shellcode(process.handle, &shellcode)?;
```
