1) System Tray Icon Manager

**Spec:** Implement a `NotifyIcon` struct that wraps the Win32 `NOTIFYICONDATAW` structure and provides a builder pattern for setting properties and methods for adding, modifying, and deleting a system tray icon.

**Constraints:**
- Must use the `windows` crate and the `Shell_NotifyIconW` function
- The `NOTIFYICONDATAW` structure must be initialized with the correct `cbSize`
- Builder methods must set appropriate flags in the `uFlags` field (e.g., `NIF_MESSAGE`, `NIF_TIP`, `NIF_ICON`)
- The `tip` method must handle UTF-16 encoding and truncation if the string is too long for the `szTip` field
- The `notify` method must return a `Result` and handle errors from `Shell_NotifyIconW`

**Signature:**
```rust
pub struct NotifyIcon {
    data: NOTIFYICONDATAW,
}

impl NotifyIcon {
    pub fn new() -> Self;
    pub fn flag(self, flag: NOTIFY_ICON_DATA_FLAGS) -> Self;
    pub fn window_handle(self, handle: HWND) -> Self;
    pub fn tip(self, s: impl Into<String>) -> Self;
    pub fn icon(self, icon: HICON) -> Self;
    pub fn callback_message(self, callback_msg: u32) -> Self;
    pub fn notify(&self, message: NOTIFY_ICON_MESSAGE) -> Result<()>;
    pub fn notify_add(&self) -> Result<()>;
    pub fn notify_modify(&self) -> Result<()>;
    pub fn notify_delete(&self) -> Result<()>;
}
```

**Example:**
```rust
let hwnd: HWND = ...; // obtain a window handle
let icon: HICON = ...; // load an icon

let tray_icon = NotifyIcon::new()
    .window_handle(hwnd)
    .icon(icon)
    .tip("My Tooltip")
    .callback_message(WM_USER + 1);

tray_icon.notify_add()?;

// Later, update the tooltip
let updated_tray_icon = tray_icon.tip("Updated Tooltip");
updated_tray_icon.notify_modify()?;

// Finally, remove the icon
tray_icon.notify_delete()?;
```
