1) Create a ListBox with Enter Key Handling

**Spec:** Create a Win32 window application that displays a list box with a fixed set of strings. When the user selects an item and presses the Enter key, the main window's title must change to the selected item's text.

**Constraints:**
- Use the `windows` crate for Win32 API bindings
- The main window must be created with a custom window procedure
- The list box must be a child window with the `LBS_NOTIFY` style and must be subclassed to handle the `WM_KEYUP` message for the Enter key
- The list box must be populated with at least 5 items, each associated with an index via `LB_SETITEMDATA`
- When the Enter key is pressed, the selected item's text must be retrieved and set as the main window's title

**Signature:**
```rust
pub fn main() -> Result<(), Box<dyn std::error::Error>>
```

**Example:**
```rust
// The program creates a window with a list box containing items like "Document 1", "Document 2", etc.
// Selecting an item and pressing Enter changes the window title to that item's text.
```
