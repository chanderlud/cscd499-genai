1) Get Primary Monitor Name

**Spec:** Write a function that retrieves the device name of the primary monitor as a UTF-8 string.

**Constraints:**
- Use `MonitorFromPoint` with coordinates (0,0) and `MONITOR_DEFAULTTOPRIMARY` to obtain the primary monitor handle
- Use `GetMonitorInfoW` to populate a `MONITORINFOEXW` structure
- The `cbSize` field must be set to `std::mem::size_of::<MONITORINFO>()`
- Return a `windows::core::Result<String>` containing the monitor name
- Handle API failures by converting the last error code to an HRESULT

**Signature:**
```rust
fn get_primary_monitor_name() -> windows::core::Result<String>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    let name = get_primary_monitor_name()?;
    println!("Primary monitor: {}", name);
    Ok(())
}
```
