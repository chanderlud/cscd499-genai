**Spec:** Implement a function that reads a file in fixed-size chunks and computes a simple checksum (sum of all bytes modulo 256) without loading the entire file into memory.

**Constraints:**
- Must use Windows API for file operations (CreateFileW, ReadFile, CloseHandle)
- Read file in chunks of exactly 4096 bytes (except possibly the last chunk)
- Process each chunk immediately and discard it before reading the next
- Return the checksum as a u8 value
- Handle files of any size, including empty files (checksum 0)

**Signature:**
```rust
pub fn file_checksum(path: &str) -> windows::core::Result<u8>
```

**Example:**
```rust
let checksum = file_checksum("example.txt")?;
println!("Checksum: {}", checksum);
```
