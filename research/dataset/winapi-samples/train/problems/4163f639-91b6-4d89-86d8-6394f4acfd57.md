1) Foreground Window Selector

**Spec:** Create a Win32 window containing a list box that displays window titles. When the user selects an item and presses Enter, bring the corresponding window to the foreground. Pressing Escape quits the application.

**Constraints:**
- Use the `windows` crate for Win32 bindings
- Main window must be a popup with title "Foreground Example"
- List box must be a child window with `LBS_NOTIFY` style and vertical scroll bar
- Must subclass the list box to handle `WM_KEYUP` for Enter/Escape keys
- Must use `SetForegroundWindow` to bring selected window to foreground
- Must store window handle data with list items using `LB_SETITEMDATA`
- Must use standard Win32 message loop with `GetMessageW`, `TranslateMessage`, `DispatchMessageW`

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// Creates window with list box containing "Notepad" and "Calculator"
// Selecting "Notepad" and pressing Enter brings that window to foreground
// Pressing Escape exits the application
```
