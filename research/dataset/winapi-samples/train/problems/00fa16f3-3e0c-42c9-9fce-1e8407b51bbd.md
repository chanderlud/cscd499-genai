1) Get and Set Cursor Clip Rectangle

**Spec:** Implement functions to retrieve the current cursor clipping rectangle and to set or clear the cursor clipping region using Win32 APIs.

**Constraints:** 
- Use the `windows` crate for Win32 bindings.
- Functions must return `Result` to handle errors from Win32 calls.
- `get_cursor_clip` should return the current `RECT` via `GetClipCursor`.
- `set_cursor_clip` should take an `Option<RECT>` to set a new clip with `ClipCursor` or unclip if `None`.
- Ensure proper use of unsafe blocks for Win32 API calls.

**Signature:**
```rust
fn get_cursor_clip() -> Result<RECT>;
fn set_cursor_clip(rect: Option<RECT>) -> Result<()>;
```

**Example:**
```rust
use windows::core::Result;
use windows::Win32::Foundation::RECT;

fn main() -> Result<()> {
    let original = get_cursor_clip()?;
    let new_clip = RECT { left: 200, top: 200, right: 300, bottom: 300 };
    set_cursor_clip(Some(new_clip))?;
    set_cursor_clip(Some(original))?;
    Ok(())
}
```
