1) Create a Rounded Corner Popup Window

**Spec:** Write a function that creates a borderless popup window with rounded corners, positions it centered horizontally and at 25% of the screen height from the top, and runs the standard Win32 message loop until the window is closed.

**Constraints:**
- Use the `windows` crate (version 0.58.0 or compatible) and the Win32 API
- The window must have rounded corners (using `DwmSetWindowAttribute` with `DWMWA_WINDOW_CORNER_PREFERENCE` and `DWMWCP_ROUND`)
- The window must be centered horizontally and positioned at 25% of the screen height from the top
- The window must be a popup (no title bar or borders) and visible
- The window size is fixed at 600x400
- The window procedure must handle `WM_DESTROY` by calling `PostQuitMessage(0)`

**Signature:**
```rust
use windows::core::Result;

pub fn create_rounded_window() -> Result<()>;
```

**Example:**
```rust
use windows::core::Result;

fn main() -> Result<()> {
    create_rounded_window()
}
```
