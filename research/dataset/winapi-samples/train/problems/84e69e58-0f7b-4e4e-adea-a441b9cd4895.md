**Spec:** Write a function that sets the process DPI aware and returns the bounding rectangle of the primary monitor by using the virtual screen rectangle and `MonitorFromRect` with `MONITOR_DEFAULTTOPRIMARY`.
**Constraints:**
- Must use Win32 APIs for DPI awareness and monitor information
- Must handle the case when the virtual screen rectangle is empty (unlikely) by returning the primary monitor
- Must return the monitor bounds as a RECT structure
**Signature:**
```rust
fn get_desktop_monitor_bounds() -> windows::core::Result<windows::Win32::Foundation::RECT>
```
**Example:**
```rust
fn main() -> windows::core::Result<()> {
    let rect = get_desktop_monitor_bounds()?;
    println!("Monitor bounds: ({}, {}) to ({}, {})", rect.left, rect.top, rect.right, rect.bottom);
    Ok(())
}
```
