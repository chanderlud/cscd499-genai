**Spec:**
- Create a topmost, click-through overlay window that can dynamically toggle between two transparency modes: color-key transparency and uniform alpha transparency.
- Implement a function to switch between these modes at runtime without recreating the window.
- The overlay must remain non-interactive (mouse pass-through) in both modes.
- Set initial transparency mode to color-key using magenta (0x00FF00FF).
- Extend glass effect into client area for proper transparency.

**Constraints:**
- Use `WS_EX_LAYERED | WS_EX_TRANSPARENT | WS_EX_TOPMOST | WS_EX_NOACTIVATE | WS_EX_TOOLWINDOW` extended styles.
- Color-key mode should use magenta (0x00FF00FF) as the transparent color.
- Alpha mode should allow setting any alpha value (0-255).
- Toggling must not cause visual flickering or window reactivation.
- Force repaint after toggling transparency mode.

**Signature:**
```rust
pub fn create_transparent_overlay(class_name: &str, width: i32, height: i32) -> windows::core::Result<HWND>
pub fn toggle_transparency_mode(hwnd: HWND, use_alpha: bool, alpha_value: u8) -> windows::core::Result<()>
```

**Example:**
```rust
let hwnd = create_transparent_overlay("MyOverlay", 800, 600).unwrap();
// Switch to 50% alpha transparency
toggle_transparency_mode(hwnd, true, 128).unwrap();
// Switch back to color-key transparency
toggle_transparency_mode(hwnd, false, 0).unwrap();
```
