**Spec:**
- Write a function that parses a PE binary's base relocation table from raw bytes.
- Return a vector of relocation blocks, each containing a page RVA and a list of relocation entries.
- Each relocation entry should include its type (high 4 bits) and offset (low 12 bits).
- Handle both 32-bit and 64-bit PE files.
- Define the following structs:
  ```rust
  #[derive(Debug, Clone)]
  pub struct RelocationEntry {
      pub relocation_type: u8,
      pub offset: u16,
  }

  #[derive(Debug, Clone)]
  pub struct RelocationBlock {
      pub page_rva: u32,
      pub entries: Vec<RelocationEntry>,
  }
  ```

**Constraints:**
- Must validate DOS and NT signatures before parsing.
- Must locate the relocation table via the data directory at index 5 (IMAGE_DIRECTORY_ENTRY_BASERELOC).
- Must convert RVAs to file offsets using section headers.
- Return `Err` for invalid PE files or malformed relocation data.
