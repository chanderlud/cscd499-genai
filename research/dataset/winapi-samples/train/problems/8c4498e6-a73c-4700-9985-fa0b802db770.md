1) Enumerate Display Monitors

**Spec:** Write a function that enumerates all display monitors connected to the system and prints detailed information for each monitor, including its handle, device name, dimensions, work area, and flags.

**Constraints:**
- Use the `windows` crate and Win32 API functions `EnumDisplayMonitors` and `GetMonitorInfoW`
- Implement a callback function matching the `MONITORENUMPROC` signature
- Handle errors gracefully and continue enumeration even if individual monitor queries fail
- Return a `Result<()>` to propagate any enumeration errors

**Signature:**
```rust
pub fn enumerate_monitors() -> Result<()>
```

**Example:**
```rust
use windows::core::Result;

fn main() -> Result<()> {
    enumerate_monitors()?;
    Ok(())
}
```
