**Spec:** Write a function that creates a new local group with a given name and description, then adds a specified user to that group. If the group already exists, the function should proceed to add the user without failing.

**Constraints:**
- Use `NetLocalGroupAdd` with `LOCALGROUP_INFO_1` to create the group with both name and comment.
- Use `NetLocalGroupAddMembers` with `LOCALGROUP_MEMBERS_INFO_3` to add the user.
- Handle `NERR_GroupExists` (2223) as a non-fatal error when creating the group.
- All string parameters must be converted to null-terminated UTF-16 internally.
- Return `windows::core::Result<()>` with appropriate error propagation.

**Signature:**
```rust
fn create_local_group_and_add_user(
    server: Option<&str>,
    group_name: &str,
    group_comment: &str,
    user_name: &str,
) -> windows::core::Result<()>
```

**Example:**
```rust
// Create a group named "Developers" with description "Development team" and add user "alice"
create_local_group_and_add_user(None, "Developers", "Development team", "alice")?;
```
