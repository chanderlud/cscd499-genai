**Spec:**
- Write a function that generates an RSA key pair using the BCrypt API
- The function should open the RSA algorithm provider, generate a key pair of specified bit length, and export both public and private key blobs
- Return the public key blob (BCRYPT_RSAPUBLIC_BLOB format) and private key blob (BCRYPT_RSAFULLPRIVATE_BLOB format) as byte vectors

**Constraints:**
- Use BCryptOpenAlgorithmProvider with BCRYPT_RSA_ALGORITHM
- Use BCryptGenerateKeyPair and BCryptFinalizeKeyPair
- Use BCryptExportKey to export both public and private key blobs
- Properly clean up all BCrypt handles (algorithm provider and key handle)

**Signature:**
```rust
fn generate_rsa_keypair(bit_length: u32) -> windows::core::Result<(Vec<u8>, Vec<u8>)>
```

**Example:**
```rust
// Generate a 2048-bit RSA key pair
let (public_blob, private_blob) = generate_rsa_keypair(2048)?;

println!("Public key blob length: {} bytes", public_blob.len());
println!("Private key blob length: {} bytes", private_blob.len());
```
