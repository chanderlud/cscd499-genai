**Spec:** Write a function that enumerates all local groups on the system and returns a list of group names along with the count of members in each group.

**Constraints:**
- Use `NetLocalGroupEnum` to retrieve group names, handling the case where the buffer is too small by retrying until all data is retrieved.
- Use `NetLocalGroupGetMembers` to count members for each group, handling the case where the buffer is too small by retrying until all data is retrieved.
- Return results as a vector of tuples `(group_name: String, member_count: u32)`.
- Handle API errors by returning `windows::core::Result`.
- Use `NetApiBufferFree` to free buffers allocated by the API.

**Signature:**
```rust
fn enumerate_local_group_memberships() -> windows::core::Result<Vec<(String, u32)>>
```

**Example:**
```rust
let groups = enumerate_local_group_memberships()?;
for (name, count) in groups {
    println!("{}: {} members", name, count);
}
```
