1) Read game window handle from fixed memory address

**Spec:** Write a function that reads the window handle of the game Grand Theft Auto: San Andreas from a fixed memory address (0xC17054) which is known to contain a pointer to the window handle.

**Constraints:** The function must be unsafe because it dereferences a raw pointer. It must return an `HWND` from the `windows::Win32::Foundation` module.

**Signature:**
```rust
fn get_window_handle() -> HWND
```

**Example:**
```rust
use windows::Win32::Foundation::HWND;

fn main() {
    let hwnd = get_window_handle();
    println!("Window handle: {:?}", hwnd);
}
```
