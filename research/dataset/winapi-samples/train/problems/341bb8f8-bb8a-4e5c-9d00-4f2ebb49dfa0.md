1) Convert Win32 TCP Row Structures to Rust Types

**Spec:** Implement `TryFrom` conversions for Win32 TCP row structures (`MIB_TCPROW_LH` and `MIB_TCP6ROW`) into a custom `TcpRow` struct that represents a TCP connection with state, local address, and remote address.

**Constraints:**
- Create a `TcpRow` struct with fields: `state` (as a `TcpState` enum), `local` (as `SocketAddr`), and `remote` (as `SocketAddr`)
- Define a `TcpState` enum with variants for all 12 standard TCP states (Closed through DeleteTcb)
- Implement `TryFrom<MIB_TCPROW_LH>` for IPv4 rows and `TryFrom<MIB_TCP6ROW>` for IPv6 rows
- Convert network byte order addresses/ports to host byte order
- Return `E_NOTIMPL` error for invalid TCP state values
- Use `windows::core::Error` for error handling

**Signature:**
```rust
impl TryFrom<MIB_TCPROW_LH> for TcpRow {
    type Error = windows::core::Error;
    fn try_from(value: MIB_TCPROW_LH) -> std::result::Result<Self, Self::Error>;
}

impl TryFrom<MIB_TCP6ROW> for TcpRow {
    type Error = windows::core::Error;
    fn try_from(value: MIB_TCP6ROW) -> std::result::Result<Self, Self::Error>;
}
```

**Example:**
```rust
let ipv4_row = MIB_TCPROW_LH {
    Anonymous: MIB_TCPROW_LH_0 { dwState: 5 }, // ESTABLISHED
    dwLocalAddr: 0x0100007f,                   // 127.0.0.1
    dwLocalPort: 0x5000,                       // Port 80
    dwRemoteAddr: 0x00000000,
    dwRemotePort: 0x0000,
};

let tcp_row = TcpRow::try_from(ipv4_row)?;
// tcp_row.state == TcpState::Established
// tcp_row.local == SocketAddr::new(IpAddr::from([127, 0, 0, 1]), 80)
```
