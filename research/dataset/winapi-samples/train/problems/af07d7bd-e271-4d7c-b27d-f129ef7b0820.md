**Spec:** Write a function that checks whether the Input Method Editor (IME) is currently open for the foreground window. If no foreground window exists, return an error. If the foreground window has no IME context, return `false`. Otherwise, return `true` if the IME is open, `false` if closed.

**Constraints:**
- Must use `GetForegroundWindow` to obtain the active window handle.
- Must use `ImmGetContext` and `ImmGetOpenStatus` from the Windows IME API.
- Must properly release the IME context with `ImmReleaseContext`.
- Return a `windows::core::Result<bool>` to handle potential API failures.

**Signature:**
```rust
fn is_ime_open_for_foreground() -> windows::core::Result<bool>
```

**Example:**
```rust
match is_ime_open_for_foreground() {
    Ok(is_open) => println!("IME is {}", if is_open { "open" } else { "closed" }),
    Err(e) => eprintln!("Failed to get IME status: {}", e),
}
```
