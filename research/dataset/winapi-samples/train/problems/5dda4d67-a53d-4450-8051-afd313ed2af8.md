1) Command-Line Spell Checker

**Spec:** Write a program that takes a string as a command-line argument and uses the Windows Spell Checking API to check it for spelling errors. For each error found, print the corrective action and relevant details (deletion, replacement text, or suggestions).

**Constraints:**
- Must use the Windows Spell Checking API (`ISpellCheckerFactory`, `ISpellChecker`)
- Must initialize COM runtime for the thread
- Must use the "en-US" locale
- Must handle the three corrective actions: delete, replace, and get suggestions
- Must properly free allocated memory using `CoTaskMemFree`

**Signature:** The program should be a binary with a `main` function that takes one command-line argument.

**Example:**
Running the program with argument "Helo wrld" might output:
```
Checking the text: 'Helo wrld'
CORRECTIVE_ACTION_GET_SUGGESTIONS
Maybe replace: Helo with Hello
CORRECTIVE_ACTION_GET_SUGGESTIONS
Maybe replace: wrld with world
```
