**Spec:**
- Create a `ProgressToast` struct that manages a toast notification with a progress bar
- The toast must display a title, optional body text, and a progress bar showing completion percentage
- Provide methods to update the progress value (0.0 to 1.0) and status text while the toast is active
- The toast should remain visible until explicitly dismissed

**Constraints:**
- Use the Windows.UI.Notifications API with ToastGeneric template
- Progress updates must modify the existing toast, not create new ones
- Must handle Windows 8.1+ (progress bar requires Windows 10+)
- All string content must be properly XML-escaped

**Signature:**
```rust
pub struct ProgressToast {
    // Implementation details
}

impl ProgressToast {
    pub fn new(app_id: &str, title: &str, initial_status: &str) -> windows::core::Result<Self>;
    
    pub fn update_progress(&self, value: f64, status: &str) -> windows::core::Result<()>;
    
    pub fn set_body(&self, body: &str) -> windows::core::Result<()>;
    
    pub fn dismiss(&self) -> windows::core::Result<()>;
}
```

**Example:**
```rust
let toast = ProgressToast::new("MyApp", "Downloading", "Starting...")?;
toast.set_body("File: setup.exe")?;

for i in 0..=10 {
    let progress = i as f64 / 10.0;
    toast.update_progress(progress, &format!("{}%", i * 10))?;
    std::thread::sleep(std::time::Duration::from_secs(1));
}

toast.dismiss()?;
```
