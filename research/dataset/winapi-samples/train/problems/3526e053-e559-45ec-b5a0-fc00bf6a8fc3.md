N) Run a Win32 message loop
**Spec:** Write a function that runs a standard Win32 message loop until a `WM_QUIT` message is received, translating and dispatching each message, and returns `Ok(())` on success or an error if `GetMessageW` fails.
**Constraints:**
- Use the `windows` crate.
- The function must return `Result<()>`.
- Handle the error case when `GetMessageW` returns a negative value by returning an error with the last error code.
- Use `unsafe` blocks for the Win32 API calls.
**Signature:**
```rust
fn run_message_loop() -> Result<()>
```
**Example:**
```rust
fn main() -> Result<()> {
    run_message_loop()
}
```
