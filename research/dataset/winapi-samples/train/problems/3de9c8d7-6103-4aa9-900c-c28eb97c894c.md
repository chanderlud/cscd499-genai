## System Uptime Query

**Spec:** Write a function that retrieves the system uptime in milliseconds using the Windows API and returns both the raw milliseconds and a formatted string in the format "HH:MM:SS".

**Constraints:**
- Must use the `GetTickCount64` function from the Windows API
- The function must be safe to call (encapsulate any unsafe operations)
- The formatted string must have two digits for hours, minutes, and seconds, separated by colons

**Signature:**
```rust
fn get_system_uptime() -> (u64, String)
```

**Example:**
```rust
let (uptime_ms, formatted) = get_system_uptime();
println!("System Uptime: {} milliseconds", uptime_ms);
println!("Formatted: {}", formatted);
```
