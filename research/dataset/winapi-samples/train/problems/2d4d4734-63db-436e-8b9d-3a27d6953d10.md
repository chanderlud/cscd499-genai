1) Thread Pool Counter

**Spec:** Write a function that uses the Windows Thread Pool API to run a callback function `n` times. The callback should increment a shared counter. The function should wait for all callbacks to complete and return the final counter value.

**Constraints:**
- Use the Windows Thread Pool API (`CreateThreadpoolWork`, `SubmitThreadpoolWork`, `WaitForThreadpoolWorkCallbacks`).
- The callback must be an `unsafe extern "system"` function.
- Use a `RwLock` to protect the counter.

**Signature:**
```rust
fn run_thread_pool_work(n: i32) -> i32
```

**Example:**
```rust
let counter = run_thread_pool_work(10);
println!("counter: {}", counter); // prints 10
```
