1) Virtual Screen Rectangle

**Spec:** Write a function `get_desktop_rect` that returns a `RECT` structure representing the bounding rectangle of the virtual screen (the area that spans all monitors).

**Constraints:**
- Use the `windows` crate and the `GetSystemMetrics` function with the appropriate constants for the virtual screen.
- The function must be safe to call (wrap the unsafe code appropriately).

**Signature:**
```rust
fn get_desktop_rect() -> RECT
```

**Example:**
```rust
fn main() {
    let rect = get_desktop_rect();
    println!("Virtual screen rectangle:");
    println!("  Left: {}", rect.left);
    println!("  Top: {}", rect.top);
    println!("  Right: {}", rect.right);
    println!("  Bottom: {}", rect.bottom);
    println!("  Width: {}", rect.right - rect.left);
    println!("  Height: {}", rect.bottom - rect.top);
}
```
