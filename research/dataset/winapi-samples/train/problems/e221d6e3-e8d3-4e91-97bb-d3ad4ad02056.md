**Spec:** Write a function that computes the SHA-256 hash of input data using only FIPS-approved algorithms. The function must:
1. Use `BCryptGetFipsAlgorithmMode` to verify the system is in FIPS mode. Return an error if not.
2. Open the SHA-256 algorithm provider with `BCryptOpenAlgorithmProvider` using `BCRYPT_SHA256_ALGORITHM` and `BCRYPT_FIPS_ALGORITHM_FLAG`.
3. Create a hash object with `BCryptCreateHash`.
4. Hash the provided data with `BCryptHashData`.
5. Retrieve the 32-byte digest with `BCryptFinishHash`.
6. Clean up the hash handle with `BCryptDestroyHash` and the algorithm handle with `BCryptCloseAlgorithmProvider`.

**Constraints:**
- Must use `BCryptGetFipsAlgorithmMode` to check FIPS mode and return an error if not enabled.
- Must use `BCRYPT_SHA256_ALGORITHM` with `BCRYPT_FIPS_ALGORITHM_FLAG` (value 0x00000001) in `BCryptOpenAlgorithmProvider`.
- Must use `BCryptCreateHash`, `BCryptHashData`, and `BCryptFinishHash` to compute the hash.
- Must properly close algorithm and hash handles using `BCryptCloseAlgorithmProvider` and `BCryptDestroyHash` in all cases (success and error).
- Input data can be any byte slice.

**Signature:**
```rust
fn compute_fips_sha256(data: &[u8]) -> windows::core::Result<[u8; 32]>
```

**Example:**
```rust
let data = b"Hello, FIPS!";
let hash = compute_fips_sha256(data)?;
println!("SHA-256: {:02x?}", hash);
```
