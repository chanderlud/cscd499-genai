1) Get Window Styles

**Spec:** Write a function that retrieves the window style and extended style flags for a given window handle.

**Constraints:**
- Use the `windows` crate and the Win32 API function `GetWindowLongW`.
- The function must return a `Result` containing a tuple of `WINDOW_STYLE` and `WINDOW_EX_STYLE`.
- Handle the unsafe FFI call appropriately within the function.

**Signature:**
```rust
fn get_window_styles(hwnd: HWND) -> Result<(WINDOW_STYLE, WINDOW_EX_STYLE)>
```

**Example:**
```rust
let desktop = unsafe { GetDesktopWindow() };
let (style, style_ex) = get_window_styles(desktop)?;
println!("Window style: 0x{:08X}", style.0);
println!("Extended style: 0x{:08X}", style_ex.0);
```
