TITLE: Split File by Size

**Spec:** Implement a function that splits a source file into multiple smaller files, each of a specified maximum size (except the last part). The function should read the source file sequentially and write each chunk to a new file named with a sequential index suffix (e.g., `output_part0`, `output_part1`).

**Constraints:**
- Use the Windows API for all file operations (CreateFileW, ReadFile, WriteFile, CloseHandle).
- Process the file in chunks to avoid loading the entire source into memory.
- Handle errors appropriately and ensure all file handles are closed.
- The part size must be a multiple of 512 bytes for alignment efficiency. If the provided part size is not a multiple of 512, it will be rounded down to the nearest multiple of 512. If the rounded part size is zero, an error is returned.
- The function must return the number of parts created.

**Signature:**
```rust
pub fn split_file_by_size(source_path: &str, part_size: u32) -> windows::core::Result<u32>
```

**Example:**
```rust
let parts_created = split_file_by_size("large_data.bin", 1024 * 1024)?; // Split into 1MB parts
println!("Created {} parts", parts_created);
```
