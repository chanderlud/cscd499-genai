**Spec:**
- Accept a sentence mode value (e.g., `IME_SMODE_AUTOMATIC`, `IME_SMODE_PHRASEPREDICT`) as an `isize`.
- Retrieve the foreground window and its default IME window.
- Send the `WM_IME_CONTROL` message with `IMC_SETSENTENCEMODE` using `SendMessageW` to set the new sentence mode.
- Return `Ok(())` on success or an appropriate Windows error on failure (including if handles are invalid or message processing fails).

**Constraints:**
- Use `GetForegroundWindow`, `ImmGetDefaultIMEWnd`, and `SendMessageW` from the Win32 API.
- The function must handle unsafe operations internally and expose a safe public API.
- The function should not block or require user interaction.
