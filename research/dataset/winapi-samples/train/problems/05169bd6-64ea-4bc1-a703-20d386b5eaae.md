**Spec:** Write a function that imports a DSA private key into CNG given the algorithm handle and the key parameters (prime modulus, prime divisor, generator, and private key). The function should return an owned key handle that automatically destroys the key when dropped.

**Constraints:**
- The prime divisor must be exactly 20 bytes.
- The private key must be exactly 20 bytes.
- The generator must have the same length as the prime modulus.
- The function must use the provided algorithm handle for DSA.

**Signature:**
```rust
fn import_dsa_private_key(
    alg_handle: BCRYPT_ALG_HANDLE,
    prime_modulus: &[u8],
    prime_divisor: &[u8],
    generator: &[u8],
    private_key: &[u8],
) -> windows::core::Result<Owned>
```

**Example:**
```rust
// Example usage with a DSA algorithm handle and key parameters.
let key_handle = import_dsa_private_key(
    alg_handle,
    prime_modulus,
    prime_divisor,
    generator,
    private_key,
)?;
```
