**Spec:** Create all missing parent directories then the final directory, using only stack-allocated buffers and without any heap allocations.
**Constraints:**
- Use `CreateDirectoryW` for directory creation.
- Tolerate "already exists" errors.
- No heap allocations allowed (no `Vec`, `String`, `PathBuf`, or dynamic memory).
- Must work for paths up to `MAX_PATH` (260) characters; return an error for longer paths.
- Use a fixed-size stack buffer for wide string conversion.

**Signature:**
```rust
pub fn create_dir_all_no_alloc(path: &Path) -> Result<()>;
```

**Example:**
```rust
create_dir_all_no_alloc(Path::new(r"C:\tmp\a\b\c"))?;
```
