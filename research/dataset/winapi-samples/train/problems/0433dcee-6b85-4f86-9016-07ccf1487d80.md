1) Minimal UWP Application

**Spec:** Create a minimal UWP (Universal Windows Platform) application that runs a window and processes events until the user quits.

**Constraints:**
- Must use the `windows` crate to access UWP and Win32 APIs
- Must implement `IFrameworkViewSource` and `IFrameworkView` traits
- Must check if running in a package context and display an error if not
- Must activate the window and process events until quit

**Signature:**
```rust
#![windows_subsystem = "windows"]

use windows::{
    core::*,
    ApplicationModel::{Core::*, Package},
    Win32::{
        System::Com::*,
        UI::WindowsAndMessaging::{MessageBoxW, MB_ICONSTOP, MB_OK},
    },
    UI::Core::*,
};

#[implement(IFrameworkViewSource)]
struct CoreApp();

impl IFrameworkViewSource_Impl for CoreApp_Impl {
    fn CreateView(&self) -> Result<IFrameworkView>;
}

#[implement(IFrameworkView)]
struct CoreAppView();

impl IFrameworkView_Impl for CoreAppView_Impl {
    fn Initialize(&self, _: Ref<CoreApplicationView>) -> Result<()>;
    fn Load(&self, _: &HSTRING) -> Result<()>;
    fn Uninitialize(&self) -> Result<()>;
    fn Run(&self) -> Result<()>;
    fn SetWindow(&self, _: Ref<CoreWindow>) -> Result<()>;
}

fn main() -> Result<()>;
```

**Example:**
```rust
// When run, the application should:
// 1. Initialize COM
// 2. Check for package context
// 3. Create and run the UWP application
// 4. Display a window that processes events until quit
```
