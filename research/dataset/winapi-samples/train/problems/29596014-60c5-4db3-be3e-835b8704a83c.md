1) Get Monitor Device Name from HMONITOR

**Spec:** Implement a function that retrieves the device name string for a given monitor handle.

**Constraints:**
- Must use the Win32 `GetMonitorInfoW` API with `MONITORINFOEXW` structure
- Must handle UTF-16 to Rust String conversion correctly
- Must return a `Result` to handle potential Win32 API failures
- Must work with any valid `HMONITOR` handle

**Signature:**
```rust
fn get_monitor_device_name(hmonitor: HMONITOR) -> Result<String>
```

**Example:**
```rust
let hmonitor = unsafe { MonitorFromWindow(HWND::default(), MONITOR_DEFAULTTOPRIMARY) };
let device_name = get_monitor_device_name(hmonitor)?;
println!("Monitor device name: {}", device_name);
```
