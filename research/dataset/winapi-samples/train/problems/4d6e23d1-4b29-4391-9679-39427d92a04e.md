1) Enable Mouse Input and Wait for Console Input

**Spec:** Write a function that enables mouse input for the console's standard input handle, then waits for input with a specified timeout, returning whether input became available.

**Constraints:**
- Must use `GetStdHandle` to obtain the standard input handle
- Must use `GetConsoleMode` and `SetConsoleMode` to modify console input mode
- Must preserve existing console mode flags when enabling `ENABLE_MOUSE_INPUT`
- Must use `WaitForSingleObject` to wait for input availability
- Must handle `INVALID_HANDLE_VALUE` and API errors appropriately

**Signature:**
```rust
fn wait_for_input_with_mouse(timeout_ms: u32) -> windows::core::Result<bool>
```

**Example:**
```rust
// Enable mouse input and wait up to 5 seconds for any console input
let input_available = wait_for_input_with_mouse(5000)?;
if input_available {
    println!("Input received within timeout period");
} else {
    println!("No input within timeout period");
}
```
