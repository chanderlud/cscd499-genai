**Spec:** Implement a method for the `Toast` struct that adds interactive buttons to the toast notification. Each button should have a label and an activation argument that gets passed to the application when the button is clicked. The method should XML-escape both the label and activation argument and store up to 5 buttons.

**Constraints:**
- Must support adding multiple buttons (up to 5)
- Button labels must be XML-escaped
- Activation arguments must be XML-escaped
- Must maintain backward compatibility with existing toast functionality
- Buttons should appear below the toast content
