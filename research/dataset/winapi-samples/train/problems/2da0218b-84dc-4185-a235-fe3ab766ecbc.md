1) Get Screen Resolution

**Spec:** Write a function that returns the width and height of the primary screen in pixels.

**Constraints:**
- Use the Win32 API function `GetSystemMetrics` from the `windows` crate
- Use the constants `SM_CXSCREEN` and `SM_CYSCREEN` for width and height respectively
- The function must be safe to call (handle unsafe internally)

**Signature:**
```rust
fn get_screen_resolution() -> (i32, i32)
```

**Example:**
```rust
let (width, height) = get_screen_resolution();
println!("Screen resolution: {}x{}", width, height);
```
