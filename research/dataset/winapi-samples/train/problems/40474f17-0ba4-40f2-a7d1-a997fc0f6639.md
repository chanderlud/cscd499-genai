**Spec:**
- Implement a function that parses the debug directory of a PE binary and returns a vector of debug entry metadata.
- Each entry is represented by a struct `DebugEntry` with fields:
  - `debug_type: u32` (the debug type)
  - `size: u32` (the size of the debug data)
  - `address: u32` (the RVA of the debug data)
- The function must handle both 32-bit and 64-bit PE files.
- Return an empty vector if no debug directory exists.

**Constraints:**
- Must use the Windows API structures for PE parsing (IMAGE_DEBUG_DIRECTORY, etc.).
- Must correctly locate the debug directory via the data directory at index 6.
- Must convert RVAs to file offsets using the section headers.
- Must not panic on invalid PE files; return an error instead.

**Signature:**
```rust
pub fn parse_debug_entries(data: &[u8]) -> windows::core::Result<Vec<DebugEntry>>
```

**Example:**
```rust
let data = std::fs::read("example.exe")?;
let entries = parse_debug_entries(&data)?;
for entry in entries {
    println!("Type: {}, Size: {}, RVA: {:#x}", entry.debug_type, entry.size, entry.address);
}
