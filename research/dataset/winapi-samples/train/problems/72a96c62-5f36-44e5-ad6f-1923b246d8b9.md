1) Show File Save Dialog

**Spec:** Write a function that displays a Windows file save dialog with two file type filters (Text files and All files) and returns the selected file path as a String, or None if the user cancels. The function must handle COM initialization and cleanup.

**Constraints:**
- Use the `windows` crate and the `IFileSaveDialog` COM interface.
- Initialize COM with `CoIncrementMTAUsage`.
- Set the file types to: "Text files" (*.txt) and "All files" (*.*).
- If the user selects a file, get the file system path and convert it to a Rust String.
- Free the memory allocated for the file path string using `CoTaskMemFree`.
- Return `Ok(None)` if the user cancels, and `Ok(Some(path))` if a file is selected.
- Propagate any COM errors as `windows::core::Error`.

**Signature:**
```rust
fn show_save_dialog() -> windows::core::Result<Option<String>>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    match show_save_dialog()? {
        Some(path) => println!("User picked: {}", path),
        None => println!("User canceled"),
    }
    Ok(())
}
```
