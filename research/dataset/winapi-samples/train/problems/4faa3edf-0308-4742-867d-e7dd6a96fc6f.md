1) Get Desktop Monitor Bounds

**Spec:** Write a function that sets the process DPI aware and returns the bounding rectangle of the monitor containing the desktop window.

**Constraints:**
- Must use Win32 APIs for DPI awareness and monitor information
- Must handle the desktop window (HWND 0) correctly
- Must return the monitor bounds as a RECT structure

**Signature:**
```rust
fn get_desktop_monitor_bounds() -> windows::core::Result<windows::Win32::Foundation::RECT>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    let rect = get_desktop_monitor_bounds()?;
    println!("Monitor bounds: ({}, {}) to ({}, {})", rect.left, rect.top, rect.right, rect.bottom);
    Ok(())
}
```
