**Spec:** Return the aligned base, the delta (offset from the base), and the allocation granularity for a requested offset, using only bitwise operations for alignment (no division or modulo).
**Constraints:** Use `GetSystemInfo`; alignment is non-negotiable; the allocation granularity is guaranteed to be a power of two; do not use division or modulo operators.
**Signature:**
```rust
pub fn align_mapping_offset_bitwise(offset: u64) -> Result<(u64, u64, u32)>;
```
**Example:**
```rust
let (base, delta, g) = align_mapping_offset_bitwise(1_234_567)?;
```
