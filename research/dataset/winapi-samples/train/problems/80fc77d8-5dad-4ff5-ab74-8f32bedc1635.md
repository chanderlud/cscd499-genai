**Spec:** Write a function that performs OCR on a PNG image and returns each recognized word along with its bounding rectangle coordinates (x, y, width, height) in pixels.

**Constraints:**
- Must use the Windows OCR engine and imaging APIs.
- Bounding box coordinates must be relative to the original image dimensions.
- Return a vector of structs containing the word text and its bounding rectangle.
- Use the system's default language for recognition.

**Signature:**
```rust
pub fn ocr_with_bounding_boxes(path: &str) -> windows::core::Result<Vec<WordRect>>
```

**Example:**
```rust
let words = ocr_with_bounding_boxes("sample.png")?;
for word in words {
    println!("'{}' at ({}, {}) {}x{}", word.text, word.rect.x, word.rect.y, word.rect.width, word.rect.height);
}
```
