Write a function that creates and displays a toast notification with an inline reply text box that captures user input.

**Spec:**
- Create a toast notification with a title, message, and an inline reply text box
- The reply text box must have placeholder text and a submit button
- The toast must use the "ToastGeneric" template
- Return a Result indicating success or failure

**Constraints:**
- Use `windows::Data::Xml::Dom::XmlDocument` for XML construction
- Use `windows::UI::Notifications::ToastNotificationManager` for notification management
- The toast must include proper XML escaping for user-provided text
- Must work on Windows 10 or later (use "ToastGeneric" template)

**Signature:**
```rust
fn show_reply_toast(app_id: &str, title: &str, message: &str, placeholder: &str) -> windows::core::Result<()>
```

**Example:**
```rust
show_reply_toast(
    "MyAppId",
    "New Message",
    "Alice: Hey, are you available?",
    "Type your reply..."
)?;
```
