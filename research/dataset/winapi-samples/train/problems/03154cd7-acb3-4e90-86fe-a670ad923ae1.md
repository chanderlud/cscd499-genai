1) Key State Query

**Spec:** Write a function that returns the current state of a specified virtual key as a byte. Bit 7 (0x80) should be set if the key is currently pressed. For toggle keys (VK_CAPITAL, VK_NUMLOCK, VK_SCROLL), bit 0 (0x01) should be set if the toggle is on.

**Constraints:** Must use `GetAsyncKeyState` to check the pressed state and `GetKeyState` to check the toggle state for the three specified toggle keys.

**Signature:**
```rust
fn get_key_state(vk: VIRTUAL_KEY) -> u8
```

**Example:**
```rust
let lshift_state = get_key_state(VK_LSHIFT);
let caps_state = get_key_state(VK_CAPITAL);
println!("Left Shift is down: {}", (lshift_state & 0x80) != 0);
println!("Caps Lock is on: {}", (caps_state & 0x01) != 0);
```
