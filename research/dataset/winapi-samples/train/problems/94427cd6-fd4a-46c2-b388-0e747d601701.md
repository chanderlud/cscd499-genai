**Spec:** Write a function that enumerates all top-level windows and returns a vector of (handle, title) for each window that is visible, enabled, not cloaked by the Desktop Window Manager (DWM), and does not have the extended window styles WS_EX_APPWINDOW or WS_EX_NOACTIVATE. The window titles must be truncated to at most 256 characters. The vector must be sorted by the window handle in ascending order.

**Constraints:**
- Use the `windows` crate for Win32 bindings.
- The function must handle errors from Win32 API calls appropriately.
- The function must be safe to call (i.e., encapsulate unsafe operations).
- When retrieving the window title via `GetWindowTextW`, use a fixed-size buffer of 256 characters. The title in the returned vector must be a `String` that is at most 256 characters long.
- The returned vector must be sorted by the window handle in ascending order.

**Signature:**
```rust
pub fn enumerate_windows_sorted() -> windows::core::Result<Vec<(windows::Win32::Foundation::HWND, String)>>
```

**Example:**
```rust
fn main() {
    match enumerate_windows_sorted() {
        Ok(windows) => {
            for (hwnd, title) in windows {
                println!("Window handle: {:?}, Title: {}", hwnd, title);
            }
        }
        Err(e) => eprintln!("Error: {:?}", e),
    }
}
```
