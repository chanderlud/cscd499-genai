1) Enumerate Credentials

**Spec:** Write a function that uses the Windows Credential Manager API to enumerate all stored credentials and print their details.

**Constraints:**
- Must use `CredEnumerateW` from the `windows` crate (module `Win32::Security::Credentials`)
- Must use the `CRED_ENUMERATE_ALL_CREDENTIALS` flag to retrieve all credentials
- Must properly free the allocated credential array using `CredFree`
- Must handle unsafe code and return a `windows::core::Result`

**Signature:**
```rust
fn enumerate_credentials() -> windows::core::Result<()>
```

**Example:**
```rust
enumerate_credentials()?;
// Output:
// /* CREDENTIALW */
// Type: SomeType
// Target: SomeTarget
// User name: SomeUser
```
