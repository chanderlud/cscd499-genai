**Spec:** Write a function that exports an ECC private key handle to a byte vector containing the uncompressed point format (x, y coordinates followed by the private scalar). The function should retrieve the key blob via `BCryptExportKey` using `BCRYPT_ECCPRIVATE_BLOB`, then parse and return the raw key material. The function must validate that the exported key is an ECC private key by checking the magic value in the header.

**Constraints:**
- Must use `BCryptExportKey` with `BCRYPT_ECCPRIVATE_BLOB` format.
- Must correctly parse the `BCRYPT_ECCKEY_BLOB` header to determine key length.
- Must validate the key type by checking the magic value against known ECC private key magic constants.
- Must extract and concatenate the x-coordinate, y-coordinate, and private scalar in that order.
- Must handle errors from the Windows API and return them as `windows::core::Error`.
- Must not leak memory; the exported blob buffer must be freed.

**Signature:**
```rust
fn export_ecc_private_key(key_handle: BCRYPT_KEY_HANDLE) -> windows::core::Result<Vec<u8>>
```

**Example:**
```rust
let key_handle = /* obtained via BCryptImportKeyPair or BCryptGenerateKeyPair */;
let private_key_bytes = export_ecc_private_key(key_handle)?;
// private_key_bytes now contains: [x_bytes, y_bytes, private_scalar_bytes]
