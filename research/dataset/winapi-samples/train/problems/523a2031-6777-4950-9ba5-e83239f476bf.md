**Spec:**
- Write a function that exports an RSA public key from a CNG key handle into a DER-encoded SubjectPublicKeyInfo structure.
- The function must retrieve the RSA public key blob using `BCryptExportKey`, then convert it to DER format.
- The DER output must be a valid ASN.1 SEQUENCE containing the RSA public key (modulus and exponent) per RFC 5280.
- The DER encoding must be built manually without using an external ASN.1 library.

**Constraints:**
- Use `BCryptExportKey` with `BCRYPT_RSAPUBLIC_BLOB` to obtain the key blob.
- The function must handle key handles from both imported and generated RSA keys.
- The DER encoding must be correct and minimal (no unnecessary padding).
- Return the DER bytes as a `Vec<u8>`.

**Signature:**
```rust
pub fn export_rsa_public_key_to_der(
    key_handle: BCRYPT_KEY_HANDLE,
) -> windows::core::Result<Vec<u8>>
```

**Example:**
```rust
// Assume `key_handle` is a valid RSA public key handle
let der_bytes = export_rsa_public_key_to_der(key_handle)?;
println!("DER encoded public key ({} bytes)", der_bytes.len());
```
