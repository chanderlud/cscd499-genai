**Spec:** Initialize COM on the current thread and return a token that must be explicitly uninitialized. Treat every successful `CoInitializeEx` call as requiring a matching `CoUninitialize`. Support both MTA and STA via a parameter. Treat both `S_OK` and `S_FALSE` as success. Return an error for invalid flags or an incompatible threading model.
**Constraints:** Do not use RAII (i.e., no Drop implementation). The token must be consumed by an explicit uninitialize function. The token must be `!Send` and `!Sync` to prevent cross-thread usage.

```rust
impl ComToken {
    pub fn uninit(self);
}

pub fn com_init(coinit: i32) -> windows::core::Result<ComToken>;
```

```rust
let token = com_init(windows::Win32::System::Com::COINIT_MULTITHREADED.0 as i32)?;
```
