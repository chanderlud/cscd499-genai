**Spec:**
- Write a function that parses the import table of a PE binary and returns a list of imported DLLs and their functions.
- The function must handle both 32-bit and 64-bit PE files.
- For each imported DLL, return its name and a vector of imported functions, where each function is either by name (with hint) or by ordinal.

**Constraints:**
- Must validate DOS and NT signatures; return an error if invalid.
- Must correctly convert RVAs to file offsets using section headers.
- Must handle both PE32 (0x10b) and PE32+ (0x20b) formats.
- Must not panic on malformed data; return appropriate errors.

**Signature:**
```rust
pub enum ImportFunction {
    ByName { name: String, hint: u16 },
    ByOrdinal(u16),
}

pub fn parse_import_table(data: &[u8]) -> windows::core::Result<Vec<(String, Vec<ImportFunction>)>>
```

**Example:**
```rust
let data = std::fs::read("example.dll")?;
let imports = parse_import_table(&data)?;
for (dll_name, functions) in imports {
    println!("DLL: {}", dll_name);
    for func in functions {
        match func {
            ImportFunction::ByName { name, hint } => println!("  {} (hint: {})", name, hint),
            ImportFunction::ByOrdinal(ord) => println!("  Ordinal: {}", ord),
        }
    }
}
```
