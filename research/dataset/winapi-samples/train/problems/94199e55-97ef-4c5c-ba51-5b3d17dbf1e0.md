**Spec:**
- Obtain the foreground window handle using `GetForegroundWindow`.
- Acquire the IME context for that window using `ImmGetContext`.
- Query the composition string attributes using `ImmGetCompositionStringW` with `GCS_COMPATTR`.
- Return the raw attribute bytes as a `Vec<u8>` wrapped in a `Result`.

**Constraints:**
- Must use `GetForegroundWindow`, `ImmGetContext`, `ImmGetCompositionStringW`, and `ImmReleaseContext` from the Windows API.
- Handle cases where no composition string is active or no IME context is available (return an empty vector).
- Properly release the IME context with `ImmReleaseContext` using a guard pattern.

**Signature:**
```rust
fn get_composition_attributes() -> windows::core::Result<Vec<u8>>
```

**Example:**
```rust
match get_composition_attributes() {
    Ok(attrs) => println!("Composition attributes: {:?}", attrs),
    Err(e) => eprintln!("Failed to get attributes: {}", e),
}
```
