**Spec:**
- Get the current IME conversion mode for the foreground window
- If currently in default mode (0), set it to the provided `target_mode` parameter
- If currently in any non-default mode, set it back to default (0)
- Return the new conversion mode value after toggling

**Constraints:**
- Must use `ImmGetDefaultIMEWnd` and `SendMessageA` with `WM_IME_CONTROL`
- Handle cases where no foreground window exists or IME window cannot be obtained
- The function should be safe to call repeatedly (idempotent toggle behavior)

**Signature:**
```rust
fn toggle_ime_conversion_mode(target_mode: isize) -> windows::core::Result<isize>
```

**Example:**
```rust
// Toggle to Chinese mode (3) if currently English, or back to English if not
let new_mode = toggle_ime_conversion_mode(3)?;
println!("Conversion mode is now: {}", new_mode);
```
