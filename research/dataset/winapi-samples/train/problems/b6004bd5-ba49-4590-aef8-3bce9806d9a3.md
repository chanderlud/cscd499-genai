**Spec:** Persist mapped changes to disk by flushing the view and then the file buffers, without using any local variables.
**Constraints:**

   - The function body must not contain any `let` bindings (i.e., no local variables).
**Signature:**
```rust
pub fn flush_mapped(view_ptr: *const u8, len: usize, file: &impl AsRawHandle) -> windows::core::Result<()>;
```
**Example:**
