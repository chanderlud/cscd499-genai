**Spec:**
- Write a function that enumerates all threads of a target process and suspends each one
- First, find the target process by name using process enumeration (with `TH32CS_SNAPPROCESS`) to get its process ID
- Then, use `CreateToolhelp32Snapshot` with `TH32CS_SNAPTHREAD` to enumerate threads
- Filter threads belonging to the target process using the found process ID (via `th32OwnerProcessID`)
- Call `OpenThread` with `THREAD_SUSPEND_RESUME` access and `SuspendThread` on each
- Return a vector of thread handles that were successfully suspended

**Constraints:**
- Must properly close all opened handles (thread handles and snapshot handles)
- Handle cases where thread enumeration fails or thread cannot be opened
- Return an error if the target process is not found
- Use `windows::core::Result<T>` for error handling

**Signature:**
```rust
pub fn suspend_process_threads(process_name: &str) -> windows::core::Result<Vec<windows::Win32::Foundation::HANDLE>>
```

**Example:**
```rust
let thread_handles = suspend_process_threads("notepad.exe")?;
println!("Suspended {} threads", thread_handles.len());
// Later: resume threads with ResumeThread and close handles
```
