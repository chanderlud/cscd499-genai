1) Get Instance Handle

**Spec:** Write a function that returns the instance handle (HMODULE) of the current module by using the `__ImageBase` linker symbol.

**Constraints:**
- Use the `windows` crate and its `Win32::Foundation::HMODULE` and `Win32::System::SystemServices::IMAGE_DOS_HEADER` types.
- Must work for both executables and DLLs (avoid `GetModuleHandle(NULL)`).
- Use the `__ImageBase` symbol via an extern block.

**Signature:**
```rust
fn get_instance_handle() -> HMODULE
```

**Example:**
```rust
let instance = get_instance_handle();
println!("Instance handle: {:?}", instance);
```
