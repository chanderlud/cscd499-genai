1) Allocate Console and Get Executable Path

**Spec:** Write a function that allocates a new console for the current process and returns the full path of the current executable as a `String`.

**Constraints:**
- Use the `windows` crate and the Win32 APIs `AllocConsole` and `GetModuleFileNameW`.
- The function must handle errors and return a `Result<String>`.
- The function must print a success message after allocating the console.

**Signature:**
```rust
fn allocate_console_and_get_path() -> windows::core::Result<String>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    let path = allocate_console_and_get_path()?;
    println!("Current executable path: {}", path);
    Ok(())
}
```
