1) Enumerate Devices with DeviceWatcher

**Spec:** Write a function that creates a `DeviceWatcher` to enumerate devices. The function must set up event handlers to print the name of each device as it is added and print a completion message when enumeration finishes. The watcher should run for a specified duration before the function returns.

**Constraints:**
- Use the `windows` crate and the `DeviceInformation::CreateWatcher` method.
- Handle the `Added` and `EnumerationCompleted` events with `TypedEventHandler`.
- The function must block the current thread for the specified duration using `std::thread::sleep`.

**Signature:**
```rust
fn enumerate_devices(duration_secs: u64) -> windows::core::Result<()>
```

**Example:**
```rust
// Enumerate devices for 10 seconds, printing names as they are added
enumerate_devices(10)?;
```
