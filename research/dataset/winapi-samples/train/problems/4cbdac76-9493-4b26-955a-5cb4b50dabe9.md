TITLE: Read File in Separate Thread
PROBLEM: Implement a program that reads the first 100 bytes of a file named "test.txt" in a separate thread and prints the data or error.

**Spec:** The program should use the `windows` crate to open and read the file. It should spawn a thread to perform the file I/O and use a channel to communicate the result (either the data as a `Vec<u8>` or an error) back to the main thread. The main thread should wait for the result and print the number of bytes read and the data, or print an error message.

**Constraints:**
- Use the `windows` crate for Windows file APIs (CreateFileW, ReadFile, CloseHandle)
- Use `std::thread` to spawn a thread for file I/O
- Use `std::sync::mpsc` channel for communication between threads
- Handle errors appropriately and ensure the file handle is closed in all cases
- The file path should be "test.txt" (relative to the current directory)

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// Running the program will read the first 100 bytes of "test.txt" and print them.
// If the file does not exist or cannot be read, an error message is printed.
```
