Create and Signal an Event

**Spec:** Write a function that creates a manual-reset event in the non-signaled state, sets it to signaled, and then waits for it with a zero timeout.

**Constraints:**
- Use the `windows` crate and Win32 threading APIs
- The event must be created as a manual-reset event (initially non-signaled)
- The event must be set to the signaled state
- The wait must use a timeout of 0 (immediate return)
- The function must return a `windows::core::Result<()>`

**Signature:**
```rust
fn create_set_and_wait_event() -> windows::core::Result<()>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    create_set_and_wait_event()?;
    Ok(())
}
```
