**Spec:** Write a function that computes the SHA-256 hash of a given byte slice using the BCrypt API. The function should initialize the hash algorithm provider once per thread using a thread-local lazy initialization pattern and reuse it for subsequent calls.

**Constraints:**
- Use `BCryptOpenAlgorithmProvider` with `BCRYPT_SHA256_ALGORITHM`, `PCWSTR::null()`, and `BCRYPT_OPEN_ALGORITHM_PROVIDER_FLAGS(0)`
- Use `BCryptCreateHash`, `BCryptHashData`, and `BCryptFinishHash` to compute the hash
- The algorithm provider handle must be initialized only once per thread using a thread-local lazy initialization pattern
- Return the hash as a 32-byte array
- Clean up hash handles appropriately after each computation using `BCryptDestroyHash`
- Ensure hash handle cleanup occurs even if hash computation fails

**Signature:**
```rust
pub fn sha256_hash(data: &[u8]) -> windows::core::Result<[u8; 32]>
```

**Example:**
```rust
let data = b"hello world";
let hash = sha256_hash(data).unwrap();
// hash should equal the SHA-256 hash of "hello world"
```
