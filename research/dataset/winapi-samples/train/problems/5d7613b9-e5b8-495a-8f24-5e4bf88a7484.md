N) Enumerate Visible Top-Level Windows

**Spec:** Write a function that enumerates all top-level windows and prints the title of each window that is visible, enabled, not cloaked by DWM, does not have tool window or no-activate extended styles, and has a non-empty title.

**Constraints:**
- Use the Win32 API via the `windows` crate
- Use `EnumWindows` with a callback function to enumerate windows
- Check window visibility with `IsWindowVisible`
- Check window enabled state with `IsWindowEnabled`
- Check if window is cloaked using `DwmGetWindowAttribute` with `DWMWA_CLOAKED`
- Check extended window styles with `GetWindowLongW` for `WS_EX_APPWINDOW`, `WS_EX_NOACTIVATE`, and `WS_EX_TOOLWINDOW`
- Get window title with `GetWindowTextW` only if `GetWindowTextLengthW` returns > 0
- Print each qualifying window's title in the format: "Visible window: {title}"

**Signature:**
```rust
fn print_visible_windows() -> windows::core::Result<()>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    print_visible_windows()
}
// Output might include lines like:
// Visible window: Calculator
// Visible window: File Explorer
```
