1) Create an Edit Control with Custom Font

**Spec:** Create a Win32 window application that displays an edit control (text input field) with a custom font.

**Constraints:**
- Use the `windows` crate for Win32 bindings
- Create a popup window (WS_POPUP) with the title "Edit Control Example"
- Create an edit control as a child window (WS_CHILD) of the main window
- Position the edit control at (10,10) with width = client width - 20 and height = 30
- Set the edit control's font to "Segoe UI" with height -24 (24 pixels)

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// In WM_CREATE handler:
let edit_control = CreateWindowExW(
    Default::default(),
    w!("Edit"),
    w!(""),
    WS_CHILD | WS_VISIBLE,
    10, 10, rect.right - 20, 30,
    Some(hwnd),
    None,
    Some(GetModuleHandleW(None)?.into()),
    None,
)?;

let font = CreateFontW(-24, 0, 0, 0, 400, 0, 0, 0, 
    FONT_CHARSET(0), FONT_OUTPUT_PRECISION(0), 
    FONT_CLIP_PRECISION(0), FONT_QUALITY(0), 0, w!("Segoe UI"));

SendMessageW(edit_control, WM_SETFONT, Some(WPARAM(font.0 as usize)), Some(LPARAM(1)));
```
