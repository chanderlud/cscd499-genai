**Spec:**
- Create a Win32 window application that displays a window and runs a standard message loop.
- The window should have a white background and handle the WM_PAINT message by validating the window area.
- The window should handle the WM_DESTROY message by posting a quit message.

**Constraints:**
- Use the Windows.Win32.UI.WindowsAndMessaging and related modules for window creation and message handling.
- The window class must be registered with a window procedure that handles at least WM_DESTROY and WM_PAINT.
- The program must create a window with the style WS_OVERLAPPEDWINDOW and show it.
- The program must run a message loop that retrieves and dispatches messages until a WM_QUIT message is received.

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    // Initialize window class and create window
    // Run message loop until WM_QUIT
    Ok(())
}
```
