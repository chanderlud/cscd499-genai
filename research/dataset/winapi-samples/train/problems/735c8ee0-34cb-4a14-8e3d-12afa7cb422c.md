N) Read a Registry String Value

**Spec:**  
Write a function `get_registry_string` that opens a specified Windows registry key and reads a string value (REG_SZ) from it, returning the value as a Rust `String`.

**Constraints:**  
- Use the `windows` crate for Win32 API bindings.
- The function must open the key with `KEY_READ` access.
- It must handle the two-step registry query pattern: first call to get required buffer size, second call to read data.
- The registry key must be properly closed even if errors occur during value reading.
- Return `Err` if the value type is not `REG_SZ` or if any Win32 API call fails.

**Signature:**  
```rust
fn get_registry_string(hive: HKEY, path: &str, value_name: &str) -> Result<String>
```

**Example:**  
```rust
let product_name = get_registry_string(
    HKEY_LOCAL_MACHINE,
    "SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion",
    "ProductName"
)?;
println!("Windows Product Name: {}", product_name);
```
