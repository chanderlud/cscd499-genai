N) Enumerate TCP Connections and Owning Processes

**Spec:** Write a Rust program that uses the Windows API to enumerate all active TCP connections (IPv4) and their owning processes, displaying local/remote addresses, ports, process IDs, and process names in a formatted table.

**Constraints:**
- Use the IP Helper API (`GetExtendedTcpTable`) to retrieve TCP connection data
- Use the Threading API (`OpenProcess`, `QueryFullProcessImageNameW`) to resolve process names from PIDs
- Handle buffer size negotiation for `GetExtendedTcpTable` (call twice: first for size, then for data)
- Extract only the filename from full process image paths
- Format output as aligned columns with headers

**Signature:**
```rust
fn main() -> windows::core::Result<()>
```

**Example:**
```rust
// Program output (abbreviated):
// Found 3 TCP connections:
// Local Address        Remote Address       PID        Process Name
// ----------------------------------------------------------------------
// 192.168.1.100:52341  93.184.216.34:443    1234       chrome.exe
// 0.0.0.0:135          0.0.0.0:0            892        svchost.exe
// 127.0.0.1:54321      127.0.0.1:8080       5678       node.exe
```
