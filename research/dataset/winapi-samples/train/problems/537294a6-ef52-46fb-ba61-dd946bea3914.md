1) Subclass a ListBox to Handle Keyboard Input

**Spec:** Create a Win32 window containing a list box control that is subclassed to handle keyboard navigation and selection. The list box must respond to Escape (exit application), Return (select current item), and Up/Down arrow keys (navigate items).

**Constraints:**
- Use the `windows` crate for Win32 API calls
- The list box must be created as a child window with `LBS_NOTIFY` and `WS_VSCROLL` styles
- Populate the list box with at least 10 items
- Implement window subclassing via `SetWindowLongPtrW`/`GetWindowLongPtrW` with `GWL_WNDPROC`
- Store application context using `GWLP_USERDATA`
- Handle `WM_KEYDOWN` and `WM_KEYUP` messages in the subclass procedure
- Call the original window procedure for default handling of Up/Down navigation

**Signature:**
```rust
fn main() -> windows::core::Result<()>;
extern "system" fn window_proc(hwnd: HWND, msg: u32, wparam: WPARAM, lparam: LPARAM) -> LRESULT;
extern "system" fn list_box_proc(hwnd: HWND, msg: u32, wparam: WPARAM, lparam: LPARAM) -> LRESULT;
```

**Example:**
```rust
// Inside window_proc WM_CREATE handler:
let list_box = CreateWindowExW(
    WINDOW_EX_STYLE(0),
    w!("ListBox"),
    w!(""),
    WS_CHILD | WS_VISIBLE | WINDOW_STYLE(LBS_NOTIFY.try_into().unwrap()) | WS_VSCROLL,
    0, 0, rect.right, rect.bottom,
    Some(hwnd), None, Some(instance), None,
)?;

let original_proc_ptr = SetWindowLongPtrW(list_box, GWL_WNDPROC, list_box_proc as isize);
```
