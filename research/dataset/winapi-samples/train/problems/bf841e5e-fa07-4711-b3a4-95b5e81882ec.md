**Spec:**
- Write a function that exports an RSA private key from a CNG key handle to DER-encoded PKCS#1 format.
- The function must retrieve the RSA private key blob using `BCryptExportKey` with the `BCRYPT_RSAFULLPRIVATE_BLOB` type.
- Parse the `BCRYPT_RSAKEY_BLOB` header and extract the following components in the order they appear in the blob: public exponent, modulus, prime1, prime2, private exponent, exponent1, exponent2, coefficient.
- Encode these components into a DER-encoded ASN.1 `RSAPrivateKey` structure (PKCS#1) with the following fields (in order): version (0), modulus, publicExponent, privateExponent, prime1, prime2, exponent1, exponent2, coefficient.

**Constraints:**
- The input key handle must be an RSA private key (full private key, as required by `BCRYPT_RSAFULLPRIVATE_BLOB`).
- The function must handle errors and return a `windows::core::Result<Vec<u8>>`.
- The output must be a valid DER byte vector representing the RSA private key in PKCS#1 format.

**Signature:**
```rust
pub fn export_rsa_private_key_to_der(
    key_handle: BCRYPT_KEY_HANDLE,
) -> windows::core::Result<Vec<u8>>
```

**Example:**
```rust
let key_handle = /* assume an RSA private key handle */;
let der_bytes = export_rsa_private_key_to_der(key_handle)?;
// der_bytes now contains the DER-encoded RSA private key
```
