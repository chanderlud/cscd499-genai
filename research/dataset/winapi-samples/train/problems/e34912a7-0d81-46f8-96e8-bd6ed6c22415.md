**Spec:** Write a function that sets the maximum frame latency for the DXGI device underlying a WinRT `IDirect3DDevice`. Use the existing `get_d3d_interface_from_object` pattern to retrieve the `IDXGIDevice1` interface and call its `SetMaximumFrameLatency` method. The function must return an error if `max_latency` is zero.

**Constraints:**
- Must use the provided `get_d3d_interface_from_object` helper to obtain the `IDXGIDevice1`.
- The function should propagate any errors from the Win32 API calls.
- The `max_latency` parameter must be a non-zero `u32`; passing zero returns `E_INVALIDARG`.

**Signature:**
```rust
pub fn set_max_frame_latency(device: &IDirect3DDevice, max_latency: u32) -> Result<()>
```

**Example:**
```rust
// Assume `create_d3d_device` and `create_direct3d_device` are available
let d3d_device = create_d3d_device()?;
let direct3d_device = create_direct3d_device(&d3d_device)?;
set_max_frame_latency(&direct3d_device, 2)?;
```
