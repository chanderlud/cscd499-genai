1) List UDP Connections with Process Names

**Spec:** Write a function that retrieves all active UDP connections for IPv4 and returns a vector of structs containing the local socket address, owning process ID, and process name (if available).

**Constraints:**
- Use the Win32 API `GetExtendedUdpTable` to retrieve the UDP table with owner PID information
- Use `OpenProcess` and `QueryFullProcessImageNameW` to resolve process names from PIDs
- Handle buffer allocation and size queries correctly for `GetExtendedUdpTable`
- Return `windows::core::Result` for error handling
- Process names should be extracted as filenames only (without full path)

**Signature:**
```rust
use std::net::SocketAddr;
use windows::core::Result;

struct UdpConnection {
    local_addr: SocketAddr,
    pid: u32,
    process_name: Option<String>,
}

fn get_udp_connections() -> Result<Vec<UdpConnection>> {
    // Implementation here
}
```

**Example:**
```rust
fn main() -> Result<()> {
    let connections = get_udp_connections()?;
    for conn in connections {
        println!("Local: {} -> PID: {} ({})", 
            conn.local_addr, 
            conn.pid, 
            conn.process_name.unwrap_or_else(|| "Unknown".to_string()));
    }
    Ok(())
}
```
