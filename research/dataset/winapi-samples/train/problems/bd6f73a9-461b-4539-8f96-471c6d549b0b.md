**Spec:**
- Implement a function that encrypts plaintext using AES-GCM with a provided key and nonce.
- The function must use BCrypt to perform the encryption and return the ciphertext and authentication tag.
- The algorithm provider for AES must be initialized with the appropriate flags for GCM mode.

**Constraints:**
- Use `windows::core::Result<T>` for error handling.
- The key must be 256 bits (32 bytes) and the nonce must be 96 bits (12 bytes) as required by AES-GCM.
- The function should handle the algorithm provider initialization internally, using a thread-local OnceCell for efficiency.

**Signature:**
```rust
pub fn encrypt_aes_gcm(
    key: &[u8; 32],
    nonce: &[u8; 12],
    plaintext: &[u8],
    aad: &[u8],
) -> windows::core::Result<(Vec<u8>, [u8; 16])>
```

**Example:**
```rust
let key = [0u8; 32];
let nonce = [0u8; 12];
let plaintext = b"secret message";
let aad = b"additional data";
let (ciphertext, tag) = encrypt_aes_gcm(&key, &nonce, plaintext, aad).unwrap();
```
