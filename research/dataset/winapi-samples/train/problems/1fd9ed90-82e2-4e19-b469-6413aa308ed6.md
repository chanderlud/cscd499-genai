1) Retrieve Network Adapters and DNS Servers

**Spec:** Write a function that uses the `GetAdaptersAddresses` Win32 API to retrieve all network adapters and their DNS server addresses. The function must handle the `ERROR_BUFFER_OVERFLOW` pattern by first querying for required buffer size, then allocating and filling the buffer. It should iterate through the linked list of adapters, printing each adapter's description and its DNS server addresses (filtering out unspecified and site-local IPv6 addresses).

**Constraints:**
- Use `GetAdaptersAddresses` with `AF_UNSPEC` and `GAA_FLAG_INCLUDE_TUNNEL_BINDINGORDER` flags
- Handle `ERROR_BUFFER_OVERFLOW` by allocating a buffer with extra alignment headroom
- Properly align the buffer to `IP_ADAPTER_ADDRESSES_LH` boundary
- Iterate through the linked list using the `Next` pointer
- For DNS addresses, handle both IPv4 (`AF_INET`) and IPv6 (`AF_INET6`) socket address families
- Filter out unspecified IP addresses and unicast site-local IPv6 addresses

**Signature:**
```rust
fn print_adapters_dns() -> windows::core::Result<()>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    print_adapters_dns()
}
```
