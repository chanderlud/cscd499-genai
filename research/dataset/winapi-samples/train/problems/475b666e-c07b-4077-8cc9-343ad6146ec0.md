1) Check Caps Lock State

**Spec:** Write a function that retrieves the current keyboard state using the Win32 API and determines whether Caps Lock is enabled.

**Constraints:**
- Must use the `GetKeyboardState` function from the `windows` crate
- Must handle the unsafe Win32 API call appropriately
- Must return a `Result` type for error handling
- The keyboard state buffer must be a 256-byte array

**Signature:**
```rust
fn get_kbd_state() -> Result<[u8; 256]>
```

**Example:**
```rust
use windows::core::Result;
use windows::Win32::UI::Input::KeyboardAndMouse::{VIRTUAL_KEY, VK_CAPITAL};

fn main() -> Result<()> {
    let kbd_state = get_kbd_state()?;
    let caps_lock_on = kbd_state[usize::from(VK_CAPITAL.0)] & 1 != 0;
    println!("Caps Lock is {}", if caps_lock_on { "ON" } else { "OFF" });
    Ok(())
}
```
