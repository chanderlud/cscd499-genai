**Spec:** Write a function that retrieves the memory address of the `GetTickCount` function from `kernel32.dll` using Win32 API calls.

**Constraints:**
- Use the `windows` crate for Win32 bindings
- Must use `GetModuleHandleW` to obtain the module handle for `kernel32.dll`
- Must use `GetProcAddress` to retrieve the function address
- Return the address as a `usize` wrapped in a `Result`
- Handle errors appropriately using the `windows::core::Error` type

**Signature:**
```rust
fn get_tick_count_addr() -> windows::core::Result<usize>
```

**Example:**
```rust
fn main() -> windows::core::Result<()> {
    let addr = get_tick_count_addr()?;
    println!("GetTickCount address: {:p}", addr as *const ());
    Ok(())
}
```
