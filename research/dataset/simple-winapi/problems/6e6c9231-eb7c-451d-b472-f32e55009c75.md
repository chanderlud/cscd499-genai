## 4) Overlapped File Write With Event Completion

* **Spec:** Create/overwrite a file and write `data` using overlapped I/O. Wait for completion via an event stored in the `OVERLAPPED` struct. Return the number of bytes written.
* **Constraints:**

    * File must be opened with `CreateFileW(... FILE_FLAG_OVERLAPPED ...)`.
    * Use `WriteFile` with an `OVERLAPPED` containing an event from `CreateEventW`.
    * If `WriteFile` returns `ERROR_IO_PENDING`, wait with `WaitForSingleObject` then use `GetOverlappedResult`.
    * Must close all handles.
