24) Mark file as sparse (FSCTL_SET_SPARSE)

* **Spec:** Enable sparse mode on a file to efficiently represent holes.
* **Constraints:** NTFS/ReFS; requires handle with write access; use `DeviceIoControl`.
* **Signature:**

```rust
pub fn set_sparse(handle: &OwnedHandle) -> Result<()>;
```

* **Example:**

```rust
set_sparse(&h)?;
```
