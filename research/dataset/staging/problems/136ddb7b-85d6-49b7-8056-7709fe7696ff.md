16. APC-based waitable timer callback (alertable wait)

* **Spec:** Schedule a waitable timer with a completion routine; enter an alertable wait; return whether the APC ran.
* **Constraints:** `CreateWaitableTimerW`, `SetWaitableTimer` with completion routine, `SleepEx(alertable=true)` or `WaitForSingleObjectEx`.
* **Signature:**

```rust
pub fn apc_timer_fires(due_ms: i64, timeout_ms: u32) -> Result<bool>;
```

* **Example:**

```rust
let ok = apc_timer_fires(50, 1000)?;
```
