17. COM inter-thread marshaling (the “COM is fun” special)

* **Spec:** Create an `IStream` on one thread, marshal it to another thread, write data there, return to original thread and read it back.
* **Constraints:** Use `CreateStreamOnHGlobal` + `CoMarshalInterThreadInterfaceInStream` + `CoGetInterfaceAndReleaseStream`.
* **Signature:**

```rust
pub fn com_marshal_stream_roundtrip(data: &[u8]) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let out = com_marshal_stream_roundtrip(b"cross-thread bytes")?;
```
