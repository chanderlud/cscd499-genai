13. Write/read HKCU registry string (Win32)

* **Spec:** Set a REG_SZ value and read it back.
* **Constraints:** Use `RegCreateKeyExW`, `RegSetValueExW`, `RegGetValueW`. Use **HKCU** only.
* **Signature:**

```rust
pub fn reg_set_get_hkcu(path: &str, name: &str, value: &str) -> Result<String>;
```

* **Example:**

```rust
let v = reg_set_get_hkcu(r"Software\MyApp\Tests", "Answer", "42")?;
```
