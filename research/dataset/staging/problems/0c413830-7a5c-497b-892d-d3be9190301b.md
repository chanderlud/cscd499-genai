24. Canonical absolute path (Win32)

* **Spec:** Convert an input path (possibly relative) into an absolute path.
* **Constraints:** Use `GetFullPathNameW` (not `std::fs::canonicalize`). Don’t resolve symlinks; just normalize to absolute.
* **Signature:**

```rust
pub fn full_path(path: &Path) -> Result<PathBuf>;
```

* **Example:**

```rust
let p = full_path(Path::new(r"..\data\file.txt"))?;
```
