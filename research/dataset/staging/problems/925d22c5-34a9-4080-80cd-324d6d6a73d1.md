4. Message-mode named pipe ping-pong (server + client)

* **Spec:** Start a pipe server that reads one message and replies with another; connect a client and return the reply.
* **Constraints:** Use `CreateNamedPipeW` (message mode) + `ConnectNamedPipe` + `CreateFileW` + `ReadFile`/`WriteFile`.
* **Signature:**

```rust
pub fn pipe_request_reply(pipe_name: &str, request: &[u8]) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let reply = pipe_request_reply(r"\\.\pipe\unit_test_pipe", b"ping")?;
```
