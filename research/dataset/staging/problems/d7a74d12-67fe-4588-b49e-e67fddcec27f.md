16) Create hard link (CreateHardLinkW)

* **Spec:** Create a hard link to an existing file.
* **Constraints:** Same volume; not for directories; requires NTFS/ReFS support.
* **Signature:**

```rust
pub fn create_hard_link(new_link: &Path, existing: &Path) -> Result<()>;
```

* **Example:**

```rust
create_hard_link(Path::new("alias.txt"), Path::new("real.txt"))?;
```
