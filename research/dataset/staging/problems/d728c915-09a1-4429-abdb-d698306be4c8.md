03) Write entire file (Create + truncate + WriteFile loop)

* **Spec:** Create/truncate a file and write all provided bytes.
* **Constraints:** Ensure full write; use `CREATE_ALWAYS`; flush optionally.
* **Signature:**

```rust
pub fn write_all(path: &Path, data: &[u8]) -> Result<()>;
```

* **Example:**

```rust
write_all(Path::new(r"C:\tmp\out.bin"), b"hello")?;
```
