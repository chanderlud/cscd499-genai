12. SHA-256 hash using BCrypt (Win32)

* **Spec:** Compute SHA-256 of input bytes.
* **Constraints:** Use CNG (`BCryptOpenAlgorithmProvider`, `BCryptCreateHash`, `BCryptHashData`, `BCryptFinishHash`).
* **Signature:**

```rust
pub fn sha256(data: &[u8]) -> Result<[u8; 32]>;
```

* **Example:**

```rust
let digest = sha256(b"abc")?;
```
