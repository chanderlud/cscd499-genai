22) Lock/unlock a file region (LockFileEx/UnlockFileEx)

* **Spec:** Exclusive lock region for concurrency control.
* **Constraints:** Must keep handle open; coordinate offset/length across processes.
* **Signature:**

```rust
pub fn lock_region(handle: &OwnedHandle, offset: u64, len: u64) -> Result<()>;
pub fn unlock_region(handle: &OwnedHandle, offset: u64, len: u64) -> Result<()>;
```

* **Example:**

```rust
lock_region(&h, 0, 4096)?;
unlock_region(&h, 0, 4096)?;
```
