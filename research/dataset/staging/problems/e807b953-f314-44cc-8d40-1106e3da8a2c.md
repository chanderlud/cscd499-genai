15. CreateProcess with redirected stdout/stderr (handle list, not “inherit everything”)

* **Spec:** Run a command, capture stdout+stderr as bytes, return exit code.
* **Constraints:** Use `CreatePipe`, `STARTUPINFOEXW`, `InitializeProcThreadAttributeList`, `UpdateProcThreadAttribute(PROC_THREAD_ATTRIBUTE_HANDLE_LIST)`, and `CreateProcessW`.
* **Signature:**

```rust
pub fn run_capture(command_line: &str, timeout_ms: u32) -> Result<(u32, Vec<u8>, Vec<u8>)>;
```

* **Example:**

```rust
let (code, out, err) = run_capture(r#"cmd.exe /c echo hello"#, 2000)?;
```
