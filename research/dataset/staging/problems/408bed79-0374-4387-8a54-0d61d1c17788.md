7. List directory entries (Win32)

* **Spec:** Return a sorted list of file/directory names in a directory (non-recursive), excluding `.` and `..`.
* **Constraints:** Use `FindFirstFileW` / `FindNextFileW` / `FindClose`.
* **Signature:**

```rust
pub fn list_dir(path: &Path) -> Result<Vec<std::ffi::OsString>>;
```

* **Example:**

```rust
let names = list_dir(Path::new(r"C:\tmp\folder"))?;
```
