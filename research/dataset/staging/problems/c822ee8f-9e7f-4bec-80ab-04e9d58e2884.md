12) Touch file (update last-write to “now”)

* **Spec:** Set `mtime` to current system time.
* **Constraints:** Convert `SystemTime` to `FILETIME`; open with `FILE_WRITE_ATTRIBUTES`.
* **Signature:**

```rust
pub fn touch(path: &Path) -> Result<()>;
```

* **Example:**

```rust
touch(Path::new("a.txt"))?;
```
