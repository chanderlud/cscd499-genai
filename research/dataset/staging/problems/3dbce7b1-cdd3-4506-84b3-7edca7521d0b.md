5. Named pipe impersonation (prove you’re actually impersonating)

* **Spec:** Create a named pipe server and client. On the server, impersonate the client and return the **impersonated** user SID string from the thread token.
* **Constraints:** `CreateNamedPipeW`, `CreateFileW`, `ImpersonateNamedPipeClient`, `OpenThreadToken`, `GetTokenInformation(TokenUser)`, `RevertToSelf`, `ConvertSidToStringSidW`.
* **Signature:**

```rust
pub fn named_pipe_impersonated_sid(pipe_name: &str) -> Result<String>;
```

* **Example:**

```rust
let sid = named_pipe_impersonated_sid(r"\\.\pipe\imp_test")?;
```
