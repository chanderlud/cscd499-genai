18. Shell Link (.lnk) create + readback (COM)

* **Spec:** Create a `.lnk` pointing to `target`, set optional args, save; then load and return `(target, args)`.
* **Constraints:** Use `CoCreateInstance(CLSID_ShellLink)`, `IShellLinkW`, `IPersistFile`.
* **Signature:**

```rust
pub fn shell_link_roundtrip(lnk: &Path, target: &Path, args: &str) -> Result<(PathBuf, String)>;
```

* **Example:**

```rust
let (t, a) = shell_link_roundtrip(Path::new(r"C:\tmp\a.lnk"), Path::new(r"C:\Windows\notepad.exe"), "x.txt")?;
```
