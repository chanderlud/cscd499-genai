11. Apply SDDL DACL to a file and verify it roundtrips

* **Spec:** Apply a provided SDDL security descriptor to a file, then query it back and return the resulting SDDL.
* **Constraints:** Use `ConvertStringSecurityDescriptorToSecurityDescriptorW` + `SetFileSecurityW` + `GetFileSecurityW` + `ConvertSecurityDescriptorToStringSecurityDescriptorW`.
* **Signature:**

```rust
pub fn set_get_file_sddl(path: &Path, sddl: &str) -> Result<String>;
```

* **Example:**

```rust
let out = set_get_file_sddl(Path::new(r"C:\tmp\a.txt"), "D:P(A;;GA;;;SY)(A;;GA;;;BA)")?;
```
