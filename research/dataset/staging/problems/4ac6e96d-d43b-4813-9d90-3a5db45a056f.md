15. COM initialize guard (COM)

* **Spec:** Initialize COM on the current thread and uninitialize on drop.
* **Constraints:** Use `CoInitializeEx` + `CoUninitialize`. Support both MTA and STA via a parameter.
* **Signature:**

```rust
pub struct ComGuard { /* opaque */ }

pub fn com_init(coinit: u32) -> Result<ComGuard>;
```

* **Example:**

```rust
let _com = com_init(windows::Win32::System::Com::COINIT_MULTITHREADED.0)?;
```
