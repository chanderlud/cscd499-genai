02) Read entire file (ReadFile loop)

* **Spec:** Read all bytes from a handle until EOF.
* **Constraints:** Loop until `ReadFile` returns 0 bytes; handle large files; avoid infinite loops.
* **Signature:**

```rust
pub fn read_all(handle: &OwnedHandle) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let data = read_all(&h)?;
```
