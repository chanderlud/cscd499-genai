15) Delete with retry on sharing violations

* **Spec:** Attempt delete; retry a few times if file is temporarily locked.
* **Constraints:** Only retry on expected errors; don’t hang forever.
* **Signature:**

```rust
pub fn delete_with_retry(path: &Path, attempts: u32) -> Result<()>;
```

* **Example:**

```rust
delete_with_retry(Path::new("busy.tmp"), 10)?;
```
