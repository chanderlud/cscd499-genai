01) Open/Create a file (CreateFileW)

* **Spec:** Open or create a file with explicit access/share/creation/flags.
* **Constraints:** Use UTF-16 paths; close handle; decide on long-path handling (`\\?\`).
* **Signature:**

```rust
pub fn open_file(
    path: &Path,
    access: FILE_ACCESS_FLAGS,
    share: FILE_SHARE_MODE,
    disposition: FILE_CREATION_DISPOSITION,
    flags: FILE_FLAGS_AND_ATTRIBUTES,
) -> Result<OwnedHandle>;
```

* **Example:**

```rust
let h = open_file(Path::new(r"C:\tmp\a.bin"), FILE_GENERIC_READ, FILE_SHARE_READ, OPEN_EXISTING, FILE_ATTRIBUTE_NORMAL)?;
```
