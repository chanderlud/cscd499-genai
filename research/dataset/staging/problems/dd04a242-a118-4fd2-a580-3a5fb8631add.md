12. IO completion port: post and drain completions

* **Spec:** Create an IOCP, post N completions with distinct keys, then drain them in FIFO-ish order (order not guaranteed), returning the keys seen.
* **Constraints:** Use `CreateIoCompletionPort` + `PostQueuedCompletionStatus` + `GetQueuedCompletionStatus`.
* **Signature:**

```rust
pub fn iocp_post_and_drain(keys: &[usize], timeout_ms: u32) -> Result<Vec<usize>>;
```

* **Example:**

```rust
let seen = iocp_post_and_drain(&[1, 2, 3], 1000)?;
```
