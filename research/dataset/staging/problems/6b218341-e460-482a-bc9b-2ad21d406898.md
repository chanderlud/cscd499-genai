10. Crypt32: self-signed cert in memory + PKCS#7 sign/verify

* **Spec:** Generate a key, create a self-signed cert, sign `message` into a PKCS#7/CMS blob, then verify and return the recovered content.
* **Constraints:** `CryptAcquireContextW`, `CryptGenKey`, `CertCreateSelfSignCertificate`, `CryptSignMessage`, `CryptVerifyMessageSignature`. No touching user/machine stores.
* **Signature:**

```rust
pub fn crypt32_sign_verify_roundtrip(message: &[u8]) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let out = crypt32_sign_verify_roundtrip(b"sign me")?;
```
