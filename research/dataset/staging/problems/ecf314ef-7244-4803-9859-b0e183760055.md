05) Seek + read exact (SetFilePointerEx + ReadFile)

* **Spec:** Read exactly `n` bytes from a given offset.
* **Constraints:** Use `SetFilePointerEx`; return error on short read (EOF).
* **Signature:**

```rust
pub fn pread_exact(handle: &OwnedHandle, offset: i64, buf: &mut [u8]) -> Result<()>;
```

* **Example:**

```rust
let mut hdr = [0u8; 16];
pread_exact(&h, 0, &mut hdr)?;
```
