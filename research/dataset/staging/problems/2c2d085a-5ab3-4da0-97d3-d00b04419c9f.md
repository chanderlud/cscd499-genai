17. COM Structured Storage in transacted mode: Revert must undo writes

* **Spec:** Create a compound file in `STGM_TRANSACTED`, write a stream, call `Revert`, close. Reopen and confirm the stream write did not persist.
* **Constraints:** `StgCreateStorageEx(STGM_TRANSACTED)`, `IStorage::CreateStream`, `IStorage::Revert`, reopen with `StgOpenStorageEx`.
* **Signature:**

```rust
pub fn stg_transacted_revert(path: &Path, stream: &str, data: &[u8]) -> Result<bool>;
```

* **Example:**

```rust
let persisted = stg_transacted_revert(Path::new(r"C:\tmp\t.stg"), "Data", b"nope")?;
```
