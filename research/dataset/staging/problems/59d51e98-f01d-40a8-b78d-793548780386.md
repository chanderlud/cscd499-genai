15. SRWLock + ConditionVariable bounded queue stress test

* **Spec:** Implement a bounded multi-producer/multi-consumer queue using only SRW locks + condition variables; run a stress scenario and return all consumed items.
* **Constraints:** `InitializeSRWLock`, `AcquireSRWLockExclusive/Shared`, `InitializeConditionVariable`, `SleepConditionVariableSRW`, `WakeConditionVariable/WakeAllConditionVariable`. No Rust `Mutex/Condvar`.
* **Signature:**

```rust
pub fn bounded_queue_stress(n_items: u32, capacity: u32, producers: u32, consumers: u32) -> Result<Vec<u32>>;
```

* **Example:**

```rust
let got = bounded_queue_stress(10_000, 64, 4, 4)?;
```
