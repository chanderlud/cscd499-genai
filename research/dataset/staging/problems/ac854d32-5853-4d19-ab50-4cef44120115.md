42) Convert path → IShellItem (SHCreateItemFromParsingName)

* **Spec:** Create an `IShellItem` from a filesystem path.
* **Constraints:** Requires COM initialized (`CoInitializeEx`); handle non-filesystem items if desired.
* **Signature:**

```rust
pub fn shell_item_from_path(path: &Path) -> Result<IShellItem>;
```

* **Example:**

```rust
let item = shell_item_from_path(Path::new(r"C:\Windows"))?;
```
