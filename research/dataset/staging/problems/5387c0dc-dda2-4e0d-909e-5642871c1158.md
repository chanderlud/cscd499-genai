11) Set file attributes (readonly/hidden/etc.)

* **Spec:** Set attributes bitmask (readonly/hidden/system/archive).
* **Constraints:** Use `SetFileAttributesW`; avoid clobbering unintended bits.
* **Signature:**

```rust
pub fn set_attrs(path: &Path, attrs: u32) -> Result<()>;
```

* **Example:**

```rust
set_attrs(Path::new("a.txt"), FILE_ATTRIBUTE_READONLY.0)?;
```
