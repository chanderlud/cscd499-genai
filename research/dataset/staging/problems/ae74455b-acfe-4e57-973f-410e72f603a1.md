20) Query free disk space (GetDiskFreeSpaceExW)

* **Spec:** Return total/free bytes for the volume containing a path.
* **Constraints:** Path must exist (or use root); handle UNC paths.
* **Signature:**

```rust
pub fn disk_free_space(path: &Path) -> Result<(u64 /*free*/, u64 /*total*/)>; 
```

* **Example:**

```rust
let (free, total) = disk_free_space(Path::new(r"C:\"))?;
```
