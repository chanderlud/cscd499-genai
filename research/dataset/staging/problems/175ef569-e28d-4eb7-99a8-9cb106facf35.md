5. Overlapped read with cancellation

* **Spec:** Kick off an overlapped `ReadFile`, then cancel it, and report that it was canceled (not “some random error”).
* **Constraints:** Use `ReadFile` with `OVERLAPPED` + `CancelIoEx` + `GetOverlappedResult`.
* **Signature:**

```rust
pub fn overlapped_read_then_cancel(path: &Path) -> Result<()>;
```

* **Example:**

```rust
overlapped_read_then_cancel(Path::new(r"C:\tmp\bigfile.bin"))?;
```
