32) Windowed mapping for huge files

* **Spec:** Map a window `[offset, offset+len)` instead of whole file.
* **Constraints:** Offset must align to allocation granularity; adjust internal offset.
* **Signature:**

```rust
pub fn map_window(handle: &OwnedHandle, mapping: &OwnedHandle, offset: u64, len: usize) -> Result<(*const u8, usize, u64 /*delta*/)>; 
```

* **Example:**

```rust
let (p, n, delta) = map_window(&fh, &mh, 1_000_000, 64 * 1024)?;
```
