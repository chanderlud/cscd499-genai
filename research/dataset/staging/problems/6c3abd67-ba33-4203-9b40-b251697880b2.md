2. Windows Compression API roundtrip (compress + decompress)

* **Spec:** Compress bytes with a chosen algorithm, then decompress back to the original.
* **Constraints:** Use `CreateCompressor`, `Compress`, `CreateDecompressor`, `Decompress`. (Handle size probing correctly.)
* **Signature:**

```rust
pub fn compress_roundtrip(algorithm: u32, data: &[u8]) -> Result<Vec<u8>>;
```

* **Example:**

```rust
use windows::Win32::Storage::Compression::COMPRESS_ALGORITHM_XPRESS_HUFF;
let out = compress_roundtrip(COMPRESS_ALGORITHM_XPRESS_HUFF.0, b"aaaaabbbbcccc")?;
```
