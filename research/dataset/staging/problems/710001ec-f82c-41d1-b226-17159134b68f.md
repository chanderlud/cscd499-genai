13. Threadpool timer that signals an event

* **Spec:** Create a threadpool timer that signals a Win32 event after `due_ms`.
* **Constraints:** Use `CreateThreadpoolTimer` + `SetThreadpoolTimer` + `WaitForSingleObject`.
* **Signature:**

```rust
pub fn tp_timer_signal(due_ms: u32, timeout_ms: u32) -> Result<bool>;
```

* **Example:**

```rust
let fired = tp_timer_signal(50, 1000)?;
```
