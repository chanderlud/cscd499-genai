22. IFileOperation copy without UI + progress count (COM)

* **Spec:** Copy N source files into a destination folder; return how many items reported “finished” via a progress sink.
* **Constraints:** Use `IFileOperation` with no-UI flags; implement `IFileOperationProgressSink` to count completed items.
* **Signature:**

```rust
pub fn fileop_copy_with_progress(srcs: &[PathBuf], dst_dir: &Path) -> Result<u32>;
```

* **Example:**

```rust
let n = fileop_copy_with_progress(&[r"C:\tmp\a.txt".into(), r"C:\tmp\b.txt".into()], Path::new(r"C:\tmp\out"))?;
```
