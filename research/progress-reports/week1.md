### Eval Server
- HTTP API designed to evaluate Rust code on a virtual machine
- The purpose is to execute arbitrary code without compromising the security of a real machine
- /evaluate endpoint accepts a snippet of Rust code and the required Cargo dependencies
- Compiles, lints, and runs unit tests reporting the results for each action
- The eval server is used by the eval runner to evaluate code generated by the LLM for each problem in the dataset

### Eval Runner
- The main program for benchmarking LLMs on a set of Rust programming problems
- Each problem consists of a prompt and a set of unit tests
- The eval runner inserts each problem's prompt into a template with code generation rules for the LLM
- The unit tests are appended onto the LLM outputs, then the complete block of code is sent to the eval server
- The results for each problem are collected, statistics calculated, then csv and json outputs are created
