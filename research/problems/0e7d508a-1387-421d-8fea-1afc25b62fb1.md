23) Directory change notifications (ReadDirectoryChangesW)

* **Spec:** Watch a directory for create/delete/modify/rename.
* **Constraints:** Needs an open directory handle; overlapped I/O recommended; parse `FILE_NOTIFY_INFORMATION`.
* **Signature:**

```rust
pub fn watch_dir(path: &Path, recursive: bool) -> Result<std::sync::mpsc::Receiver<Vec<u8>>>;
```

* **Example:**

```rust
let rx = watch_dir(Path::new(r"C:\tmp"), true)?;
```
