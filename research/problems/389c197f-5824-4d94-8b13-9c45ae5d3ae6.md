30) Map file read-write

* **Spec:** Map file read-write for in-place edits.
* **Constraints:** Open with write; mapping protection `PAGE_READWRITE`; use `FlushViewOfFile` optionally.
* **Signature:**

```rust
pub struct MappedViewMut { pub ptr: *mut u8, pub len: usize }
pub fn map_rw(path: &Path) -> Result<(OwnedHandle, OwnedHandle, MappedViewMut)>;
```

* **Example:**

```rust
let (_fh, _mh, mut v) = map_rw(Path::new("a.bin"))?;
```
