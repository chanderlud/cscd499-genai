19) Create unique temp file

* **Spec:** Create a unique temp file and return its path + handle.
* **Constraints:** Use `GetTempPathW` + `GetTempFileNameW` (or `CREATE_NEW` loop).
* **Signature:**

```rust
pub fn create_temp_file(prefix: &str) -> Result<(std::path::PathBuf, OwnedHandle)>;
```

* **Example:**

```rust
let (p, h) = create_temp_file("rs")?;
```
