39) Double-map ring buffer (MapViewOfFileEx trick)

* **Spec:** Create a ring buffer where wrap-around is contiguous via two adjacent mappings.
* **Constraints:** Requires careful address reservation; may fail on ASLR; retry strategy.
* **Signature:**

```rust
pub fn map_double_view(mapping: &OwnedHandle, size: usize) -> Result<(*mut u8, usize)>;
```

* **Example:**

```rust
let (ptr, cap) = map_double_view(&m, 1 << 20)?;
```
