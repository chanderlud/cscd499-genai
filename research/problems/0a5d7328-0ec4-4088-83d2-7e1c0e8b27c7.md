07) Create directory recursively

* **Spec:** Create all missing parents then final directory.
* **Constraints:** Use `CreateDirectoryW`; tolerate “already exists”.
* **Signature:**

```rust
pub fn create_dir_all(path: &Path) -> Result<()>;
```

* **Example:**

```rust
create_dir_all(Path::new(r"C:\tmp\a\b\c"))?;
```
