17) Create symlink (CreateSymbolicLinkW)

* **Spec:** Create a symbolic link to file or directory.
* **Constraints:** May require admin or Developer Mode; set correct flag for dir.
* **Signature:**

```rust
pub fn create_symlink(link: &Path, target: &Path, is_dir: bool) -> Result<()>;
```

* **Example:**

```rust
create_symlink(Path::new("docs_link"), Path::new(r"C:\Users\me\Documents"), true)?;
```
