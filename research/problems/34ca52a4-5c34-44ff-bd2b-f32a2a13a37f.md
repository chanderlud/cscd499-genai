36) Producer/consumer shared memory with events

* **Spec:** Use shared memory + named events for “data ready” signaling.
* **Constraints:** Deadlocks are a lifestyle; use timeouts; document protocol.
* **Signature:**

```rust
pub fn shm_channel_names(base: &str) -> (String, String, String); // mapping, evt_write, evt_read
```

* **Example:**

```rust
let (map, ew, er) = shm_channel_names("Local\\pipe1");
```
