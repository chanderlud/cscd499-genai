44) File Open dialog (IFileOpenDialog) pick one file

* **Spec:** Show Windows file picker and return selected file path.
* **Constraints:** Must be STA thread; call `CoInitializeEx(...APARTMENTTHREADED)`; UI requires message pump friendliness.
* **Signature:**

```rust
pub fn pick_file(title: &str, filter: Option<&[(&str, &str)]>) -> Result<Option<std::path::PathBuf>>;
```

* **Example:**

```rust
let p = pick_file("Pick an image", Some(&[("Images", "*.png;*.jpg")]))?;
```
