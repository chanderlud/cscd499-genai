27) Write an alternate data stream

* **Spec:** Write bytes to `file:streamname` (create/overwrite stream).
* **Constraints:** Same as normal file write; stream name validation.
* **Signature:**

```rust
pub fn write_stream(path: &Path, stream: &str, data: &[u8]) -> Result<()>;
```

* **Example:**

```rust
write_stream(Path::new("a.txt"), "meta", b"{\"k\":1}")?;
```
