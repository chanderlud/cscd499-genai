41) Get Known Folder path (SHGetKnownFolderPath)

* **Spec:** Resolve known folder (Documents, Downloads, Desktop, etc.) to a filesystem path.
* **Constraints:** COM not required, but you must free returned memory (`CoTaskMemFree`).
* **Signature:**

```rust
pub fn known_folder_path(id: KNOWNFOLDERID) -> Result<std::path::PathBuf>;
```

* **Example:**

```rust
let p = known_folder_path(FOLDERID_Downloads)?;
```
