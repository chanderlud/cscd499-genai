09) Enumerate directory entries (FindFirstFileW/FindNextFileW)

* **Spec:** List names + basic metadata for items in a directory.
* **Constraints:** Skip `.` and `..`; handle long names; close find handle.
* **Signature:**

```rust
pub fn list_dir(path: &Path) -> Result<Vec<(String, WIN32_FIND_DATAW)>>;
```

* **Example:**

```rust
for (name, _) in list_dir(Path::new(r"C:\Windows"))? { println!("{name}"); }
```
