08) Delete directory tree (recursive)

* **Spec:** Remove all files/subdirs under a directory, then the directory.
* **Constraints:** Handle readonly attributes; junctions/symlinks carefully (don’t follow unless intended).
* **Signature:**

```rust
pub fn remove_dir_tree(path: &Path) -> Result<()>;
```

* **Example:**

```rust
remove_dir_tree(Path::new(r"C:\tmp\old"))?;
```
