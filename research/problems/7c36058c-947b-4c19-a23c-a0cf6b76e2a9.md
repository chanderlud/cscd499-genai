50) WinRT: Enumerate files in LocalFolder

* **Spec:** List files stored in LocalFolder for basic app storage management.
* **Constraints:** Async; items are `StorageFile`; return names/paths as available.
* **Signature:**

```rust
pub async fn appdata_list_files() -> Result<Vec<String>>;
```

* **Example:**

```rust
for n in appdata_list_files().await? { println!("{n}"); }
```
