26) Read an alternate data stream

* **Spec:** Read bytes from `file:streamname`.
* **Constraints:** Construct `path:stream` carefully; treat as separate file.
* **Signature:**

```rust
pub fn read_stream(path: &Path, stream: &str) -> Result<Vec<u8>>;
```

* **Example:**

```rust
let data = read_stream(Path::new("a.txt"), "meta")?;
```
