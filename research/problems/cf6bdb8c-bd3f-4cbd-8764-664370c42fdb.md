31) Resize file for mapping (SetEndOfFile)

* **Spec:** Ensure a file is exactly `new_len` bytes.
* **Constraints:** Use `SetFilePointerEx` + `SetEndOfFile`; careful with truncation.
* **Signature:**

```rust
pub fn resize_file(handle: &OwnedHandle, new_len: u64) -> Result<()>;
```

* **Example:**

```rust
resize_file(&h, 10 * 1024 * 1024)?;
```
