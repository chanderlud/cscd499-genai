29) Map file read-only

* **Spec:** Map entire file read-only and return pointer+len.
* **Constraints:** Must unmap; pointer is unsafe; length from file size.
* **Signature:**

```rust
pub struct MappedView { pub ptr: *const u8, pub len: usize }
pub fn map_ro(path: &Path) -> Result<(OwnedHandle /*file*/, OwnedHandle /*mapping*/, MappedView)>;
```

* **Example:**

```rust
let (_fh, _mh, v) = map_ro(Path::new("a.bin"))?;
```
