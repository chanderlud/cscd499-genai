21) Volume information (GetVolumeInformationW)

* **Spec:** Get FS name, volume label, serial, max component length, flags.
* **Constraints:** Provide root path (`C:\`); buffer sizing.
* **Signature:**

```rust
pub fn volume_info(root: &Path) -> Result<(String, String, u32, u32)>;
```

* **Example:**

```rust
let (label, fs, serial, flags) = volume_info(Path::new(r"C:\"))?;
```
