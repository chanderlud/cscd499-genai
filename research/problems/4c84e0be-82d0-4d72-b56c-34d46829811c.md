35) Map shared memory as a struct header + payload

* **Spec:** Define a header (magic/version/len) and validate before use.
* **Constraints:** Alignment + `repr(C)`; atomic fields if concurrent.
* **Signature:**

```rust
#[repr(C)]
pub struct ShmHeader { pub magic: u32, pub version: u32, pub len: u32 }
pub fn shm_view<T>(mapping: &OwnedHandle) -> Result<(*mut T, usize)>;
```

* **Example:**

```rust
let (hdr, _bytes) = shm_view::<ShmHeader>(&m)?;
```
