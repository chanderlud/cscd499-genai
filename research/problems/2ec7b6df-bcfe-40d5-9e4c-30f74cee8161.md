34) Open named shared memory (OpenFileMappingW)

* **Spec:** Open existing mapping by name.
* **Constraints:** Must match permissions; handle “not found”.
* **Signature:**

```rust
pub fn open_named_mapping(name: &str, write: bool) -> Result<OwnedHandle>;
```

* **Example:**

```rust
let m = open_named_mapping("Local\\myshare", true)?;
```
