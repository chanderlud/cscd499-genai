10) Get file metadata (attributes/size/times)

* **Spec:** Return size, attributes, creation/access/write times.
* **Constraints:** Prefer handle-based queries; handle reparse points as desired.
* **Signature:**

```rust
pub struct FileMeta { pub size: u64, pub attrs: u32, pub ctime: u64, pub atime: u64, pub mtime: u64 }
pub fn file_meta(path: &Path) -> Result<FileMeta>;
```

* **Example:**

```rust
let m = file_meta(Path::new(r"C:\tmp\a.bin"))?;
```
