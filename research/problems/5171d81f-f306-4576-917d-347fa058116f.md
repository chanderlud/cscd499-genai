13) Copy with progress (CopyFileExW)

* **Spec:** Copy a file while reporting progress callback.
* **Constraints:** Callback must be fast; support cancel; handle overwrite policy.
* **Signature:**

```rust
pub fn copy_with_progress<F: Fn(u64, u64) -> bool>(src: &Path, dst: &Path, progress: F) -> Result<()>;
```

* **Example:**

```rust
copy_with_progress(Path::new("a.bin"), Path::new("b.bin"), |done, total| { println!("{done}/{total}"); true })?;
```
